📚 Part 1: 核心理念
什么是 Tool Selection？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  【定义】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Tool Selection = 在正确的时机，选择正确的工具，用正确的方式

不是:
  ✗ "我有锤子，所以一切都是钉子"
  ✗ "我就用这个工具，不管合不合适"
  ✗ "随便选一个试试"

而是:
  ✓ 分析任务需求
  ✓ 匹配工具能力
  ✓ 选择最优方案
  ✓ 优化参数配置

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  【为什么重要？】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

选对工具 = 事半功倍
选错工具 = 事倍功半

例子:
  
  任务: 生成一张产品图
  
  ❌ 错误选择:
     → 用 Midjourney (艺术风格强，但指令跟随弱)
     → 结果: 很美，但产品细节不对
  
  ✅ 正确选择:
     → 用 GPT-Image-2 (指令跟随强，通用性好)
     → 结果: 准确还原产品需求

技巧1: 质量设置 (gpt_image_2)
  
  原则:
    ✓ 默认使用 medium
    ✓ 仅在用户明确要求时使用 high
    ✓ 快速迭代时使用 low
    ✓ 让模型决定时使用 auto
  
  场景匹配:
    high → 最高质量、印刷级、商业交付、英雄镜头
    medium → 标准质量、大多数任务 (默认)
    low → 快速草稿、预览、迭代
    auto → 用户明确说"auto"或场景质量很重要

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

技巧2: 尺寸设置
  
  原则:
    ✓ 根据用途选择合适尺寸
    ✓ 不要盲目追求大尺寸
    ✓ 考虑生成时间和成本
  
  常用尺寸:
    社交媒体:
      Instagram Post: 2048x2048 (1:1)
      Instagram Story: 1440x2560 (9:16)
      Twitter Post: 2048x1024 (2:1)
    
    产品图:
      主图: 2048x2048 (1:1)
      详情页: 2400x1600 (3:2)
    
    海报:
      横版: 2560x1440 (16:9)
      竖版: 1440x2560 (9:16)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

技巧3: 分辨率设置 (nano_banana系列)
  
  原则:
    ✓ 默认使用 1K
    ✓ 仅在用户明确要求时使用更高分辨率
    ✓ 考虑任务复杂度
  
  场景匹配:
    512 → 快速预览、草稿
    1K → 默认、大多数任务
    2K → 高质量输出、打印
    4K → 最高质量、大幅打印

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

技巧4: 提示词优化
  
  gpt_image_2 / nano_banana系列:
    ✓ 详细、全面的描述
    ✓ 明确的指令
    ✓ 结构化的提示词

    ✓ 短而简单
    ✓ 避免长列表
    ✓ 具体描述性词汇
    ✓ 关注想要的,而非不想要的

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  【常见错误】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

错误1: 工具选择不当
  
  ❌ 用Midjourney生成产品图
     → 艺术风格太强,产品细节不准确
  
  ✅ 用gpt_image_2生成产品图
     → 精确控制,产品细节准确

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

错误2: 参数设置不当
  
  ❌ 所有任务都用 quality="high"
     → 浪费时间和成本
  
  ✅ 根据需求设置质量
     → medium为默认,仅在需要时用high

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

错误3: 委托时机不当
  
  ❌ 单图生成也委托给SubAgent
     → 增加复杂度,降低效率
  
  ✅ 单图生成自己做
     → 直接、高效

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

错误4: SubAgent上下文不完整
  
  ❌ 委托时不提供参考图像URL
     → SubAgent看不到,无法使用
  
  ✅ 在project_context中明确列出所有参考图像URL
     → SubAgent可以正确使用

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

错误5: 盲目追求高分辨率
  
  ❌ 所有任务都用4K
     → 浪费时间,可能导致质量问题
  
  ✅ 根据用途选择合适分辨率
     → 1K为默认,仅在需要时用更高
