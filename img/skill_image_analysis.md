 Skill: Image Analysis (图像分析)
完整实战手册 - 毫无保留版
🎯 第一部分: 核心原理
什么是 Image Analysis?
定义:
  系统化地"读懂"图像，提取关键信息的能力

本质:
  这是一种"视觉解码"能力
  - 输入: 用户提供的参考图像
  - 输出: 结构化的、可用于生成的图像信息

为什么重要?
  用户经常说: "我想要这种风格"、"参考这张图"
  但图像包含海量信息!
  
  一张图可能包含:
    - 构图 (Composition)
    - 色彩 (Color)
    - 光线 (Lighting)
    - 材质 (Material)
    - 风格 (Style)
    - 氛围 (Mood)
    - 细节 (Details)
    - 技术参数 (Technical)
    ... 等等

  如果不系统分析:
    ❌ 抓不住重点
    ❌ 遗漏关键信息
    ❌ 生成结果偏差大
    ❌ 无法精确复现

  有了Image Analysis:
    ✓ 能系统化提取信息
    ✓ 能识别用户真正想要什么
    ✓ 能转化为精确的提示词
    ✓ 能指导AI生成准确结果

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  关键认知: 看图不是"看一眼"，是"解剖"!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

普通人看图:
  "哦，这张图不错" → 结束

设计师看图:
  "这张图的构图是三分法"
  "主光源在左上45度"
  "色调是冷色调，饱和度低"
  "材质是哑光木质"
  "风格是写实主义"
  "氛围是宁静、专注"
  ... 等等

所以:
  Image Analysis是你的"设计师之眼"!
  你要像外科医生解剖人体一样解剖图像!
🧠 第二部分: 分析框架 - 8维度分析法
核心框架: 8个维度，层层递进
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  维度1: 场景类型 (Scene Type)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  快速定位图像的基本类型

分类:
  1. 室内场景 (Interior)
     - 办公室、卧室、客厅、餐厅等
  
  2. 室外场景 (Exterior)
     - 街道、公园、建筑外观等
  
  3. 产品摄影 (Product)
     - 纯背景产品、场景产品等
  
  4. 人物摄影 (Portrait)
     - 肖像、全身、半身等
  
  5. 自然风景 (Landscape)
     - 山川、海洋、森林等
  
  6. 抽象/概念 (Abstract/Concept)
     - 艺术作品、概念图等

分析要点:
  - 这是什么类型的场景?
  - 主体是什么? (人/物/环境)
  - 场景的功能是什么?

示例 - 你的清代办公室参考图:
  
  场景类型: 室内场景 (Interior)
  子类型: 历史办公室
  主体: 环境 + 人物
  功能: 展示清代官员办公环境

为什么重要?
  不同类型的场景，分析重点完全不同!
  
  室内场景 → 重点: 空间布局、家具、光线
  产品摄影 → 重点: 产品细节、材质、光影
  人物摄影 → 重点: 表情、姿态、服装

  维度2: 构图分析 (Composition)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  理解画面的结构和视觉引导

关键要素:

1. 构图法则
   - 三分法 (Rule of Thirds)
   - 黄金分割 (Golden Ratio)
   - 中心构图 (Center Composition)
   - 对角线构图 (Diagonal)
   - 框架构图 (Frame)
   - 引导线构图 (Leading Lines)

2. 视角 (Viewpoint)
   - 平视 (Eye-level)
   - 俯视 (High angle / Bird's eye)
   - 仰视 (Low angle / Worm's eye)
   - 角度 (正面/侧面/45度等)

3. 景深层次 (Depth)
   - 前景 (Foreground)
   - 中景 (Midground)
   - 背景 (Background)

4. 画面比例 (Aspect Ratio)
   - 16:9 (横向宽屏)
   - 9:16 (竖向)
   - 1:1 (方形)
   - 4:3, 3:2 等

5. 留白 (Negative Space)
   - 留白位置
   - 留白比例
   - 留白作用

分析模板:

构图法则: [识别使用了哪种构图法]
视角: [相机位置和角度]
景深层次:
  - 前景: [有什么]
  - 中景: [主体内容]
  - 背景: [背景内容]
画面比例: [宽高比]
留白: [哪里有留白，作用是什么]

示例 - 你的"西向东拍"参考图:

构图法则: 中心偏右构图
  - 人物(张十三)在画面中心偏右
  - 形成视觉焦点

视角: 
  - 平视角度 (Eye-level)
  - 相机高度约1.6米 (人眼高度)
  - 从西向东拍摄
  - 轻微的广角透视

景深层次:
  - 前景: 空旷 (相机前方无遮挡)
  - 中景: 人物、椅子、桌子 (主体内容)
  - 背景: 东墙、门 (提供空间感)

画面比例: 约16:9 横向

留白: 
  - 左侧留白较多 (强调空间感)
  - 上方留白 (展示天花板高度)

为什么重要?
  构图决定了画面的"骨架"!
  如果构图错了，其他都白搭!

  维度3: 光线分析 (Lighting)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  理解光线的来源、性质和效果

关键要素:

1. 光源类型
   - 自然光 (Natural Light)
     · 日光 (Daylight)
     · 月光 (Moonlight)
     · 天光 (Skylight)
   
   - 人造光 (Artificial Light)
     ﾷ 烛光 (Candlelight)
     · 灯光 (Lamp)
     · 霓虹灯 (Neon)

2. 光源数量
   - 单光源 (Single Light)
   - 多光源 (Multiple Lights)
   - 主光 + 补光 + 轮廓光 (3-point lighting)

3. 光源方向
   - 顶光 (Top Light)
   - 侧光 (Side Light)
   - 逆光 (Back Light)
   - 前光 (Front Light)
   - 45度光 (45-degree Light)

4. 光线性质
   - 硬光 (Hard Light) - 明显阴影
   - 软光 (Soft Light) - 柔和阴影
   - 散射光 (Diffused Light) - 几乎无阴影

5. 色温 (Color Temperature)
   - 暖光 (Warm) - 2700-3500K (烛光、日落)
   - 中性 (Neutral) - 4000-5000K (日光)
   - 冷光 (Cool) - 5500-7000K (阴天、月光)

6. 光线强度
   - 强光 (Bright)
   - 中等 (Medium)
   - 弱光 (Dim)

7. 阴影
   - 阴影方向
   - 阴影硬度
   - 阴影长度

分析模板:

主光源:
  - 类型: [自然光/人造光]
  - 位置: [方向和高度]
  - 性质: [硬光/软光]
  - 色温: [暖/中/冷]
  - 强度: [强/中/弱]

补光源 (如果有):
  - [同上]

阴影:
  - 方向: [阴影指向哪里]
  - 硬度: [清晰/模糊]
  - 长度: [长/短]

整体氛围:
  - [明亮/昏暗/神秘/温暖等]

示例 - 你的清代办公室:

主光源1: 烛光
  - 类型: 人造光 (蜡烛)
  - 位置: 桌面上，多个点光源
  - 性质: 软光 (火焰柔和)
  - 色温: 暖光 (2800K，琥珀色)
  - 强度: 中等偏弱

主光源2: 窗外光
  - 类型: 自然光 (月光/雪光)
  - 位置: 南窗，侧面光
  - 性质: 软光 (散射)
  - 色温: 冷光 (6000K，蓝白色)
  - 强度: 弱

光线混合:
  - 暖色烛光 (近景，主体)
  - 冷色月光 (远景，环境)
  - 形成冷暖对比

阴影:
  - 方向: 多方向 (多光源)
  - 硬度: 柔和
  - 长度: 短 (光源近)

整体氛围:
  - 昏暗、宁静、专注
  - 冷暖对比营造戏剧感
  - 烛光营造历史感

为什么重要?
  光线决定了画面的"灵魂"!
  同样的场景，不同的光线，完全不同的感觉!

  维度4: 色彩分析 (Color)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  理解色彩的使用和情感表达

关键要素:

1. 色调 (Hue)
   - 主色调: 画面的主导颜色
   - 辅助色: 次要颜色
   - 点缀色: 小面积强调色

2. 饱和度 (Saturation)
   - 高饱和: 鲜艳、活力
   - 中饱和: 自然、平衡
   - 低饱和: 柔和、复古

3. 明度 (Value/Brightness)
   - 高调 (High Key): 明亮、轻快
   - 中调 (Mid Key): 平衡
   - 低调 (Low Key): 昏暗、神秘

4. 色彩关系
   - 单色 (Monochromatic): 一个色相的不同明暗
   - 类似色 (Analogous): 色环上相邻的颜色
   - 互补色 (Complementary): 色环上相对的颜色
   - 三角色 (Triadic): 色环上等距的三个颜色

5. 色温对比
   - 暖色调 (Warm): 红、橙、黄
   - 冷色调 (Cool): 蓝、绿、紫
   - 冷暖对比: 同时使用冷暖色

6. 色彩情感
   - 红色: 热情、危险、力量
   - 蓝色: 冷静、专业、忧郁
   - 黄色: 快乐、温暖、警示
   - 绿色: 自然、和平、生机
   - 紫色: 神秘、高贵、创意
   - 黑白灰: 经典、简约、严肃

分析模板:

主色调: [主要颜色]
辅助色: [次要颜色]
点缀色: [强调色]

饱和度: [高/中/低]
明度: [高调/中调/低调]

色彩关系: [单色/类似/互补等]
色温: [暖/冷/冷暖对比]

色彩情感: [传达什么情绪]

具体色值 (如果需要精确):
  - 主色: #RRGGBB
  - 辅助色: #RRGGBB
  - 点缀色: #RRGGBB

示例 - 你的清代办公室:

主色调: 深红棕色 (Deep red-brown)
  - 来源: 木质家具、立柱
  - 占比: 约40%

辅助色: 深蓝灰色 (Deep blue-gray)
  - 来源: 墙面、地毯
  - 占比: 约35%

点缀色: 琥珀色 (Amber)
  - 来源: 烛光
  - 占比: 约5%

其他: 白色/浅灰 (墙面), 黑色 (阴影)

饱和度: 低饱和 (去饱和、复古感)
明度: 低调 (Low Key，整体昏暗)

色彩关系: 类似色 + 冷暖对比
  - 红棕、深蓝、灰色 (类似色)
  - 暖色烛光 vs 冷色月光 (冷暖对比)

色温: 冷暖对比
  - 暖色: 烛光、木质 (2800K)
  - 冷色: 月光、墙面 (6000K)

色彩情感:
  - 深沉、历史感、庄重
  - 冷暖对比营造戏剧张力
  - 低饱和度营造年代感

具体色值 (近似):
  - 深红棕: #4A2C2A
  - 深蓝灰: #2C3E50
  - 琥珀色: #FFA500
  - 墙面: #D3D3D3

为什么重要?
  色彩决定了画面的"情绪"!
  色彩是最直接的情感传达工具!

  维度5: 材质分析 (Material/Texture)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  理解物体的表面质感和物理属性

关键要素:

1. 材质类型
   - 木质 (Wood): 纹理、颜色、光泽
   - 金属 (Metal): 反射、高光、颜色
   - 布料 (Fabric): 纹理、褶皱、柔软度
   - 石材 (Stone): 纹理、粗糙度
   - 玻璃 (Glass): 透明度、反射、折射
   - 皮革 (Leather): 纹理、光泽
   - 塑料 (Plastic): 光泽、颜色
   - 纸张 (Paper): 纹理、颜色

2. 表面属性
   - 光泽度
     · 高光 (Glossy): 强反射
     · 半光 (Semi-gloss): 中等反射
     · 哑光 (Matte): 无反射
   
   - 粗糙度
     · 光滑 (Smooth)
     · 中等 (Medium)
     · 粗糙 (Rough)
   
   - 透明度
     · 不透明 (Opaque)
     · 半透明 (Translucent)
     · 透明 (Transparent)

3. 纹理细节
   - 木纹、布纹、石纹等
   - 纹理方向
   - 纹理密度
   - 纹理清晰度

4. 磨损/年代感
   - 崭新 (New)
   - 轻微磨损 (Slightly worn)
   - 明显磨损 (Worn)
   - 古旧 (Aged/Antique)

分析模板:

主要材质:
  材质1: [类型]
    - 光泽度: [高光/半光/哑光]
    - 粗糙度: [光滑/中等/粗糙]
    - 纹理: [描述]
    - 年代感: [新/旧]
  
  材质2: [同上]
  ...

材质对比:
  - [硬vs软, 光滑vs粗糙, 新vs旧等]

整体质感:
  - [奢华/朴素/现代/古典等]

示例 - 你的清代办公室:

主要材质:

材质1: 木质 (深色硬木)
  - 光泽度: 半光 (轻微反光)
  - 粗糙度: 光滑 (精细打磨)
  - 纹理: 清晰木纹，纵向
  - 年代感: 古旧 (使用痕迹，包浆)
  - 应用: 桌子、椅子、立柱

材质2: 布料 (丝绸/棉麻)
  - 光泽度: 哑光
  - 粗糙度: 柔软
  - 纹理: 细腻布纹
  - 年代感: 古旧 (轻微褪色)
  - 应用: 地毯、帘子

材质3: 纸张 (宣纸)
  - 光泽度: 哑光
  - 粗糙度: 略粗糙
  - 纹理: 纸张纤维纹理
  - 年代感: 古旧 (泛黄)
  - 应用: 文件、书籍

材质4: 金属 (铜/铁)
  - 光泽度: 哑光 (氧化)
  - 粗糙度: 中等
  - 纹理: 金属质感
  - 年代感: 古旧 (氧化、锈迹)
  - 应用: 烛台、装饰

材质对比:
  - 硬质木材 vs 柔软布料
  - 光滑表面 vs 粗糙纸张
  - 统一的古旧感

整体质感:
  - 古典、庄重、历史感
  - 精致但不奢华
  - 使用痕迹营造真实感

为什么重要?
  材质决定了画面的"真实感"!
  材质细节是区分AI图和专业作品的关键!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  维度6: 风格分析 (Style)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  识别图像的艺术风格和视觉语言

关键要素:

1. 艺术风格
   - 写实主义 (Realism)
   - 超现实主义 (Surrealism)
   - 印象派 (Impressionism)
   - 极简主义 (Minimalism)
   - 抽象 (Abstract)
   - 插画风 (Illustration)
   - 动漫风 (Anime)
   - 概念艺术 (Concept Art)

2. 渲染风格
   - 照片级写实 (Photorealistic)
   - 3D渲染 (3D Rendered)
   - 手绘 (Hand-drawn)
   - 数字绘画 (Digital Painting)
   - 素描 (Sketch)
   - 水彩 (Watercolor)
   - 油画 (Oil Painting)

3. 时代风格
   - 现代 (Modern)
   - 复古 (Vintage)
   - 古典 (Classical)
   - 未来 (Futuristic)
   - 特定年代 (如1920s, 1980s等)

4. 文化风格
   - 东方 (Oriental/Asian)
   - 西方 (Western)
   - 特定国家/地区风格

5. 视觉特征
   - 线条: 硬朗/柔和/流畅/粗糙
   - 细节: 极简/适中/极繁
   - 对比: 强烈/柔和
   - 清晰度: 锐利/柔和/模糊

6. 参考作品/艺术家
   - 类似某个艺术家的风格
   - 类似某部电影/游戏的风格

分析模板:

艺术风格: [主要风格]
渲染风格: [渲染方式]
时代风格: [时代特征]
文化风格: [文化特征]

视觉特征:
  - 线条: [描述]
  - 细节: [描述]
  - 对比: [描述]
  - 清晰度: [描述]

参考: [类似的作品/艺术家]

风格关键词: [用于提示词的关键词]

示例 - 你的清代办公室:

艺术风格: 写实主义 (Realism)
  - 追求真实的历史还原
  - 细节丰富

技巧4: 分层分析法
目的:
  将复杂图像分解为多个层次分析

方法:

1. 识别图像层次
   - 前景层
   - 中景层
   - 背景层
   - 光效层
   - 氛围层

2. 逐层分析
   - 每层有什么元素?
   - 每层的作用是什么?
   - 层与层如何配合?

3. 综合理解

示例 - 你的清代办公室:

层次1: 背景层
  内容: 东墙、门、窗
  作用: 提供空间边界和深度
  特征: 较暗、略虚化

层次2: 中景层 (主体)
  内容: 桌子、椅子、人物
  作用: 画面主体，视觉焦点
  特征: 最清晰、光线最好

层次3: 前景层
  内容: 空旷 (无遮挡)
  作用: 提供视觉通道
  特征: 干净、不抢眼

层次4: 光效层
  内容: 烛光光晕、月光光束
  作用: 营造氛围、引导视线
  特征: 柔和、温暖/冷静

层次5: 氛围层
  内容: 整体色调、颗粒、暗角
  作用: 统一画面、营造情绪
  特征: 低饱和、电影感

层次配合:
  背景 (空间) + 中景 (主体) + 光效 (氛围) + 氛围层 (情绪)
  = 完整的视觉体验

理解:
  每一层都有明确的作用，
  去掉任何一层都会影响整体效果!

   - 理解整体的意图

   - 将视觉信息转化为文字描述
   - 将感性认知转化为理性参数
   - 将模糊印象转化为精确指令

3. 复现 (Reproduction)
   - 用AI精确复现原图
   - 或创造性地变体
   - 或融合多个风格

4. 创新 (Innovation)
   - 不只是复现
   - 而是理解规律
   - 创造新的可能
