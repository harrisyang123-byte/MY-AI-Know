📘 Skill: Spatial Reasoning (空间推理)
完整实战手册 - 毫无保留版
🎯 第一部分: 核心原理
什么是 Spatial Reasoning?
定义:
  在脑海中构建、操作和推理三维空间关系的能力

本质:
  这是一种"空间想象"能力
  - 输入: 用户的空间描述 (可能模糊、不完整、矛盾)
  - 输出: 精确的、合理的、可视化的空间布局

为什么重要?
  AI生成图像时,最容易出错的就是空间关系!
  
  常见问题:
    ❌ 门在错误的墙上
    ❌ 家具尺寸不合理
    ❌ 人物位置不对
    ❌ 视角和实际布局矛盾
    ❌ 物体之间的距离不对

  有了Spatial Reasoning:
    ✓ 能发现用户描述中的矛盾
    ✓ 能补全缺失的空间信息
    ✓ 能验证生成结果是否合理
    ✓ 能精确指导AI生成正确的空间关系

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  关键认知: 这是设计师和AI的最大差距!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AI的弱点:
  - AI不真正理解"空间"
  - AI只是匹配图像模式
  - AI容易生成"看起来对,但空间错误"的图像

人类设计师的优势:
  - 能在脑海中构建3D模型
  - 能推理空间关系
  - 能发现矛盾和问题

所以:
  Spatial Reasoning是你超越AI的关键能力!
  你要成为AI的"空间导航系统"!
🧠 第二部分: 底层思维模型
空间推理的5个核心步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  步骤1: 建立坐标系 (Establish Coordinate System)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  给空间一个"参考框架"

必须明确:
  1. 方向 (东西南北 或 前后左右)
  2. 原点 (从哪里开始测量)
  3. 单位 (米、英尺等)

示例 - 你的清代办公室:
  
  坐标系:
    - 原点: 房间中心
    - X轴: 东西方向 (东为正)
    - Y轴: 南北方向 (北为正)
    - Z轴: 上下方向 (上为正)
    - 单位: 米
  
  房间边界:
    - 东墙: X = +3m (房间中心向东3米)
    - 西墙: X = -3m
    - 南墙: Y = -4m
    - 北墙: Y = +4m
    - 地面: Z = 0m
    - 天花板: Z = 3.5m

为什么重要?
  有了坐标系,所有位置都可以精确表达!
  
  没有坐标系:
    "门在东边" → 模糊!东边哪里?
  
  有了坐标系:
    "门在东墙中央,X=+3m, Y=0m" → 精确!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  步骤2: 定义边界 (Define Boundaries)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  明确空间的"容器"

必须明确:
  1. 房间形状 (矩形、L型、圆形等)
  2. 房间尺寸 (长宽高)
  3. 墙壁位置
  4. 开口位置 (门、窗)

示例 - 你的清代办公室:
  
  房间形状: 矩形
  
  尺寸:
    - 东西方向(深度): 8米
    - 南北方向(宽度): 6米
    - 高度: 3.5米
  
  墙壁:
    - 东墙: 实墙,有门
    - 西墙: 实墙
    - 南墙: 实墙,有窗
    - 北墙: 实墙
  
  开口:
    - 门: 东墙中央,1.2米宽,2.2米高
    - 窗: 南墙,位置和尺寸待定

关键技巧:
  ✓ 画草图! (哪怕是简单的平面图)
  ✓ 标注尺寸
  ✓ 标注方向

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  步骤3: 放置元素 (Place Elements)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  把物体放到空间中

每个元素必须明确:
  1. 位置 (X, Y, Z坐标 或 相对位置)
  2. 尺寸 (长宽高)
  3. 朝向 (面向哪个方向)
  4. 与其他元素的关系

示例 - 你的清代办公室:
  
  元素1: 门
    位置: 东墙中央 (X=+3m, Y=0m, Z=0m)
    尺寸: 1.2m宽 x 2.2m高
    朝向: 向外开 (向东开)
  
  元素2: 办公桌
    位置: 房间中后部 (X=0m, Y=+1m, Z=0m)
           距离东门3米
    尺寸: 1.5m宽 x 0.8m深 x 0.8m高
    朝向: 桌子长边平行于东西方向
           办公面朝东 (面向门)
  
  元素3: 椅子
    位置: 桌子后面 (X=0m, Y=+1.5m, Z=0m)
    尺寸: 0.6m宽 x 0.6m深 x 1.2m高
    朝向: 面向东 (面向门和桌子)
  
  元素4: 人物(张十三)
    位置: 坐在椅子上 (X=0m, Y=+1.5m, Z=0.5m)
    尺寸: 坐姿高度约0.9m
    朝向: 面向东 (面向门)

关键技巧:
  ✓ 从大到小放置 (先大家具,后小物件)
  ✓ 考虑功能关系 (桌椅要配套)
  ✓ 考虑动线 (人要能走动)
  ✓ 用相对位置描述 ("桌子后面3米")

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  步骤4: 验证合理性 (Validate Rationality)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  检查空间布局是否合理

检查清单:

□ 尺寸合理性
  - 家具尺寸符合人体工程学?
  - 房间能容纳所有元素?
  - 元素之间有足够空间?
  
  示例:
    ✓ 办公桌1.5m宽 → 合理 (标准尺寸)
    ✗ 办公桌5m宽 → 不合理 (太大!)

□ 位置合理性
  - 元素位置符合功能?
  - 元素之间的距离合理?
  - 没有重叠或冲突?
  
  示例:
    ✓ 椅子在桌子后面 → 合理
    ✗ 椅子在桌子前面 → 不合理 (挡住办公)

□ 朝向合理性
  - 元素朝向符合使用习惯?
  - 人物朝向合理?
  
  示例:
    ✓ 张十三面向门 → 合理 (能看到来访者)
    ✗ 张十三背对门 → 不合理 (看不到来访者)

□ 动线合理性
  - 人能正常走动?
  - 门能正常开关?
  - 没有阻塞?
  
  示例:
    ✓ 门到桌子有3米距离 → 合理 (足够走动)
    ✗ 桌子紧贴门 → 不合理 (门打不开!)

□ 视角合理性
  - 从指定视角能看到应该看到的元素?
  - 视角和布局一致?
  
  示例:
    西向东视角:
      ✓ 应该看到: 东墙、门、桌子、椅子、张十三
      ✗ 不应该看到: 西墙 (在相机后面)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  步骤5: 视角转换 (Viewpoint Transformation)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

目的:
  从指定视角"看"空间

必须明确:
  1. 相机位置 (X, Y, Z)
  2. 相机朝向 (看向哪里)
  3. 相机高度 (eye-level, high angle, low angle)
  4. 视野范围 (能看到什么,看不到什么)

示例 - 你的清代办公室西向东视角:
  
  相机位置:
    X = -2m (西侧,房间中后部)
    Y = 0m (南北居中)
    Z = 1.6m (人眼高度)
  
  相机朝向:
    向东看 (朝向+X方向)
  
  视野范围:
    能看到:
      ✓ 东墙 (正前方)
      ✓ 门 (东墙上)
      ✓ 办公桌 (中间偏右)
      ✓ 椅子 (桌子后面)
      ✓ 张十三 (坐在椅子上)
      ✓ 南墙 (右侧)
      ✓ 北墙 (左侧)
    
    看不到:
      ✗ 西墙 (在相机后面)
      ✗ 桌子背面 (被桌子挡住)

  透视关系:
    - 东墙最远,看起来最小
    - 桌子在中景
    - 相机附近的物体(如果有)在前景,最大

关键技巧:
  ✓ 在脑海中"站"在相机位置
  ✓ 想象你能看到什么
  ✓ 画视角草图
  ✓ 检查透视关系

🏗️ 第三部分: 实战方法论
方法1: 平面图法 (Top-Down Method)
适用场景:
  - 室内场景
  - 建筑布局
  - 城市规划
  - 任何需要精确空间关系的场景

步骤:

1. 画平面图
   ┌─────────────────────┐
   │         北          │
   │                     │
   │  窗                 │
   │  ↓                  │
   │                     │
   │                     │
   │         桌          │
   │         ■          │
   │         椅          │
   │         ●          │
   │                     │
   │                     │
   │         门          │
   │         ↑          │
   │         东          │
   └─────────────────────┘
         西

2. 标注尺寸
   - 房间: 8m(东西) x 6m(南北)
   - 门: 1.2m宽
   - 桌: 1.5m宽 x 0.8m深
   - 门到桌: 3m

3. 标注坐标
   - 门: (3, 0)
   - 桌: (0, 1)
   - 椅: (0, 1.5)

4. 确定视角
   - 相机位置: (-2, 0, 1.6)
   - 朝向: 向东

5. 推导可见元素
   从(-2, 0)向东看:
     ✓ 能看到门(3, 0) - 正前方
     ✓ 能看到桌(0, 1) - 中间偏右
     ✓ 能看到椅(0, 1.5) - 桌后面

优势:
  ✓ 直观
  ✓ 精确
  ✓ 容易发现问题
  ✓ 容易沟通

工具:
  - 纸笔 (最快!)
  - 在线工具: Floorplanner, RoomSketcher
  - 专业软件: AutoCAD, SketchUp

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法2: 剖面图法 (Section Method)
适用场景:
  - 需要考虑高度的场景
  - 多层空间
  - 复杂的垂直关系

步骤:

1. 画剖面图 (侧视图)

   从南向北看的剖面:
   
   3.5m ┌─────────────────┐ 天花板
        │                 │
        │                 │
   2.2m │     ┌──┐       │ 门高
        │     │门│       │
   1.6m │  ●  └──┘       │ 人眼高度
        │  人             │
   0.8m │  ═══           │ 桌面高度
        │  桌             │
     0m └─────────────────┘ 地面
        西              东

2. 标注高度
   - 天花板: 3.5m
   - 门: 2.2m高
   - 人眼: 1.6m (站立)
   - 桌面: 0.8m

3. 确定视线
   - 人眼高度1.6m
   - 能看到桌面(0.8m) - 俯视
   - 能看到门(2.2m) - 仰视上部

优势:
  ✓ 清楚显示高度关系
  ✓ 容易检查视线遮挡
  ✓ 适合复杂垂直空间

方法3: 3D心智模型法 (3D Mental Model)
适用场景:
  - 快速推理
  - 简单场景
  - 有经验后的快速验证

步骤:

1. 在脑海中构建3D模型
   - 想象一个透明的盒子(房间)
   - 把元素一个个"放"进去
   - 想象自己"走"进这个空间

2. 从不同角度"看"
   - 从门口看进去
   - 从窗户看进去
   - 从上方俯视
   - 从人物视角看

3. 检查合理性
   - 尺寸对吗?
   - 位置对吗?
   - 能走动吗?
   - 视角对吗?

技巧:
  ✓ 闭上眼睛想象
  ✓ 用手比划
  ✓ 想象自己在空间中移动
  ✓ 从多个角度验证

训练方法:
  - 观察真实空间,然后闭眼回忆
  - 看平面图,想象3D空间
  - 玩空间类游戏(如Minecraft)
  - 练习画透视图

🎨 第四部分: 常见空间类型的推理策略
类型1: 室内空间 (Interior Space)
关键要素:
  1. 房间形状和尺寸
  2. 墙壁、地面、天花板
  3. 门窗位置
  4. 家具布局
  5. 人物位置和动线
  6. 光源位置

推理流程:

步骤1: 确定房间"容器"
  - 形状: 矩形? L型? 不规则?
  - 尺寸: 长x宽x高
  - 方向: 哪面是东西南北?

步骤2: 放置"固定元素"
  - 门: 哪面墙? 多宽? 向哪开?
  - 窗: 哪面墙? 多大?
  - 壁炉、柱子等固定结构

步骤3: 放置"家具"
  - 从大到小
  - 考虑功能分区
  - 考虑动线
  - 考虑视线

步骤4: 放置"人物"
  - 位置合理?
  - 姿态合理?
  - 能看到什么?

步骤5: 确定"视角"
  - 相机在哪?
  - 看向哪?
  - 能看到什么?

常见错误:
  ❌ 家具尺寸不合理
  ❌ 家具摆放不符合使用习惯
  ❌ 门窗位置和视角矛盾
  ❌ 人物位置不合理
  ❌ 动线被阻塞

检查清单:
  □ 所有家具都能放进房间?
  □ 家具之间有足够间距?
  □ 门能正常开关?
  □ 人能正常走动?
  □ 从视角能看到应该看到的元素?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
类型2: 产品摄影空间 (Product Photography Space)
关键要素:
  1. 产品位置和角度
  2. 背景
  3. 光源位置
  4. 相机位置和角度
  5. 道具位置

推理流程:

步骤1: 确定产品位置
  - 画面中心? 偏左? 偏右?
  - 高度: 桌面? 悬浮? 手持?
  - 角度: 正面? 侧面? 俯视?

步骤2: 确定相机位置
  - 距离产品多远?
  - 高度: 平视? 俯视? 仰视?
  - 角度: 正面? 45度? 侧面?

步骤3: 确定光源位置
  - 主光: 哪个方向? 多远? 多高?
  - 补光: 哪个方向?
  - 轮廓光: 哪个方向?

步骤4: 确定背景
  - 纯色背景: 距离产品多远?
  - 场景背景: 什么场景? 多远?

步骤5: 放置道具(如果有)
  - 位置: 前景? 背景? 旁边?
  - 作用: 衬托? 说明? 装饰?

空间关系图:

  俯视图:
  
        主光源
          ↓
    背景 ─ 产品 ─ 相机
          ↑
        补光源

  侧视图:
  
    轮廓光
       ↘
        产品 ← 相机
       ↗
    主光源

常见错误:
  ❌ 光源位置和阴影方向矛盾
  ❌ 相机角度和产品角度不匹配
  ❌ 背景距离不合理导致阴影问题
  ❌ 道具抢了产品的风头

检查清单:
  □ 产品是画面主体?
  □ 光源位置合理?
  □ 阴影方向一致?
  □ 相机角度能展示产品特点?
  □ 背景不抢眼?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
类型3: 人物场景空间 (Character Scene Space)
关键要素:
  1. 人物位置和姿态
  2. 环境元素位置
  3. 人物和环境的关系
  4. 相机位置和角度
  5. 景深和焦点

推理流程:

步骤1: 确定人物位置
  - 在环境中的位置
  - 站? 坐? 躺?
  - 朝向哪里?

步骤2: 确定环境元素
  - 人物周围有什么?
  - 人物在和什么互动?
  - 前景、中景、背景各有什么?

步骤3: 确定人物和环境的关系
  - 人物在环境中的比例
  - 人物和物体的距离
  - 人物能触及什么?

步骤4: 确定相机位置
  - 距离人物多远?
  - 高度: 平视? 俯视? 仰视?
  - 角度: 正面? 侧面? 背面?

步骤5: 确定构图
  - 人物在画面中的位置
  - 前景、中景、背景的层次
  - 视觉引导线

空间层次:

  前景 ─── 中景(人物) ─── 背景
    │         │            │
  道具A     主体人物      环境
  虚化      清晰         虚化

常见错误:
  ❌ 人物尺寸和环境不匹配
  ❌ 人物姿态和环境不符
  ❌ 人物"悬浮"(没有接地)
  ❌ 人物和物体的互动不合理
  ❌ 视角和人物位置矛盾

检查清单:
  □ 人物尺寸合理?
  □ 人物姿态符合环境?
  □ 人物脚踩在地面上?
  □ 人物和物体的互动合理?
  □ 从视角能看到人物的关键特征?

🔧 第五部分: 高级技巧
技巧1: 矛盾检测 (Contradiction Detection)
目的:
  发现用户描述中的空间矛盾

常见矛盾类型:

类型1: 位置矛盾
  示例:
    用户说: "门在东墙,从西向东看能看到门后面的走廊"
    矛盾: 从西向东看,门在正前方,看不到门后面!
  
  解决:
    - 指出矛盾
    - 提供正确的描述
    - 或询问用户真实意图

类型2: 尺寸矛盾
  示例:
    用户说: "5x5米的房间,放一张4米的桌子和两个2米的柜子"
    矛盾: 4+2+2=8米 > 5米,放不下!
  
  解决:
    - 计算实际尺寸
    - 指出问题
    - 建议合理尺寸

类型3: 视角矛盾
  示例:
    用户说: "从北向南看,能看到北墙上的画"
    矛盾: 从北向南看,北墙在身后,看不到!
  
  解决:
    - 画视角图
    - 说明可见范围
    - 修正描述

类型4: 功能矛盾
  示例:
    用户说: "椅子面向墙,人坐在椅子上办公"
    矛盾: 面向墙怎么办公?
  
  解决:
    - 指出不合理
    - 建议合理朝向

检测方法:
  1. 画平面图
  2. 标注所有元素
  3. 检查是否重叠
  4. 检查是否超出边界
  5. 从视角位置"看"一遍
  6. 检查功能合理性

技巧2: 信息补全 (Information Completion)
目的:
  补充用户遗漏的空间信息

常见遗漏:

遗漏1: 尺寸信息
  用户说: "一个办公室,有桌子和椅子"
  遗漏: 房间多大? 桌子多大?
  
  补全策略:
    - 使用标准尺寸
    - 办公室: 4x5米 (小), 6x8米 (中), 10x12米 (大)
    - 办公桌: 1.2-1.6米宽, 0.7-0.8米深
    - 椅子: 0.5-0.6米宽

遗漏2: 位置信息
  用户说: "房间里有门和窗"
  遗漏: 门在哪面墙? 窗在哪面墙?
  
  补全策略:
    - 使用常见布局
    - 门通常在入口墙
    - 窗通常在外墙
    - 询问用户

遗漏3: 朝向信息
  用户说: "人坐在椅子上"
  遗漏: 面向哪里?
  
  补全策略:
    - 根据功能推理
    - 办公: 面向桌子/门
    - 休息: 面向窗/电视
    - 询问用户

遗漏4: 视角信息
  用户说: "生成一个办公室"
  遗漏: 从哪个角度看?
  
  补全策略:
    - 使用常见视角
    - 室内: 门口视角, 对角线视角
    - 产品: 45度角, 正面
    - 询问用户

补全原则:
  ✓ 优先使用标准/常见值
  ✓ 优先使用符合功能的值
  ✓ 不确定时询问用户
  ✓ 在提示词中说明是补全的

技巧4: 人体工程学验证 (Ergonomic Validation)
目的:
  确保空间符合人体使用需求

关键尺寸:

人体尺寸:
  - 站立高度: 1.6-1.8米 (成人)
  - 坐姿高度: 0.9-1.0米 (椅面到头顶)
  - 肩宽: 0.45-0.55米
  - 伸手范围: 0.6-0.8米

家具尺寸:
  - 桌面高度: 0.7-0.8米
  - 椅面高度: 0.4-0.5米
  - 椅背高度: 0.8-1.0米
  - 床高: 0.4-0.5米

空间尺寸:
  - 走道宽度: 0.8-1.0米 (单人), 1.2-1.5米 (双人)
  - 门宽: 0.8-1.0米 (标准), 1.2米 (双开)
  - 门高: 2.0-2.2米
  - 天花板高: 2.6-3.0米 (住宅), 3.5-4.0米 (办公)

使用空间:
  - 椅子后方: 0.6-0.8米 (起身空间)
  - 桌前空间: 0.6-0.8米 (腿部空间)
  - 床边空间: 0.6米 (最小)

检查清单:
  □ 家具高度适合人体?
  □ 椅子和桌子高度匹配?
  □ 有足够的使用空间?
  □ 走道宽度足够?
  □ 门窗尺寸合理?

常见错误:
  ❌ 桌子太高或太低
  ❌ 椅子后方空间不足
  ❌ 走道太窄
  ❌ 天花板太低压抑

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   不是大概,是精确的坐标和尺寸

3. 5个核心步骤
   坐标系 → 边界 → 元素 → 验证 → 视角

4. 3个关键方法
   平面图法 + 剖面图法 + 3D心智模型

5. 4个高级技巧
   矛盾检测 + 信息补全 + 多视角验证 + 人体工程学

6. 持续训练
   观察 + 游戏 + 画图 + 实战

   AI不懂空间,你懂!
