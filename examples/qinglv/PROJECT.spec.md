# Qinglv — 项目 Spec

## 项目概述

一个小红书图文故事系列，讲述一对个性迥异的情侣（ENFJ 小茉 vs INTJ 小源）在日常生活中的差异、摩擦与和解。

**核心设定：**
- 主角：小茉（ENFJ，爱出门，元气少女）和小源（INTJ，超级宅，高冷理性）
- 故事类型：伙伴之爱（Buddy Love），短篇图文，小红书格式
- 叙事结构：简化四幕（开场→冲突→转折→结局）
- 控制性理念：当差异化的情侣愿意在抱怨中相互理解并积极寻找平衡点时，他们就能将潜在的冲突转化为增进亲密和共同成长的契机

**画风定义（重要）：**
- 风格：手绘插画风，韩国风格
- 质感：水粉 + 彩铅，纹理纸张，可见笔触，略带不完美的手绘感
- 色彩：整体暖色调，营造温馨积极氛围
- 构图：人物与环境结合，展现情侣互动与场景
- 光线：强调暖光，增强温馨感
- 禁止：超写实、3D渲染、过度光滑的数字感

## 角色设定

### 小茉（女主）
- 年龄：30 岁
- MBTI：ENFJ
- 外形：中长发、齐刘海，单眼皮，开朗爱笑，元气满满
- 标志动作：主动牵小源的手
- 性格：超爱出门，做决策靠感受，社交=充电，独处=孤独，有计划的行动派
- 核心需求：被理解、被陪伴、情感共鸣、新鲜感
- 弱点：过于依赖外部刺激，容易委屈
- 成长弧线：从渴望外部探索，到学会欣赏宅家宁静，更有效表达需求

### 小源（男主）
- 年龄：30 岁
- MBTI：INTJ
- 外形：斜刘海短发，高冷中带宠溺，左眉有淡淡疤痕，戴旧款数字手表
- 标志动作：手搭在小茉肩膀上
- 性格：超级宅，做决策靠逻辑，社交=消耗，独处=充电，启动困难但出门后其实挺开心
- 核心需求：独处空间、被尊重、逻辑自洽、稳定节奏
- 弱点：不擅长表达情感，无法共情伴侣感受
- 成长弧线：从固守宅家习惯，到愿意为伴侣踏出舒适区，尝试理解情感需求
