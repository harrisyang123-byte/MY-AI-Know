## 场景零：S01 · 威海卫（开场画面）

**节拍：节拍1（开场画面）| 情绪：压迫，预兆，全片情感底色**
**时长：45-60秒 | 镜头数：9-11 | 剪辑节奏：极快（3-5秒/镜头）**

**第一步：确认输入**
1895年黎明，威海卫海域外。无人物出场（纯环境+象征）。
在整体色彩框架中：冷/极低饱和+炮火微红，接近黑场。

**第二步：色彩定调**
- **主色**：深海蓝灰（RGB: 25,30,35，接近黑场）
- **辅色**：炮火微红（RGB: 120,40,30，远处闪烁，不稳定）+ 鹰翼金色（RGB: 255,180,50，逆光燃烧，极短暂）
- **禁用色**：任何稳定暖色——这个空间里没有希望

**第三步：置景四要素**

【布景·威海湾全景】
叙事功能：情绪氛围（空间在第一个镜头就告诉观众"这里是坟墓"）
空间权力：无人物，但权力结构清晰——北洋舰队被困在港内（猎物），日军在外围合围（猎人），老鹰在空中盘旋（旁观者=李鸿章的隐喻）
前景：无（或海面波纹的微光）
中景：北洋舰队（定远舰、镇远舰等），炮口沉默，黄龙旗垂落
背景：日军舰船轮廓（灰色剪影），天际线压低，云层厚重
空间线条：
- 水平线：海面平静如镜（死亡般的平静）→ 占画面下方1/3
- 垂直线：舰船桅杆、炮管 → 权威但被困住
- 斜线：老鹰俯冲轨迹 → 不稳定的自由

【布景·老鹰特写系列】
叙事功能：象征（老鹰=李鸿章，全片最重要的视觉隐喻）
空间权力：老鹰在画面中很小（对比巨大空间），但它是最醒目的存在
关键帧：
1. 翅膀逆光（特写）：羽毛边缘燃烧，金色
2. 俯冲穿云（远景→中远景）：速度感，急速下坠
3. 盘旋（中远景）：舰队上方，不落下
4. 收翅下坠（中远景→大远景）：突然收翅，急速坠落
5. 硬切黑场：坠落未完成，断裂感

【道具】
- 背景道具：
  - 北洋舰队：深灰色船体，有锈迹，炮口朝向港口内部（无法向外开火）
  - 日军包围圈：更深的灰色剪影，数量优势明显，炮口朝向港内
  - 海面：近乎黑色的镜面，只有微弱反光
- 手持道具：无（本场无人物）
- 象征道具：
  - **老鹰（金雕）**：翼展2米以上，羽毛深褐色，翼尖和尾羽有金色光泽（逆光时可见）
    叙事功能：全片最重要的视觉隐喻 = 李鸿章
    - 盘旋 = 被困住的清醒者（知道局势，但无力改变）
    - 不落下 = 还在坚持，还在观察
    - 收翅下坠 = 最终的选择（预告马关遇刺，预告牺牲）
    - 逆光燃烧的鹰翼 = 尊严的预告（即使坠落，也是有光的坠落）

【灯光·自然光（黎明前）】
光质：硬光（黎明前的冷光）→ 揭示，无处可藏，但这种揭示是残酷的
方向：顶光+逆光（从云层缝隙透出的光打在鹰翼上）
来源：黎明前的天光，6500K，极冷
调性：极低调 | 主补比：几乎无补光（只有一个光源：天光）
特殊处理：
- 炮火的间歇性暖光是唯一的"人造光源"，但不稳定，像垂死的心跳
- 鹰翼逆光是全场最亮的一刻（约1-2秒），然后立刻回到压抑的冷调
- 海面的反光极弱，几乎是黑色的镜子——"平静比炮声更令人不安"

【服装与化妆】
- 无人物（本场纯环境+象征）

**第四步：材质情感温度校验**
- 深海蓝灰（金属/深海）→ 冰冷/死亡 ✓
- 炮火微红（火焰/血）→ 暴力/毁灭 ✓
- 鹰翼金色（阳光/羽毛）→ 尊严/希望（但极短暂）✓
- 舰船锈铁（锈铁）→ 衰败/时间流逝 ✓

**第五步：声音设计（先行，与视觉同步）**

```
时间轴：0s ---- 5s ---- 8s ---- 15s ---- 25s ---- 35s ---- 40s ---- 45s
画面：黑场  鹰翅膀   俯冲     展开      盘旋      收翅      黑场     字幕
声音：海浪  海浪+风  风渐起   海浪+炮声  交替     寂静    寂静+hum  字幕音
强度：低    低       中       中        不安      突停     极轻     淡入淡出
```

**详细音轨：**
- **0-5s（黑场）**：海浪声（低频，像远处的引擎空转）+ 极低频嗡鸣（20Hz以下，身体能感受到但耳朵听不清）→ 建立不安底色
- **5-8s（亮起）**：海浪继续，无变化 → 画面亮起但声音不变，制造"视觉冲击但听觉延续"的错位感
- **8-15s（俯冲）**：风声渐起（从无到有，渐进式，不是突然爆发）→ 速度感的听觉化
- **15-25s（展开）**：炮声隐约加入（很远，像心跳，不规则节奏：... ... . ...）→ 信息释放的声音版
- **25-35s（盘旋）**：海浪声+炮声交替主导（不安的节奏，观众不知道下一个声音会是什么）→ 情绪积累
- **35-40s（收翅）**：风声骤停（突然的寂静，像呼吸被掐住）→ 预警
- **40-45s（黑场）**：绝对寂静持续1秒（让观众喘不过气），然后字幕淡入音（低频 hum，50Hz，单音持续）
- **字幕**："1895年·威海卫"（白字，衬线体，淡入淡出2秒）

**第六步：技术参数（供AI绘图/视频生成参考）**

| 参数 | 数值 | 说明 |
|------|------|------|
| 画面比例 | 2.39:1 (Cinemascope) | 史诗宽银幕，强调空间的压迫感 |
| 帧率 | 24fps | 标准电影帧率 |
| 色彩空间 | Log/C-Log | 为后期调色留足空间 |
| 景深 | 深景深（全景）/ 浅景深（特写） | 全景清晰展示空间关系；特写聚焦鹰翼细节 |
| 颗粒度 | 中高（模拟胶片质感） | 增加历史感和粗糙感 |
| 透光率 | 低（整体偏暗） | 接近黑场的基调 |

## 场景一：S02 · 总理衙门·审讯

**节拍：节拍2-3 | 情绪：审判，压迫，叙事引擎启动**

**第一步：确认输入**
1895年冬夜，总理海军事务衙门内室。现实线，全片最冷的色调。
在整体色彩框架中：冷/低饱和，接近去色。

**第二步：色彩定调**
主色：青灰（冷灰，接近去色）
辅色：烛火琥珀（单一暖点，脆弱）
禁用色：任何高饱和暖色——这个空间不允许希望存在

**第三步：置景四要素**

【布景】
叙事功能：情绪氛围（空间在对话开始前就告诉观众"这里是审判"）
空间权力：张十三居中偏后，被奏折包围；李莲英坐在对面，刀放在桌上——刀比人更有权力
前景：奏折堆成山，像一堵纸墙，把张十三困在里面
中景：深色榆木长案，漆面磨损，案上奏折、砚台、烛台
背景：封闭，格扇门关着，高丽纸泛黄，透进来的光是冷的
空间线条：垂直线主导（柱子、格扇门的直棂）→ 压迫、审判

道具：
- 背景道具：格扇门（朱红漆褪色剥落，露出木胎）、青灰色砖墙、横梁
- 手持道具：
  - 奏折：米黄色宣纸，墨迹密集，边角磨损，有折痕——张十三把它压平，像在压住什么
  - 烛台：黄铜制，氧化发黑，蜡泪凝固在台身，烛火摇曳
- 象征道具：
  - 遏必隆刀：刀鞘深褐色皮革，铜制刀环，刀身未出鞘——威胁藏在礼貌里
  - 桌上的物件（望远镜，此刻不给观众看）：黄铜，西洋纹路，被奏折压着

【灯光】
光质：软光（蜡烛散射）→ 隐藏，真相在阴影里
方向：侧光+底光（烛火从桌面向上打）→ 人物面部有阴影，道德模糊
来源：蜡烛，2800K，暖黄但微弱
调性：低调，高对比 | 主补比：8:1，黑色电影风格
特殊处理：烛火是画面里唯一的暖色，但它很小，很脆弱——随时可能熄灭

【服装与化妆】
- 张十三：藏青色厚棉袍，粗麻布内衬，腰带松弛——他已经在这里坐了很久
  与环境关系：融合（藏青与青灰背景接近）→ 他被环境吞没，失去主体性
- 李莲英：深紫色官服，丝绸质地，有光泽——他是这个空间里唯一有光泽的东西
  与环境关系：对比（丝绸光泽 vs 粗糙背景）→ 他有主体性，他是入侵者

**第四步：材质情感温度校验**
- 榆木长案（温暖/人性）→ 与场景情绪冲突？不冲突，这是张十三的空间，木质带来一点人性温度
- 遏必隆刀（丝绸/抛光金属）→ 权力/危险 ✓
- 蜡烛（蜡质/脆弱）→ 短暂/记忆 ✓
- 奏折（旧纸）→ 脆弱/记忆 ✓

## 场景二：S03+S09 · 江宁贡院初遇 + 油菜花田

**节拍：节拍3-4 | 情绪：希望，知己，承诺（全片最亮）**

**第一步：确认输入**
1844年黄昏，江宁贡院考棚外；1847年正午，北京郊外油菜花田。
在整体色彩框架中：暖/高饱和，全片最亮，与其他所有场景形成最大对比。

**第二步：色彩定调**
主色：金黄（油菜花）+ 暖橙（夕阳）
辅色：青衫的靛蓝（人物与环境的对比）
禁用色：冷色、低饱和——这两场戏是全片的情感储备，不允许任何阴影

**第三步：置景四要素**

【布景·贡院考棚外】
叙事功能：决定人物（这个空间揭示两人是谁——一个在人群中心，一个在边缘）
空间权力：李鸿章居中，被人群簇拥；张十三在边缘，站在阴影里
前景：老槐树的枝干，制造自然框架
中景：考棚长廊，夕阳把廊柱拉出长影，像琴弦
背景：开放，天空，夕阳
空间线条：水平线（廊柱的影子）+ 斜线（夕阳角度）→ 动态，充满可能性

道具：
- 背景道具：考棚木结构（深褐色，有使用痕迹）、青砖地面
- 手持道具：
  - 李鸿章的折扇：象牙白扇骨，深蓝色扇面，挥动时有风声
  - 张十三的砚台和笔：黑色砚台，毛笔搁在砚台上，整齐
- 象征道具：无（这场戏不需要象征道具，情感本身就够了）

【布景·油菜花田】
叙事功能：情绪氛围（全片唯一的纯粹快乐，为后续所有沉重蓄力）
空间权力：两人在开阔的花田里，没有任何压迫——这是全片唯一一次他们是自由的
前景：油菜花，金黄，高饱和，随风摇曳
中景：两匹白马，青衫的两个人
背景：开放，天空，无边界
空间线条：斜线（马背上的动态）→ 全片唯一的"好的斜线"，自由，不是不稳定

道具：
- 象征道具：
  - 白马：纯白，无杂色，马鬃飞扬——青春，自由，这个意象在全片不再出现
  - 青衫：靛蓝色，轻薄棉布，随风飘动——他们还是书生，还没有被染色

【灯光·贡院】
光质：软光（夕阳散射）→ 温柔，不揭示，只美化
方向：逆光（夕阳从背后打）→ 人物轮廓发光，神秘感，美好感
来源：夕阳，3200K，暖橙
调性：中间调偏高 | 主补比：2:1，柔和

【灯光·油菜花田】
光质：硬光（正午阳光）→ 揭示，清晰，没有阴影
方向：顶光+逆光（李鸿章勒马时逆光）→ 英雄感，不可一世
来源：正午阳光，5500K，中性偏暖
调性：高调 | 主补比：1:1，全片最亮

【服装与化妆】
- 李鸿章（21-24岁）：靛蓝色青衫，轻薄棉布，腰束玉带，发辫飞扬
  与环境关系：对比（靛蓝 vs 金黄花田）→ 人物从环境中突出，有主体性
- 张十三（20-23岁）：同款靛蓝青衫，但更整洁，发髻不乱
  与环境关系：对比（靛蓝 vs 金黄）→ 同样有主体性，但他的整洁与李鸿章的飞扬形成对比

**第四步：材质情感温度校验**
- 油菜花（自然/轻薄）→ 希望/转变 ✓
- 白马（有机/自然）→ 温暖/人性 ✓
- 青衫棉布（棉麻）→ 温暖/人性 ✓
- 玉带（抛光/光滑）→ 权力/欲望——李鸿章的玉带暗示他的野心，即使在最纯粹的时刻 ✓

## 场景三：S10 · 苏州杀降·牛肉沉默

**节拍：节拍9（中点）| 情绪：虚假胜利→道义死亡**

**第一步：确认输入**
1863年深夜，苏州城外淮军大营帅帐。暴雨。
在整体色彩框架中：冷/低饱和，道义死亡，色彩随之死去。

**第二步：色彩定调**
主色：暗红（血色，但不鲜艳，是凝固的暗红）
辅色：烛火琥珀（比审讯场景更昏暗）
禁用色：任何明亮色——这个空间里没有希望

**第三步：置景四要素**

【布景】
叙事功能：反映人物（帐内的混乱外化李鸿章此刻的内心）
空间权力：李鸿章坐在桌前，但他不是主人——那块牛肉才是这场戏的主角
前景：碎裂的盘子（张十三打翻后），碎片散落在地
中景：深色木桌，西洋刀叉（不合身的道具），银票，牛肉
背景：封闭，帐篷布，暴雨声从外面渗进来
空间线条：混乱（碎片、散落的银票）→ 失控，道义崩塌

道具：
- 背景道具：军帐内壁（粗麻布，深褐色，有烟熏痕迹）、油灯（比蜡烛更昏暗）
- 手持道具：
  - 西洋刀叉：银色金属，光泽，与木桌的粗糙形成对比——异物，不属于这里
  - 银票：米黄色纸，厚厚一叠，被拍在桌上
- 象征道具：
  - 半熟牛肉：带血丝，暗红色，在盘子里——后来脏了，被捡起，被放下，没有被吃
    叙事功能：他知道自己做了什么，但他选择继续。牛肉是他道义的具象——脏了，但他没有扔掉，也没有吃掉。
  - 《安民告示》：米黄色宣纸，张十三手里攥着，后来被他带走——他的道义，他带走了

【灯光】
光质：硬光（油灯直射）→ 揭示，无处可藏
方向：侧光（油灯在桌侧）→ 李鸿章脸上有深重阴影，道德模糊
来源：油灯，2600K，比蜡烛更暗更黄
调性：低调，极高对比 | 主补比：8:1+，黑色电影风格
特殊处理：帐外的枪声停止后，帐内的寂静比任何声音都重。灯光在寂静里显得更孤独。

【服装与化妆】
- 李鸿章（40岁）：深灰色军服，有磨损，面容消瘦，胡须，眼神里有戾气
  与环境关系：融合（深灰 vs 深褐帐篷）→ 他被这个空间吞没了
- 张十三（40岁）：深蓝色长衫，整洁，与李鸿章的军服形成对比
  与环境关系：对比（深蓝 vs 深褐）→ 他是这个空间里的异物，他不属于这里

**第四步：材质情感温度校验**
- 粗麻布军帐（锈铁/破布）→ 衰败/压抑 ✓
- 西洋刀叉（不锈钢）→ 冰冷/体制化 ✓
- 牛肉（有机/血腥）→ 暴力/道义死亡 ✓
- 银票（旧纸）→ 脆弱/交易 ✓

## 场景四：S12 · 定远舰·炮台积灰

**节拍：节拍10（坏人逼近）| 情绪：希望→幻灭**

**第一步：确认输入**
1889年，威海卫，定远舰甲板。阳光下的阴影。
在整体色彩框架中：暖/中低饱和——表面是洋务的辉煌，实际是空心的。

**第二步：色彩定调**
主色：钢铁灰（定远舰的主色）
辅色：海蓝（背景）+ 白发（两个老人）
禁用色：高饱和暖色——这里的阳光是假的，是"阳光下的阴影"

**第三步：置景四要素**

【布景】
叙事功能：象征（定远舰是李鸿章半生心血的具象，炮口的灰是体制腐蚀的具象）
空间权力：两个老人站在巨大的钢铁舰船上，人比船小——他们被自己建造的东西压倒了
前景：主炮炮管，黑色钢铁，炮口有积灰
中景：甲板，钢铁，有锈迹，有海盐腐蚀的痕迹
背景：开放，海面，天空——但这个开放是虚假的，因为他们知道这艘船撑不住
空间线条：水平线（甲板、海平线）→ 平静，但这种平静是死亡前的平静

道具：
- 背景道具：定远舰甲板（钢铁，有锈迹，有海盐腐蚀）、炮台、缆绳
- 手持道具：无（这场戏的道具是炮口的灰，它不被手持，它被手指捻出）
- 象征道具：
  - 炮口的黑灰：张十三的手指捻出，举起来，让李鸿章看
    叙事功能：这把灰是全片最重要的道具之一——它是体制腐蚀的证据，是"里子朽了"的具象
  - 船舷：李鸿章攥住，指节泛白，然后松开——他撑着，但他知道

【灯光】
光质：硬光（正午阳光）→ 揭示，无处可藏，阳光把一切都照得清清楚楚
方向：顶光（正午）→ 压迫，审判感
来源：正午阳光，5500K，中性
调性：中间调，但阴影很深（硬光制造的对比）| 主补比：4:1
特殊处理：阳光照在炮管上，把积灰照得清晰可见——这是全片最残忍的光，它让一切无处遁形

【服装与化妆】
- 李鸿章（65岁）：深蓝色官服，黄马褂（此时还干净），白发，背微驼
  与环境关系：对比（黄马褂 vs 钢铁灰甲板）→ 他还有主体性，但黄马褂在钢铁上显得孤独
- 张十三（65岁）：深灰色长衫，白发，腰杆笔直
  与环境关系：融合（深灰 vs 钢铁灰）→ 他和这艘船融为一体，他是见证者

**第四步：材质情感温度校验**
- 钢铁甲板（不锈钢/混凝土）→ 冰冷/体制化 ✓
- 锈迹（锈铁）→ 衰败/无力 ✓
- 黄马褂（丝绸）→ 权力/虚伪繁荣——黄马褂的光泽在锈铁上是一个讽刺 ✓
- 炮口的灰（锈铁/剥落）→ 衰败/时间流逝 ✓

## 场景七点五：S17 · 张十三的笑（v6 强化专项）

> v6 关键强化：张十三第一次笑了——全片唯一一次，必须有足够的情感重量

**位置**：S17场景的最后10-15秒（回答完"忠臣还是奸臣"之后）
**镜头**：特写（第8个，也是最后一个特写）| 时长：8-10秒 | 运动：固定

### 面部细节设计

**笑的解剖学（演员指导）：**

```
时间轴：0s ---- 2s ---- 4s ---- 6s ---- 8s ---- 10s
部位：  静止   嘴角    眼睛    眼角    保持     微微收敛
状态：  无表情 笑意浮现 眼睑下垂 皱纹展开 稳定    笑意未消
```

**逐部位分解：**

1. **嘴角（0-3s）**：
   - 先是右嘴角微微上扬（不经意的）
   - 然后左嘴角跟上（对称，但是轻的）
   - 上扬幅度：约3-5mm（不是大笑，是"嘴角有了弧度"）
   - 唇形：保持自然闭合，不露齿（露齿会破坏克制的基调）

2. **眼睛（3-6s）**：
   - 眼睑微微下垂（放松的标志，不是眯眼）
   - 眼角出现鱼尾纹（这是关键——皱纹是时间的痕迹，笑让这些痕迹变得温柔）
   - 瞳孔：略微失焦（不是在看张念海，是在看很远的地方——或者在回忆）
   - 眉毛：不动（额头保持平静，只有眼睛和嘴在参与这个笑）

3. **整体（6-10s）**：
   - 头部：微微向右倾斜2-3度（松弛的姿态）
   - 肩膀：下沉1cm（卸下了最后一点重量）
   - 呼吸：可以看到胸口有极轻微的起伏（像一个长长的呼气）
   - 这个笑保持到镜头结束，不收敛——它不是一个瞬间的表情，是一种状态

### 灯光与笑的互动

**光线分布：**
- 主光（灯笼，2800K）：从左侧45度角打过来，照亮他的左半边脸
  - 照亮的部分：左脸颊、左眼角、左嘴角
  - 效果：笑的纹理（鱼尾纹、嘴角弧度）被光照亮，清晰可见
- 补光（雪地反光，7500K冷白）：从右下方补入
  - 功能：填充右侧阴影，但不抢主光
  - 效果：冷暖交织——他的笑既有温度（灯笼），又有清澈（雪地）
- 轮廓光（如果有窗外微光）：从后方勾勒头发和肩膀边缘
  - 效果：把他从背景中分离出来，让他成为画面的唯一主体

**关键光影时刻：**
- 当眼角鱼尾纹展开时，灯光刚好照在皱纹的沟壑里——光在皱纹里"定居"，这让皱纹看起来不像衰老的痕迹，像智慧的印记
- 这是一个需要在现场调整的光位——摄影师必须确保：当演员笑的时候，光落在正确的地方

### 声音与笑的配合

**选项A：无声的笑（推荐）**
- 完全没有笑声
- 只有呼吸声（极轻，像白色的噪音）
- 背景音：雪落无声（真的是无声，或者只有极远的狗吠一声，然后又没了）
- 效果：这个笑是完全内在的——他不分享给任何人，包括观众。我们只是看见了。

**选项B：极轻的叹息式笑**
- 不是"哈"的声音，是气息从鼻腔出来的声音（像"哼"但没有声带振动）
- 音量：-30dB左右（几乎听不见）
- 时机：在嘴角上扬的同时，气息出来
- 效果：这个笑有一点点声音，但与其说是笑，不如说是"终于可以呼吸了"

**推荐选项A**：因为无声更符合"克制内敛"的全片表演风格，而且无声的笑更有力量——他不需要用声音来证明这个笑的真实性。

### 后续：叠化到油菜花田

**叠化时机**：笑的镜头结束后，保持1-2秒（让观众消化），然后开始叠化

**叠化过程（约3秒）：**
- 0-1s：张十三的脸开始变透明，雪景开始渗透
- 1-2s：张十三的脸消失，雪地充满画面，然后油菜花的金色开始从画面边缘渗入
- 2-3s：雪地完全消失，油菜花田充满画面，年轻的李鸿章（24岁左右）策马回头，笑
- 3s后：定格1-2秒，然后暗场

**李鸿章的笑（终幕）vs 张十三的笑（S17）：**
- 张十三的笑：释然，疲惫，温暖，内在的
- 李鸿章的笑：明亮，自信，英雄式的，外在的
- 两个笑的对比：见证者的笑 vs 被见证者的笑 —— 他们终于在不同的时空里，以同样的表情回应彼此
