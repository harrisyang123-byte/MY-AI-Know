## 视觉决策框架（film-visual-language 模式一）

### 全片视觉基调

**混合策略，高亲和为主，关键节拍用高对比打破。**

日常段落（回忆线青年期）高亲和——色调统一、运动流畅、空间开阔。
随弧光推进，亲和度逐渐降低，对比度逐渐升高。
马关遇刺达到全片最高对比。

---

### 七组件基准

**空间**
- 默认：深空间，人物在空间中有位置感
- 转变触发：李鸿章独处场景（S14、S10）→ 浅空间，人物被环境吞没
- 现实线审讯 → 浅空间，烛光把空间压缩成一个小圈
- S01开场 → 极深空间（威海湾全景），老鹰在巨大空间里显得渺小

**线条**
- 主导方向：垂直线（权威、压迫、脊梁）
- 李鸿章的腰杆笔直是全片最重要的视觉符号
- 转变触发：李鸿章开始弯腰（洋务晚期、马关谈判）→ 引入斜线，权威感开始瓦解
- 例外：油菜花田的斜线是全片唯一的"好的斜线"——动态，自由
- S01老鹰俯冲 → 斜线（下坠轨迹）+ 水平线（海面平静）→ 不稳定的自由 vs 死亡般的平静

**色调**
- 默认：低调（暗部为主）
- 例外：油菜花田（高调，全片唯一）→ 与其他所有场景形成最大对比
- 洋务蒙太奇 → 中间调偏暖，全片第二亮
- S01威海卫 → 极低调（接近黑场），只有炮火和鹰翼有微弱光

**色彩**

| 节拍 | 色彩状态 | 叙事理由 |
|------|---------|---------|
| 节拍1 开场画面(S01) | 冷/极低饱和+炮火微红 | 压迫底色，老鹰的金色翼尖是唯一暖点（预告终幕金色） |
| 节拍2 审讯(S02) | 冷/低饱和，接近去色 | 审判空间，不允许希望存在 |
| 节拍3 贡院初遇(S03) | 暖/中饱和 | 希望的开始 |
| 节拍4 油菜花田(S09) | 暖/高饱和 | 全片情感顶点的预告，最纯粹的时刻（铺垫高潮，非催化剂） |
| 节拍9 苏州杀降(S10) | 冷/低饱和 | 道义死亡，色彩随之死去 |
| 节拍8 洋务蒙太奇(S11) | 暖/中低饱和 | 真实的进步，但根基是空的（冲突：李鸿章 vs 时间） |
| 节拍10 定远舰(S12) | 暖/中低饱和(假暖) | 表面辉煌，实际空心 |
| 节拍11 一败涂地(S14) | 冷/极低饱和 | 接近去色，绝望 |
| 节拍12 灵魂黑夜(S13+S15) | 冷/极低饱和→碎裂瞬间闪光 | 张十三的内崩塌，不是外部冲击触发 |
| 节拍14 马关遇刺(S16) | 冷底+血红高饱和 | 唯一的暖色是血，尊严的代价 |
| 节拍14' 李莲英离开(S16) | 冷/低饱和+门缝暖光 | 他带着秘密离去，门缝的光是他选择的温度 |
| 节拍15 终幕(S17) | 冷/低饱和+灯笼暖点 | 释然，不是绝望；张十三第一次笑了 |

**运动**
- 默认：静止或缓慢跟随，摄影机是旁观者
- 运动的叙事意图：摄影机开始运动 = 情绪在积累，即将爆发
- S01老鹰俯冲 → 急速跟随+下摇（鹰视角），然后突然静止（坠落后黑场）
- 油菜花田：跟随运动，自由感
- 马关遇刺：慢镜头+静音前置，时间被拉伸
- S15望远镜碎裂 → 静止到极致，碎裂瞬间无运动（让碎裂本身成为唯一的运动）
- 手持：只在苏州杀降的混乱段落，其余全部稳定

**节奏**
- 默认：慢，镜头平均时长 8-15 秒
- 加速触发：战场（S08）、马关交叉剪辑（A线 vs B线）
- 减速触发：独处场景（S14）、终幕雪夜、S15碎裂前
- S01开场：极快剪辑（平均3-5秒），建立压迫感和信息密度

### 视觉动机

**动机一：望远镜**
- S05 宣南会馆：第一次出现，张十三拒绝接受 → 两人世界观的裂缝
- S14 望远镜独处：李鸿章举起，镜片模糊 → 清醒者看不见前路
- S15 北洋覆没：从张十三手中滑落，碎裂 → 一个时代的终结（v6修正：由张十三讲述中撑不住而滑落，非急报触发）

**动机二：脊背/腰杆**
- 全片李鸿章的脊背始终笔直，这是他最重要的视觉符号
- 唯一例外：S12定远舰，"弯了，但没有倒"
- 终幕叠化：年轻的李鸿章在油菜花田策马，脊背笔直 → 与S16血染黄马褂的脊背笔直呼应
- S01老鹰收翅下坠 → 翅膀折叠的姿态像弯下的脊背（隐喻预告）

**动机三：油菜花田的金色**
- S09 布局情感高潮（铺垫）：金色，高饱和，全片最亮（v6明确：非催化剂）
- S17 终幕叠化：雪夜（冷）→ 油菜花田（金色）→ 主题落地
- 开场威海卫的压迫（冷）与终幕金色的光形成视觉证明
- S01老鹰翼尖的金色（逆光燃烧）→ 微弱的金色预告，与终幕形成对照

**动机四：门/门槛（v6新增）**
- S02李莲英进门 → 刀放在桌上，审判开始
- S16李鸿章推门 → 血染黄马褂，尊严入场
- S15李莲英出门 → 望远镜碎裂后起身离开，"中堂没事"，带着秘密走回风雪中
- 三次门的意象：进入审判 / 进入牺牲 / 带着真相离开

### 景别逻辑

- 景别收紧触发（→特写）：情绪转折点、关键道具出现、人物做出不可逆的选择
- 景别放开触发（→全景）：时间跨越、人物被环境吞没、史诗感场景
- 特写策略：**极度克制**，全片特写只在以下时刻出现：
  - 老鹰翅膀（S01开场，逆光燃烧的羽毛边缘）
  - 望远镜（三次，S05/S14/S15）
  - 牛肉（S10苏州杀降）
  - 炮口的灰（S12定远舰）
  - 李鸿章的手（S16马关谈判桌）
  - 碎裂的镜片（S15北洋覆没）
  - **李莲英的背影**（S15离开时，不回头，走向门外的风雪）【v6新增】
  - **张十三的嘴角**（S17终幕，第一次笑）【v6新增】
  - 特写出现即意味着这个细节是全片的叙事锚点

## 表演风格基准（film-performance 模式一）

### 人物脊柱

**李鸿章**：撑住，不让它在我手里垮掉
每个场景的行动都是这个脊柱的具体表现——无论是杀降、建工厂、还是推开谈判厅的门，他都在"撑"。

**张十三**：看见他，并且留下来
他不追随李鸿章，但也从未真正离开。他的每一次"不扶"、每一次沉默，都是这个脊柱的表现——用旁观来保护李鸿章，也保护自己。
**v6补充**：S15他的崩塌不是被外力击倒，是他在讲述中自己撑不住了——这是他一生第一次失控，也是他最诚实的时刻。

**李莲英**：证明北洋的失败不是我们的错
他来审问，是因为他需要一个答案。当他得到的答案是"是我们的错"，他的脊柱断了。
**v6补充**：S15他临走前说"中堂没事"，这句话不是感化，是他的选择——他选择把真相带走，而不是交给慈禧。他的脊柱没有断，他换了一种方式站着。

### 给定情境

时代背景：1844-1895年，晚清，从鸦片战争后的余震到甲午战败，大清从"天朝上国"的幻觉走向彻底崩塌。

人物处境：1895年，甲午战败，北洋水师覆没，张十三在总理衙门等待前线最后的消息，同时被宫廷审问。

人物关系：李鸿章与张十三，同年进士，半生挚友，走上截然不同的路。他们的关系是全片的情感核心——不是爱情，不是师徒，是见证。

---

### 表演风格

**整体风格：克制内敛**

行动藏在细节里——一个放下杯子的动作比一句台词更重要。

**沉默策略**：沉默是常态。开口意味着重要，沉默意味着更重要的事情正在发生。全片最重要的时刻几乎都是无声的：苏州杀降后李鸿章独处、望远镜独处、张十三不扶、终幕祖孙看雪。

**倾听密度**：极高。每个人物都在真正听对方说话。张十三的倾听是全片最重要的表演——他听李鸿章说话的方式，就是他爱李鸿章的方式。

**v6表演调整：**

| 场景 | 调整内容 | 表演指导 |
|------|---------|---------|
| S15 北洋覆没 | 从"急报触发"改为"讲述中自己说出" | 张十三的声音先稳后颤，说到"丁汝昌拒降殉国"时停顿——不是犹豫，是喉咙发紧。望远镜从他手中滑落不是刻意动作，是手松了。碎裂声让他回神，但他没有去捡。 |
| S15 李莲英离开 | 望远镜碎裂后起身，临走前告知"中堂没事" | 李莲英站起来，脚步比来时慢。他停在门口，背对张十三，不回头——"老佛爷那边，咱家自有说法。"（停顿）"中堂没事。"说完后不等待回答，走出去。门关上的声音要轻。 |
| S17 雪落无声 | 张十三第一次笑了 | 这个笑不是开心，是释然。嘴角先动，然后眼睛弯了。很轻，但张念海看见了。这是全片张十三唯一一次笑，必须让观众感受到它的重量。 |

## S01 · 威海卫（开场画面）专项视觉设计

> v6 新增：基于 narrative "老鹰被流弹击中坠落 + 字幕交代"

**节拍：节拍1（开场画面）| 情绪：压迫，预兆，全片情感底色**

### 色彩定调
- **主色**：深海蓝灰（接近黑场）
- **辅色**：炮火微红（远处闪烁，不稳定）+ 鹰翼金色（逆光燃烧，极短暂）
- **禁用色**：任何稳定暖色——这个空间里没有希望，只有即将到来的毁灭

### 空间与线条
- **空间**：极深空间（威海湾全景），舰队、日军包围圈、天空三层纵深
- **线条**：
  - 水平线：海面平静如镜（死亡般的平静）
  - 斜线：老鹰俯冲轨迹（不稳定的自由）
  - 垂直线：舰船桅杆、炮管（权威，但被困住）
- **景别逻辑**：大远景→远景→中远景→特写（鹰翼）→大远景→黑场（快速切换，建立压迫感）

### 运动与节奏
- **摄影机运动**：
  - 开头：静止（黑场+声音先行）
  - 亮起：固定镜头（鹰翅膀特写，逆光）
  - 俯冲：急速跟随+下摇（鹰视角，速度感）
  - 展开：缓慢横移（海湾全景，信息释放）
  - 盘旋：缓慢环绕（舰队困守细节）
  - 下坠：急速下摇→突然静止→硬切黑场
- **节奏**：极快剪辑（3-5秒/镜头），总时长约45-60秒
- **特殊处理**：鹰收翅下坠后，画面不是慢慢变黑，是**硬切黑场**——这种断裂感预示全片的叙事风格

### 光影设计
- **主光源**：黎明前的微光（冷，约6500K）+ 炮火的间歇性暖光（不稳定）
- **鹰翼逆光**：来自云层缝隙的唯一强光，把羽毛边缘烧成金色——这是全片第一个金色，与终幕油菜花田呼应
- **海面反光**：极弱，几乎是黑色的镜子——"平静比炮声更令人不安"的具象化

### 声音设计（先行）
- **0:00-0:05**：纯黑场，海浪声+低频嗡鸣（像远处的引擎空转）
- **0:05-0:08**：画面亮起，声音不变（海浪继续）
- **0:08-0:15**：鹰俯冲，风声渐起（从无到有，渐进）
- **0:15-0:25**：海湾展开，炮声隐约（很远，像心跳）
- **0:25-0:35**：盘旋+舰队细节，海浪声+炮声交替（不安的节奏）
- **0:35-0:40**：鹰收翅，风声骤停（突然的寂静）
- **0:40-0:45**：硬切黑场，寂静持续1秒，然后字幕淡入音（低频 hum）
- **字幕**："1895年·威海卫"（白字，衬线体，淡入淡出2秒）

### 核心视觉锚点
1. **老鹰** = 李鸿章（全片最重要的视觉隐喻）
   - 盘旋在炮火上空 = 被困住的清醒者
   - 不落下 = 还在坚持，还在观察
   - 收翅下坠 = 最终的选择（预告马关遇刺）
2. **海面的平静** = 表面平静下的毁灭（大清的现状）
3. **逆光燃烧的鹰翼** = 尊严的预告（即使坠落，也是有光的坠落）

## S15 · 北洋覆没（v6 重设计：内崩塌）

> v6 关键修改：从"急报触发"改为"张十三讲述中自己说出，撑不住了"

**原设计问题**：急报从外面传来 → 外部冲击 → 张十三被动反应
**新设计目标**：张十三主动讲述 → 内部积累 → 自己撑不住 → 崩塌来自内部

### 视觉重心转移

| 维度 | 原设计（急报触发） | 新设计（内崩塌） |
|------|------------------|----------------|
| **触发源** | 外部（急报、仆人闯入） | 内部（张十三自己的声音） |
| **摄影机距离** | 中景→特写（反应镜头） | 近景固定（凝视他的脸） |
| **运动** | 随急报切入而动 | **完全静止**（让他的崩溃成为唯一的运动） |
| **光线变化** | 急报带入的光/阴影 | 烛火慢慢暗下去（时间流逝+内心熄灭） |
| **声音设计** | 急报的 urgency | 他的声音越来越轻，直到碎裂声打破寂静 |
| **特写时机** | 急报内容 | **碎裂的镜片**（唯一的特写） |

### 逐镜视觉逻辑

**镜头1-3：讲述继续（近景，固定）**
- 景别：近景（胸部以上）
- 运动：**完全静止**（三脚架锁定）
- 焦点：张十三的脸，烛光在他脸上跳动
- 表演：他的声音平稳，但眼神在飘——不是在看李莲英，是在看记忆里的画面
- 时长：每个镜头15-20秒（让观众感受到时间的重量）

**镜头4：关键句（"丁汝昌拒降殉国"）（中近景，固定）**
- 景别：中近景（头部+肩部）
- 运动：仍然静止
- 焦点：他的嘴，然后慢慢回到眼睛
- 表演：说到"拒降"时停顿，说到"殉国"时喉结动了——他在忍
- 时长：20-25秒（这个停顿必须被充分感受）

**镜头5：停顿（近景，固定）**
- 景别：近景
- 运动：静止
- 焦点：他的整张脸
- 表演：他不说话了。嘴唇微张，但没有声音。眼神从记忆里回来，落到面前的桌子上
- 时长：10-15秒（这个 silence 比任何台词都重）

**镜头6：望远镜滑落（中景，固定）**
- 景别：中景（腰部以上，能看到桌上的望远镜）
- 运动：静止
- 焦点：他的手 → 望远镜
- 动作：他的手松了。不是放下，是失去了握力。望远镜从桌边滑落

**镜头7：碎裂（特写，固定）【全片第6个特写】**
- 景别：**特写**（碎裂的镜片，黄铜框架变形）
- 运动：静止（连摄影机都不动）
- 焦点：碎片，然后慢慢失焦
- 声音：碎裂声（清脆，在寂静里格外响）

## S17 · 雪落无声（v6 强化：第一次笑）

> v6 关键强化：张十三第一次笑了——这是全片他唯一一次笑，必须有足够的情感重量

### 笑的设计

**不是哪种笑：**
- ❌ 不是开心的笑（他没有理由开心）
- ❌ 不是苦笑（那太廉价）
- ❌ 不是自嘲的笑（他不需要自我辩护）

**是哪种笑：**
- ✅ 是释然的笑——"我说完了，我可以放下了"
- ✅ 是疲惫的笑——"这条路，我终于走完了"
- ✅ 是温暖的笑——对着孙儿，对着雪，对着那个已经不在的人

### 视觉呈现

**时机**：回答完"忠臣还是奸臣"之后，停顿，然后笑

**镜头设计**：
- 景别：**特写**【全片第8个特写，也是最后一个】（张十三的嘴部和眼睛）
- 运动：固定（完全静止，让这个笑成为全片最后的视觉事件）
- 焦点：先嘴角（笑意浮现），然后眼睛（眼角弯了）
- 时长：8-10秒（足够长，让观众看完整个笑的过程）

**光线**：
- 灯笼的暖光从侧面打过来，照在他的半边脸上
- 笑的时候，眼角的皱纹被光照亮——这些皱纹是他一生的痕迹
- 雪地的冷光作为补光，冷暖交织（释然，不是单纯的温暖）

**声音**：
- 没有笑声（他不出声，只是嘴角和眼睛在笑）
- 或者：极轻的一声叹息式的笑（像一个长长的呼吸）
- 背景音：雪落无声（真的是无声，或者只有极远的狗吠）

**后果**：
- 笑完之后，镜头保持1-2秒（让观众消化）
- 然后开始叠化：雪夜 → 油菜花田 → 年轻的李鸿章策马回头，笑
- 暗场

### 与开场老鹰的终极对照

|------|----------------|----------------------------------|
