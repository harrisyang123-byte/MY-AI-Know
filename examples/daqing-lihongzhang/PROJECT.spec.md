# 大清李鸿章 — 项目 Spec

## 项目概述

用 creative-architect 电影全才智能体走完整个创作流程（叙事定调 → 视听定调 → 艺术指导 → 分镜构思）。

**核心设定：**
- 主角：张砚卿（御史）和李鸿章（洋务派代表）
- 时间跨度：1844-1895年，五十年
- 叙事框架：现实线（1895年审讯）+ 回忆线（一生回忆）
- 参考风格：《长安三万里》的双线结构

## 关键决策记录

**为什么张砚卿是主视角？**
- 旁观者视角让观众看到李鸿章的全貌，同时保持距离感
- 张砚卿的弧光：从"守旧"到"理解"，最后"看透但无奈"

**为什么只有一个暖色场景？**
- 全片冷灰基调传递"历史的沉重"
- 油菜花田的金黄是唯一的希望时刻，用色彩孤立它，强化其重要性

**为什么前期服装不换？**
- 参考《长安三万里》的做法
- 服装连续性强化"同一个人"的连续性
- 后期（1863年之后）必须换，因为身份和权力发生了变化

   - 调用 film-directing + film-visual-language（模式二）+ film-performance（模式二）
   - 每个场景走完：第零步（钩子）→ 第一步（估算镜头数）→ 第二步（轴线）→ 第三步（行动节拍）→ 第四步（完整分镜稿）→ 衔接检查

- 台词必须从飞书剧本提取，不能自己编
- 视觉元素必须符合 03-art-direction.md
- 表演风格必须符合 02-visual-style.md（克制内敛为主）
- 每完成一个场景就停下，等待确认
