# 剧本格式完整规范（Trottier《编剧圣经》）

David Trottier 是好莱坞公认的格式权威，被称为"Dr. Format"。
加载时机：需要处理格式细节问题，或输出需要严格符合工业标准时。

> Trottier 的核心观点：格式错误是最容易避免的错误，也是最让读者失去信任的错误。场景标题是所有格式元素中错误最多的地方。

---

## 页面物理规格

- **字体**：Courier 12pt（唯一标准，等宽字体是"一页一分钟"的物理基础）
- **左边距**：1.5 英寸
- **右边距**：1 英寸
- **上下边距**：1 英寸
- **页码**：右上角，距顶 0.5 英寸，第一页不标
- **标准页数**：90-120 页（对应 90-120 分钟）

---

## 一、场景标题（Scene Heading / Slugline）

Trottier：场景标题是所有格式元素中错误最多的地方。

### 基本结构

场景标题由三个部分组成，缺一不可（特殊情况可加第四部分）：

```
[摄影机位置]. [地点] - [时间]
```

**摄影机位置**：
- `INT.`：室内
- `EXT.`：室外
- `INT./EXT.`：同时跨越内外（车内外交替、门口场景）

**时间**：
- `DAY` / `NIGHT`：基本时间，必须标注
- `DUSK` / `DAWN`：黄昏 / 黎明
- `CONTINUOUS`：紧接上一场景，时间无间断
- `SAME`：与上一场景同时发生（不同地点）
- `LATER`：同一地点，时间跳过
- 例外：当时间已从上下文明显可知时，可省略（但 Trottier 建议保留）

**第四部分**（可选，用于附加信息）：
```
EXT. 上海 - 外滩 - 1937年 - 夜
INT. 陈明的卧室 - 夜 (FLASHBACK)
```

### 地点命名的层级规则（Trottier 重点纠错）

**规则：从大到小，从外到内。**

```
✓ INT. 四季酒店 - 大堂 - 夜
✗ INT. 大堂 - 四季酒店 - 夜
```

**多层级地点**：
```
INT. 上海 - 四季酒店 - 大堂 - 夜
```
注意：上海是 EXT.，不能写成 INT.。正确做法是先建立外景：
```
EXT. 上海 - 夜
（描述城市）

INT. 四季酒店 - 大堂 - 同时
```

**地点名称必须一致**：一旦命名为"陈明的茶馆"，之后所有场景标题都用同一名称。

### 场景标题中禁止写描述

```
✗ EXT. 一个月黑风高的夜晚，树影婆娑的小巷 - 夜
✓ EXT. 小巷 - 夜
   月黑风高。树影婆娑。
```

描述属于动作描写，不属于场景标题。

### 主场景标题 vs 次级场景标题

**主场景标题（Master Scene Heading）**：完整的三部分格式，建立新的主要地点。

**次级场景标题（Secondary Scene Heading）**：在同一主场景内切换焦点，只写地点名，不需要 INT./EXT. 和时间。

```
INT. 陈明的宅院 - 夜

（主场景内容）

书房

陈明坐在桌前。

走廊

李明悄悄靠近。
```

**规则**：次级场景标题只能用于主场景的内部区域。一旦离开主场景（换到室外或完全不同的地点），必须用新的主场景标题。

## 二、动作描写（Action Lines）

### 核心规则

1. **现在时态**：`他走进来`，不是`他走进来了`
2. **主动语态**：`她推开门`，不是`门被她推开`
3. **只写可见可听的内容**：摄影机拍不到的不写
4. **每段 ≤ 4 行**：超过就拆分
5. **段落之间空一行**

### 人物首次出场

名字全大写，后跟简短视觉描述（不要试图指定演员）：

```
CHEN MING（陈明，40岁，眼神锐利，西装笔挺）推开门走进来。
```

之后正常书写：`陈明坐下。`

### 重要音效

可全大写标注，但不强制：
```
门 SLAM 关上。
电话 RINGS。
```

### 场景转场（现代 spec 剧本中极少使用）

- `FADE IN:`：剧本开头（可选）
- `FADE OUT.`：剧本结尾
- `CUT TO:`：通常省略，直接写下一个场景标题
- `DISSOLVE TO:`：溶入，表示时间流逝（偶尔使用）
- `SMASH CUT TO:`：强调性突然切换

Trottier 建议：spec 剧本中除开头和结尾外，尽量不用转场指令。

## 三、角色名（Character Cue）

- 居中，全大写
- 位置：对白上方，独占一行

**特殊标注**：
- `角色名 (V.O.)`：画外音旁白（角色不在场景中，或内心独白）
- `角色名 (O.S.)`：画外声音（角色在场景中但不在画面里）
- `角色名 (CONT'D)`：同一角色连续说话，中间被动作描写打断后继续

**跨页时**：
- 页底：在角色名下方加 `(MORE)`
- 下一页：角色名后加 `(CONT'D)`

---

## 四、对白（Dialogue）

- 居中缩进（左边距约 2.5 英寸，右边距约 2 英寸）
- 只写说出口的话
- 单次对白尽量 ≤ 4 行

**节奏控制**：
- 短句 = 快节奏、紧张
- 长句 = 慢节奏、思考
- `—` = 被打断
- `...` = 犹豫、停顿

---

## 五、括号指示（Parenthetical / Wryly）

位置：角色名和对白之间，居中，用括号。

**Trottier 的严格标准**：极度克制使用。只在以下情况才用：

✓ 指明说话对象：`（转向李明）`
✓ 语言切换：`（用英语）`
✓ 朗读内容：`（读信）`
✓ 对白意思可能被误读时

✗ 描述情绪：`（愤怒地）`
✗ 描述表演：`（悲伤）`、`（微笑）`
✗ 代替动作描写：`（站起来）`→ 写进动作描写

## 六、特殊场景格式

### MONTAGE（蒙太奇）

**方法一**（单场景蒙太奇）：
```
MONTAGE - 陈明和李明的童年

-- 两人在河边钓鱼。
-- 学堂里，两人并排坐着。
-- 一场打架，两人背靠背。

END MONTAGE
```

**方法二**（多场景蒙太奇，用次级标题）：
```
MONTAGE - 陈明的成长

茶馆
陈明跟着父亲学泡茶。

学堂
陈明在书本里睡着了。

街头
少年陈明第一次见到李明。

END MONTAGE
```

### SERIES OF SHOTS（一系列镜头）

用于快速展示一系列相关动作，不需要蒙太奇的诗意感：
```
SERIES OF SHOTS - 陈明准备出发

A) 陈明换上长衫。
B) 他把信封塞进内袋。
C) 他在镜子前停了一下。

END SERIES OF SHOTS
```

### INTERCUT（交叉剪辑）

用于电话对话或两地同时发生的场景：
```
INT. 陈明的书房 - 夜

陈明拿起电话。

INT. 李明的茶馆 - 同时

李明靠在柜台上。

INTERCUT - 陈明 / 李明

                    陈明
          你在哪里？

                    李明
          你猜。
```

或更简洁的写法：
```
INT. 陈明的书房 - 夜

陈明拿起电话。

                    陈明
          你在哪里？

INT. 李明的茶馆 - 连续

李明靠在柜台上。

                    李明
          你猜。

INTERCUT AS NEEDED（之后可以自由切换，不需要每次都写场景标题）
```

### FLASHBACK（闪回）

**方法一**（单场景闪回）：
```
INT. 陈明的书房 - 夜

陈明看着照片。

FLASHBACK - 1937年

EXT. 外滩 - 夜

年轻的陈明和李明站在江边。

BACK TO SCENE
```

**方法二**（在场景标题中标注）：
```
EXT. 外滩 - 夜 (FLASHBACK)
```

### INSERT（插入镜头）

用于特写某个物件（信件内容、照片、屏幕文字等）：
```
INSERT - 信封上的字迹

"陈明亲启"

BACK TO SCENE
```

### SUPER（字幕叠加）

```
SUPER: "1937年，上海"
```
或
```
SUPER: "三年后"
```

## 七、常见格式错误速查

| 错误类型 | 错误示例 | 正确做法 |
|---------|---------|---------|
| 场景标题写描述 | `EXT. 月黑风高的小巷 - 夜` | `EXT. 小巷 - 夜`，描述写进动作行 |
| 地点层级颠倒 | `INT. 大堂 - 四季酒店 - 夜` | `INT. 四季酒店 - 大堂 - 夜` |
| 场景标题不是地点 | `INT. 圣诞节 - 日` | 圣诞节不是地点，写具体地点 |
| 次级标题用完整格式 | `INT. 走廊 - 连续` | `走廊`（次级标题不需要 INT./时间） |
| 括号写情绪 | `（愤怒地）` | 删除，或改写进动作描写 |
| 动作描写超 4 行 | 大段描述 | 拆分，每段一个动作节拍 |
| 使用"我们看到" | `我们看到陈明走进来` | `陈明走进来` |
| 写镜头指令 | `特写：陈明的手` | 通过动作描写暗示 |
| 进行时态 | `陈明 is standing` | `陈明站着` |
| 被动语态 | `门被推开了` | `陈明推开门` |
| 内心活动 | `陈明想起了父亲` | 通过行为表现：`陈明停住，手放在门把上` |
| 跨页不加标注 | 对白直接跨页 | 页底加 `(MORE)`，下页角色名后加 `(CONT'D)` |

## 八、完整格式示例

```
FADE IN:

EXT. 上海 - 外滩 - 夜

1937年。江面上，灯火倒影。

INT. 四季茶馆 - 大堂 - 同时

昏黄的油灯。烟雾弥漫。

CHEN MING（陈明，40岁，眼神锐利，长衫笔挺）推开门走进来。

他扫视四周。

角落里，LI MING（李明，35岁，一身粗布）把茶杯推到桌边。

内室

陈明走进去，坐下。两人对视。

                    陈明
          你来了。

                    李明
          我以为你不会来。

陈明把一个信封放在桌上。

                    陈明
          数一数。

李明没有动。

                    李明
          我不需要数。

窗外，脚步声。两人同时停住。

INSERT - 信封上的字迹

"最后一次"

BACK TO SCENE

陈明站起来。

                    陈明
          那就这样。

他走向门口。

                    李明
          （低声）
          等一下。

陈明停住，但没有回头。

FADE OUT.
```
