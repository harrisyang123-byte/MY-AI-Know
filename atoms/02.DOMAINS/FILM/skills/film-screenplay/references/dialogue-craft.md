# 对白技法深度参考
## 基于 Robert McKee《对白：文字、舞台、银幕的言语行为艺术》

> 来源：Robert McKee, *Dialogue: The Art of Verbal Action for Page, Stage, and Screen* (2016)
> 本文档为 film-screenplay skill 的核心参考，写台词前必须内化这些原则。

---

## 一、对白的根本定义

McKee 的核心命题，也是整本书的基础：

> **"对白不是对话。说话就是在做某件事。因此，在每一行角色台词之下，作者必须创造一个欲望、意图和行动。"**
> — mckeestory.com

对白是**言语行为（verbal action）**，不是信息传递，不是性格展示，不是情节推进的工具。它是行动本身。

这意味着：
- 每句台词都在执行一个意图，对另一个人物产生影响，并引发反应
- 没有任何角色说话是没有原因的
- 沉默也是一种言语行为——拒绝给对方想要的反应，本身就是权力

---

## 二、对白的三条轨道

McKee 指出，角色的话语沿三条截然不同的轨道运行：

1. **说给他人的（Said to Others）**：戏剧化对白，角色在场景中互相说话
2. **说给自己的（Said to Oneself）**：独白，内心活动的外化
3. **说给读者/观众的（Said to the Reader/Audience）**：叙述化对白，打破第四堵墙

**这三条轨道的规则完全不同。** 混淆它们是初学者最常见的错误。

---

## 三、戏剧化对白 vs 叙述化对白

这是 McKee《对白》中最重要的区分，也是最常被误解的。

### 戏剧化对白（Dramatized Dialogue）

角色在场景中互相说话，有冲突，有行动和反应。

- "show don't tell"的规则**适用于此**
- 每句话都是一个行动，引发对方的反应
- 潜台词规则在这里全面生效
- 这是电影对白的主体

### 叙述化对白（Narratized Dialogue）

角色跳出场景，直接对观众说话（旁白、独白、直接叙述）。

- "show don't tell"的规则**不适用于此**
- 直接叙述有两个核心优势：**速度**和**反衬**
  - **速度**：快速传递大量信息，压缩时间
  - **反衬（Counterpoint）**：叙述内容与画面形成对比，产生讽刺或悲剧效果
    - 例：旁白说"那是我们最快乐的时光"，画面却是战场
- McKee 明确说：**"'展示而非讲述'的告诫只适用于戏剧化对白。"**

**《长安三万里》的高适旁白为什么有效：**
高适的叙述是叙述化对白，他在对观众讲述李白的一生。这条轨道允许直接告知，允许大量信息，允许情感直白——因为它的功能是速度和反衬，不是戏剧冲突。张十三的叙述框架同理。

**关键判断**：这段台词是角色在和另一个角色说话（戏剧化），还是在对观众说话（叙述化）？判断错了，规则就用错了。

## 四、三层内容球体

McKee 将对白的内容分为三个同心球体，好的台词在三层之间制造张力：

```
最外层：说出的（The Said）
    ↓ 张力
中间层：未说出的（The Unsaid）
    ↓ 张力
最内层：无法说出的（The Unsayable）
```

- **说出的（The Said）**：台词的字面内容，观众直接听到的
- **未说出的（The Unsaid）**：人物想说但没说的——人物知道自己在隐藏什么，这是潜台词的主要来源
- **无法说出的（The Unsayable）**：人物自己都不完全意识到的内心深处——最深层的动机、恐惧、欲望，通过行为和无意识的选择流露，而非台词

> "这三层复合体用思想和感受的内容填充对白，同时通过姿态和语言塑造表达。最内层球体涌动着无法言说的；中间层约束着已说出的；外层球体释放着未说出的。"
> — medium.com/@pirangy

**潜台词的深度 = 说出的和未说出的之间的距离。** 距离越大，台词越有张力。

**实操方法**：写完一句台词，问三个问题：
1. 这句话的字面意思是什么？（The Said）
2. 这个人物真正想说的是什么？（The Unsaid）
3. 这个人物自己都不知道的是什么？（The Unsayable）

如果三层都是同一个答案，这句台词是 on-the-nose 的。

## 五、台词的行动类型

每句台词必须能说出它在执行哪种行动。说不出来，考虑删除。

| 行动类型 | 含义 | 示例 |
|---------|------|------|
| 威胁（Threaten） | 用后果施压 | "你最好想清楚。" |
| 引诱（Seduce） | 用利益或情感吸引 | "我们可以一起做这件事。" |
| 攻击（Attack） | 直接伤害对方 | "你从来就不在乎任何人。" |
| 回避（Deflect） | 转移话题，拒绝正面回应 | "你喝茶吗？" |
| 安慰（Comfort） | 减轻对方的痛苦 | "没事的，我在这里。" |
| 投降（Surrender） | 放弃立场或欲望 | "好，随你。" |
| 揭示（Reveal） | 主动暴露信息 | "其实我一直知道。" |
| 隐藏（Conceal） | 掩盖真相或意图 | "我不知道你在说什么。" |
| 说服（Persuade） | 改变对方的想法或行动 | "听我说完，你会明白的。" |
| 拒绝（Refuse） | 拒绝对方的要求或期待 | "这件事我不会做。" |

**注意**：同一句台词可以同时执行多个行动。"你喝茶吗"表面是回避，实际可能是引诱或威胁，取决于语境和人物关系。

## 六、On-the-Nose 台词（最常见的致命错误）

**定义**：人物直接说出自己的真实意图、感受或想法。

**为什么是错的**：
- 真实的人在压力下不会直接说出内心想法
- 直接说出 = 没有潜台词 = 没有戏剧张力
- 观众失去了"读懂"人物的乐趣

**识别方法**：
- 台词是否直接描述了人物的情绪？（"我很愤怒"）
- 台词是否直接说出了人物的意图？（"我想要控制你"）
- 台词是否直接解释了人物的行为？（"我这样做是因为我爱你"）
- 台词是否在解释剧情？（"你知道，自从三年前那件事之后..."）

**修复方法**：

1. **说相反的话**
   - ✗ "我很在乎你离开这件事"
   - ✓ "没事，走吧"（但动作显示他在乎）

2. **谈论不相关的事**
   - ✗ "我害怕失去这份工作"
   - ✓ "你知道这个城市的房租涨了多少吗？"

3. **用动作代替台词**
   - ✗ "我不想再谈这件事了"
   - ✓ 他站起来，把椅子推回桌子，走向窗口

4. **打断和沉默**
   - 没说出口的比说出口的更重要
   - 沉默本身就是一种台词

5. **用感官意象代替抽象陈述**（McKee 特别强调）
   - ✗ "她的声音很难听"（抽象）
   - ✓ "她的声音是酸的"（感官）
   - 感官意象直接触发观众的身体反应，比抽象描述更有力

## 七、人物声音的独特性

McKee 的核心观点：

> "每个角色都用一种只有他才会用的句法、节奏、音调，以及最重要的——词汇选择来说话。理想情况下，每个角色都是他独特词汇集合的行走字典。因此，对白的原创性始于词汇。"
> — medium.com/@pirangy

**声音独特性的四个维度**：

1. **词汇（Vocabulary）**：这个人物用什么词？
   - 受教育程度、阶层、职业、时代决定词汇范围
   - 具体词汇比模糊词汇更有力：说"钉子"是一回事，说"道钉"是另一回事
   - 人物的个性往往体现在修饰词上（形容词和副词的选择）

2. **句法（Syntax）**：这个人物怎么组织句子？
   - 短句 vs 长句：思维方式的体现
   - 完整句 vs 碎片句：情绪状态的体现
   - 主动语态 vs 被动语态：权力感的体现

3. **语调（Tone）**：这个人物说话的情感基调是什么？
   - 克制 vs 外放
   - 讽刺 vs 直白
   - 正式 vs 随意

4. **意象（Imagery）**：这个人物用什么比喻和意象？
   - 意象来自人物的生活经验和职业背景
   - 一个将军和一个诗人描述同一件事，用的意象完全不同

**检验方法**：遮住角色名，只看台词，还能认出是谁说的吗？不能 → 根据人物三维骨架重写。

**本项目的应用**：
- 李鸿章：词汇直接有力，句子短，用军事和实务意象，不绕弯子
- 张十三：词汇古雅，句子完整，用儒家经典意象，说话前有停顿
- 李莲英：词汇宫廷化，句子有礼貌的威胁感，用"咱家"自称

## 八、感官语言（Sensory Language）

McKee 特别强调对白的感官维度：

> "语言构成的对白提供了一个从心理意义到感官体验的光谱。例如，一个角色可以把歌手的声音称为'糟糕的'或'酸的'。两者都有意义，但'糟糕的'是一个死隐喻，而'酸的'还有生命。观众一听到'酸的'，嘴唇就开始皱起来。"
> — goodreads.com

**原则**：越感官的表达，越深入、越难忘。

| 抽象表达 | 感官表达 |
|---------|---------|
| 她走路像模特 | 她移动像一首缓慢的热歌 |
| 他的声音很难听 | 他的声音是酸的 |
| 这件事很痛苦 | 这件事像吞了一块生铁 |

**在古代汉语语境中的应用**：
- 古典汉语本身就富含感官意象，要充分利用
- 避免现代白话的抽象表达混入古代人物的台词
- 李鸿章说"里子朽了，能捂，能遮"——这是感官的，有触觉

## 九、冲突的四种类型

McKee 分析了对白冲突的四种形态，每种产生不同的叙事效果：

**对称冲突（Symmetric Conflict）**
- 双方势均力敌，直接对抗
- 效果：紧张、激烈
- 适用：高潮对决、正面冲突场景

**不对称冲突（Asymmetric Conflict）**
- 双方力量悬殊，弱者用间接手段对抗强者
- 效果：张力来自不平等，弱者的策略更有趣
- 适用：权力不对等的场景（审讯、谈判）

**间接冲突（Indirect Conflict）**
- 双方都不直接说出真正的冲突所在，通过表面话题掩盖真实争论
- 效果：最强的潜台词，观众需要"读懂"真正的争论
- 适用：关系复杂、不能直说的场景
- 这是最有电影感的冲突类型

**最小冲突（Minimal Conflict）**
- 几乎没有明显冲突，但通过细节和沉默传递张力
- 效果：压抑、克制、余韵悠长
- 适用：情绪沉淀场景、关系即将破裂前的平静

**选择原则**：根据场景的情绪目标选择冲突类型。不是所有场景都需要对称冲突——间接冲突和最小冲突往往更有电影感。

## 十、权力转移（Power Shift）

每一轮对话交换都应该改变权力关系。这是对白场景的核心动力。

**权力的来源**：
- 信息（谁知道更多）
- 情感（谁更需要对方）
- 物理（谁在更强的位置）
- 道德（谁占据道德高地）
- 沉默（拒绝给对方想要的反应，本身就是权力）

**检验方法**：
- 这段对话开始时，谁更强？
- 结束时，谁更强？
- 如果没有变化，这段对话可能是废话

## 十一、对白的常见病症（Flaws and Fixes）

McKee 在书的第二部分专门讨论对白的病症，按严重程度排列：

### 病症1：不可信（Incredibility）
人物说了在这个情境下不可能说的话。
- 原因：编剧需要传递信息，让人物说出他们不会说的话
- 修复：把信息藏进冲突里，让人物在争论中自然流露

### 病症2：陈词滥调（Cliché）
人物用了观众已经听过一千遍的表达。
- 原因：编剧没有深入思考这个人物会怎么说
- 修复：回到人物的词汇库，找到只有这个人物才会用的表达

### 病症3：On-the-Nose（直白说出）
见第六节。

### 病症4：重复（Repetition）
同一个信息或情感被重复表达。
- 原因：编剧不信任观众，或不信任自己的台词
- 修复：说一次，信任观众

### 病症5：过度解释（Over-Explanation）
人物解释了不需要解释的事情。
- 原因：编剧担心观众看不懂
- 修复：删掉解释，让行动说话

### 病症6：台词太长（Overwriting）
单次台词超过观众能消化的长度。
- 电影台词比舞台剧短得多
- 长独白必须有戏剧性的理由，不是为了传递信息
- 修复：把长独白拆成对话，或用动作替代

## 十二、创意限制原则（Principle of Creative Limitation）

McKee 的一个重要但常被忽视的原则：

**限制激发创意。** 当你给人物施加限制（他不能直说、他不能表现情绪、他不能承认某件事），你就被迫找到更有创意的表达方式。

**应用**：
- 给每个人物设定一个"不能说的话"——他最想说但永远不会直说的那句话
- 整个场景围绕这个"不能说的话"设计，所有台词都在接近它但永远不触碰它
- 这个"不能说的话"在结尾可以通过行动而非台词来表达

**本项目的应用**：
- 李鸿章不能说的话："我知道这一切都是徒劳的，但我没有退路"
- 张十三不能说的话："我一直在保护你，这就是我的沉默的意义"
- 李莲英不能说的话："是我们害死了北洋"

## 十三、电影对白的特殊要求

McKee 专门讨论了电影对白与舞台、小说的区别：

> "摄影机和麦克风如此放大和扩展行为，以至于每一个虚假的眼神、每一个错误的姿态、每一句做作的台词，看起来和听起来都比最糟糕的晚宴表演更业余。银幕表演需要一种自然主义的、可信的、看似随意的技巧。为了使这成为可能，银幕对白必须感觉是自发的。"
> — medium.com/@pirangy

**电影对白的核心要求**：
1. **自发感（Spontaneity）**：台词必须感觉是当下想到的，不是背好的
2. **简洁（Economy）**：电影是视觉媒介，图像承担了大量叙事，台词只需要做图像做不到的事
3. **可听性（Speakability）**：台词必须能被演员自然说出，读起来顺口

**量的问题**：
- 电影对白的量不是越少越好，而是**恰好够用**
- 动作片可以几乎无台词（《孤独求生》），对话片可以全是台词（《与安德烈共进晚餐》）
- 关键不是量，是质——每句话都必须感觉是必要的

## 十四、台词自查流程（写完每段对白后执行）

```
第一步：判断对白类型
这是戏剧化对白（角色互相说话）还是叙述化对白（对观众说话）？
→ 叙述化对白：允许直接告知，重点是速度和反衬
→ 戏剧化对白：继续以下步骤

第二步：逐句标注行动类型
每句台词 → 威胁/引诱/攻击/回避/安慰/投降/揭示/隐藏/说服/拒绝
说不出行动类型 → 考虑删除或改写

第三步：检查三层球体
这句话的 Said / Unsaid / Unsayable 分别是什么？
三层都是同一个答案 → On-the-Nose，需要修复

第四步：检查权力转移
对话开始时谁更强？结束时谁更强？
没有变化 → 这段对话可能是废话

第五步：检查冲突类型
这段对话是对称/不对称/间接/最小冲突？
是否符合场景的情绪目标？

第六步：检查声音独特性
遮住角色名，还能认出是谁说的吗？
不能 → 根据人物三维骨架重写

第七步：检查感官语言
有没有可以用感官意象替换的抽象表达？
→ 替换，让台词有触感

第八步：检查病症
不可信 / 陈词滥调 / On-the-Nose / 重复 / 过度解释 / 台词太长
→ 逐一修复
```

## 十五、本项目（大清李鸿章）的具体应用指南

### 叙述化对白的使用

张十三是叙事者，他的旁白是叙述化对白，规则不同于戏剧化对白：
- 允许直接告知时间、地点、背景
- 允许直接表达情感判断
- 重点是速度（快速跨越时间）和反衬（他说的话与画面形成对比）
- 但旁白不能替代戏剧化场景——它是补充，不是替代

### 三个人物的声音建立

**李鸿章**：
- 词汇：直接、军事化、实务性，偶尔引用诗句（但用来宣言，不用来抒情）
- 句法：短句为主，不绕弯子
- 意象：战场、刀、铁、血
- 不能说的话："我知道这一切都是徒劳的"

**张十三**：
- 词汇：古雅、儒家经典，但不迂腐
- 句法：完整句，有停顿，说话前会想
- 意象：书、灰、纸、水
- 不能说的话："我的沉默是我对你的保护"

**李莲英**：
- 词汇：宫廷礼仪用语，"咱家"自称，但威胁感藏在礼貌里
- 句法：先礼后兵，句子有层次
- 意象：刀、茶、老佛爷
- 不能说的话："是我们害死了北洋"

### 关键场景的冲突类型
