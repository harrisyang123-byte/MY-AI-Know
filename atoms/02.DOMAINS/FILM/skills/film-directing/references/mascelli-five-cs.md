# Mascelli 五个C 完整参考

Joseph V. Mascelli《镜头在说话》（The Five C's of Cinematography，1965）的核心规则。
加载时机：需要深入处理特定问题时（轴线细节、构图决策、特写规范、剪辑时机）。

## C1：Camera Angles（镜头角度）

### 三种角度类型

**客观角度（Objective）**
- 旁观者视角，人物不看镜头
- 观众是不可见的旁观者
- 大多数叙事场景的默认选择

**主观角度（Subjective）**
- 摄影机就是人物的眼睛
- 对方演员必须直视镜头
- 持续使用会令人疲倦（观众看不到主角的反应）
- 最有效的用法：短暂插入、直接叙述（纪录片）、集体视角

**POV 角度（Point of View）**
- 介于客观和主观之间
- 摄影机在人物侧面，人物看镜头旁边（不是直视镜头）
- 提供亲密感而不产生主观角度的疲倦感
- 通常跟在 OTS 镜头之后，先建立关系再进入 POV

**POV 的正确拍法**：
- 视线必须看向镜头旁边，不是直视镜头
- 对方演员站在摄影机尽量靠近的位置
- 当两人身高不同时，视线方向需要作弊（高个子看镜头略下方，矮个子看镜头略上方）

### 镜头高度的心理效果

**高角度（俯拍）**
- 削弱人物的力量感和存在感
- 使人物显得渺小、脆弱、失去权力
- 建立地理关系，展示纵深中的动作
- 减慢快速移动的视觉感受

**平角度（眼平）**
- 最少失真，最客观
- 特写几乎总应该从被摄者眼平拍摄
- 非剧情片摄影师常犯的错误：对坐着的人从站立高度拍摄，产生不讨好的俯视角度

**低角度（仰拍）**
- 增加人物的高度和主导感
- 产生威严、权威、威胁感
- 将人物从背景中分离，可以用天空作背景
- 扭曲构图线条，产生更有力的透视感

**荷兰角（Dutch Tilt）**
- 故意倾斜摄影机，产生失衡画面
- 用于暴力、情绪失控、精神扭曲场景
- 必须足够明显，不能像是意外倾斜
- 向右倾斜：主动、有力；向左倾斜：被动、软弱
- 最有效的组合：低角度 + 广角镜头 + 荷兰角

### 45°三维角度原则

避免正面拍摄人物或物体。约45°的三维角度能同时记录正面和侧面，产生立体感。
- 正面拍摄只在特定叙事需要时使用（法庭、讲台、正式场合）
- 物体拍摄时应能看到正面、侧面、顶面三个面

### 角度+角度（Angle-plus-Angle）

最有力的镜头组合：侧面角度 + 仰拍或俯拍同时使用。
- 记录最多的面，产生最强的三维效果
- 消除正面拍摄的平面感
- 广角镜头 + 低角度 + 三维角度 = 最强的速度感和力量感

## C2：Continuity（连续性）

### 动态屏幕方向

**建立方向后必须保持**：一旦确定人物或车辆的移动方向（左→右 或 右→左），整个序列必须保持一致。突然改变方向会让观众以为人物转身了。

**中性镜头（Neutral Shot）**：
- 正面迎来或背面离去的镜头是中性的
- 可以用来在两个方向之间过渡
- 跟踪镜头（直接跟在前方或后方）也是中性的

**合法的方向切换方法**：
1. 拍摄人物转身的动作
2. 在曲线或转角处越轴（观众看到了转向过程）
3. 插入一个反应特写（观众的注意力被转移）
4. 使用正面迎来镜头，从错误的一侧出画
5. 切到车辆内部，通过机位调整建立新方向

### 静态屏幕方向

**轴线规则（180°规则）**：
- 在两个人物之间画一条想象的轴线
- 所有机位必须在轴线同侧
- 越轴会导致人物左右位置互换，视线方向错误

**视线匹配的精确规则**：
- 眼睛决定视线方向
- 一个完全正面的人物可以向左或向右看，不移动头部
- 视线的任何改变必须在镜头内发生，不能在镜头之间发生
- 当人物在场景中移动时，轴线随之移动，必须在每个镜头结束时重新确定

**特殊情况**：
- 曲线：摄影机可能不知不觉越轴，需要用全景镜头展示完整的曲线运动
- 转角：轴线绕转角延伸，摄影机保持在同侧
- 门口：人物进入新场景时，方向可以重新建立（"新局面"）

### 时间连续性

**当下时间**：事件正在发生，观众参与感最强
**过去时间**：闪回，需要明确的时间标记
**条件时间**：梦境、幻觉、意识流，可以打破所有规则

## C3：Cutting（剪辑）

### 在动作上剪切

**核心原则**：优先在动作进行中剪切，而非在静止时剪切。动作掩盖剪辑点，使切换更流畅。

**具体操作**：
- 拍摄时，在每个镜头的开头和结尾重叠动作（前一个镜头的结尾动作在下一个镜头的开头重复）
- 不要在摄影机运动中剪切（推、拉、摇、移），应在运动开始前和结束后各有静止帧
- 不要在重要对话中途停止摄影机

### Cut-in vs Cut-away

**Cut-in 特写**（主要动作的放大）：
- 必须匹配前后镜头的动作、位置、视线
- 必须在前面的宽镜头中建立（观众需要知道它在哪里）
- 六种用途：强调高潮、放大细节、时间压缩、掩盖跳切、替代无法拍摄的动作、建立情绪基调

**Cut-away 特写**（次要动作）：
- 不需要匹配主要动作
- 但如果被摄者之前在画面中有确定位置，视线方向必须符合已建立的轴线
- 用途：展示反应、暗示观众如何感受、掩盖跳切或方向切换、评论主要动作

**选择原则**：Cut-in 通常比 Cut-away 更有效，因为它与主要动作更紧密相连。

### 保护镜头（Protection Shots）

为了给剪辑留余地，拍摄时应额外拍摄：
- 各种速度和方向的摇镜头
- 长移动镜头的静止开头和结尾
- 荷兰角的左右两个方向
- 各种视线方向的反应特写（"万能反应镜头"，可以插入任何地方）

### 直切比移动镜头更快

Mascelli 的反直觉观察：直切总是比移动镜头更快，因为直切直接到达目的地。长时间的推拉摇移往往包含无用的过渡画面。除非有叙事动机，多个可以直切的静止镜头通常比一个长移动镜头更有效。

## C4：Close-Ups（特写）

### 特写尺寸

| 名称 | 范围 |
|------|------|
| 中特写（MCU） | 腰部到肩部以上 |
| 头肩特写（Head & Shoulder） | 肩部以下到头顶（默认特写） |
| 头部特写（Head CU） | 仅头部 |
| 卡喉特写（Choker） | 嘴唇以下到眼睛以上 |
| 极端特写（ECU） | 极小物体或动作的全画面 |

### OTS 镜头规范

- 前景人物应从后四分之三角度拍摄，可见脸颊轮廓但看不到鼻子
- 如果前景人物的面部特征变得可识别，就变成了双人镜头
- 应该成对拍摄（两个人物各一个），保持相近的摄影机距离、角度和画面大小

### 特写的背景处理

- 背景应简洁，避免繁杂元素分散注意力
- 避免暖色系背景（红色等前进色会与人物竞争注意力）
- 人物应从背景中分离（通过对比度、色调差异）
- 移动摄影机几英寸往往就能解决背景问题

### 特写的运动处理

- 人物应该移动进入和离开特写，让剪辑师可以在动作上剪切
- 大幅放大的动作在画面上看起来会加速，所以特写中的动作应该刻意放慢

## C5：Composition（构图）

### 三分法

- 将画面横竖各分三等份，产生四个交叉强点
- 兴趣中心应落在强点附近，而非画面正中
- 地平线应在上三分之一或下三分之一处，不能平分画面
- 不要机械地每次都用同一个强点，会产生重复感

### 引导空间

- 人物视线方向需要留更多空间（鼻部空间/Nose Room）
- 人物移动方向前方需要留更多空间（引导空间/Lead Room）
- 头部上方留适当空间（头部空间/Head Room）
- 画面边缘不能切在关节处（膝盖、腰部、脖子）

### 正式平衡 vs 非正式平衡

**正式平衡（对称）**：
- 两侧视觉重量相等
- 静态、平和、平等关系
- 适用于势均力敌的对话场景

**非正式平衡（不对称）**：
- 主体偏移，用次要元素平衡
- 动态、有张力、主次关系明确
- 大多数叙事场景的选择
- 特写中人物必须偏离中心，在视线方向留出空间

**视觉重量规则**：
- 运动的物体比静止的重
- 向摄影机移动的物体比横向移动的重
- 画面上方比下方重
- 右侧比左侧更吸引注意力（左侧可以承受更大重量）
- 孤立的物体比群体中的重
- 明亮的物体比暗的重
- 暖色比冷色重

### 构图线条

| 线条 | 心理效果 |
|------|---------|
| 水平线 | 平静、休息（也可暗示速度） |
| 垂直线 | 力量、尊严、权威 |
| 对角线（左下→右上） | 上升、希望、前进 |
| 对角线（左上→右下） | 下降、危险、衰落 |
| 对角线（一般） | 动态、能量、最具戏剧性 |
| 曲线 | 柔和、流动、女性感 |
| 荷兰角 | 不稳定、危险、失控 |

### 构图深度

好的构图应该有纵深感，而不是平面的：
- 人物和物体应该在场景中前后分布，而不是排成一排
- 使用前景框架（门框、窗格、树枝）增加纵深层次
- 选择能产生汇聚线的角度和镜头
- 用对比光线区分前景和背景平面
- 让人物向摄影机移动，而不是横向移动

### 动态构图

在静止场景中制造突然变化：
- 人物或车辆突然进入画面（最有效：观众已被引导专注于平静场景时）
- 悬念构图：隐藏、延迟或延长重要动作（空白画面保持几秒，让观众等待）
