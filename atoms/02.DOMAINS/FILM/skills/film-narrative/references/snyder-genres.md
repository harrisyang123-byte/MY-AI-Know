# Snyder 10种叙事类型详解

Blake Snyder 的类型体系不是传统的动作/爱情/恐怖，而是基于叙事结构和人物动力的分类。每种类型有其核心规则，违反这些规则会导致结构失衡。

---

## 1. 怪物入侵（Monster in the House）

**核心公式**：封闭空间 + 怪物 + 有人犯了罪引来怪物

**三个必要元素**：
- 怪物：可以是字面上的怪物，也可以是连环杀手、自然灾害、超自然力量——关键是它的力量超出正常范围
- 房子：封闭空间，逃脱不是选项（海滩、飞船、房子、小镇、地球）
- 原罪：有人做了某件事引来了怪物（贪婪关闭海滩、好奇心打开了门）

**节拍特点**：
- 催化剂 = 怪物首次出现或原罪被揭示
- 娱乐与游戏 = 怪物逐步逼近，受害者一个个消失
- 一败涂地 = 主角意识到自己也无法逃脱
- 结局 = 主角用原罪的代价换来怪物的消灭

**典型例子**：大白鲨、异形、驱魔人、惊声尖叫、恐慌室

---

## 2. 寻宝之旅（Golden Fleece）

**核心公式**：主角踏上旅程寻找某物，最终发现自己

**三个必要元素**：
- 路线图：明确的目的地或目标
- 旅途伙伴：在路上遇到的人，帮助主角成长
- 路途考验：一系列挑战，每个都迫使主角改变

**节拍特点**：
- 催化剂 = 旅程的起点（被迫或主动出发）
- 娱乐与游戏 = 旅途中的探索和冒险（这是"promise of the premise"最充分的类型）
- 中点 = 主角以为找到了目标，但发现这不是真正想要的
- 结局 = 主角得到的不是出发时寻找的东西，而是自我发现

**典型例子**：星球大战、绿野仙踪、回到未来、指环王、公路电影类型

---

## 3. 愿望成真（Out of the Bottle）

**核心公式**：愿望被实现，但代价是学会珍惜原本的生活

**三个必要元素**：
- 愿望：主角强烈渴望某种改变（"我希望我是大人"/"我希望我能说谎"）
- 魔法：愿望以某种方式实现（字面上的魔法、科技、命运）
- 教训：魔法带来的问题迫使主角学会珍惜原本拥有的

**节拍特点**：
- 催化剂 = 愿望被实现的那一刻
- 娱乐与游戏 = 探索愿望成真的世界（通常是喜剧性的）
- 中点 = 愿望开始失控，代价开始显现
- 结局 = 主角主动放弃魔法，回归真实

**典型例子**：大话王、变身国王、周五、美梦成真、变身

---

## 4. 普通人遇上大麻烦（Dude with a Problem）

**核心公式**：普通人被卷入非凡处境，必须找到内心的英雄

**三个必要元素**：
- 无辜的主角：普通人，没有特殊能力，不是天生英雄
- 突发的问题：被迫卷入，没有选择
- 生死攸关的处境：代价是生命或同等重要的东西

**节拍特点**：
- 催化剂 = 普通生活被打破的那一刻（通常是突然的、不可预见的）
- 娱乐与游戏 = 主角用普通人的方式应对非凡处境（这种反差是核心卖点）
- 中点 = 主角意识到自己必须成为英雄，没有人来救他
- 结局 = 普通人用普通人的方式解决了非凡问题

**典型例子**：虎胆龙威、泰坦尼克号、辛德勒的名单、空军一号

---

## 5. 成长仪式（Rite of Passage）

**核心公式**：人生转折点，主角必须接受某个痛苦的真相才能继续前进

**三个必要元素**：
- 人生转折：青春期、婚姻危机、中年危机、失去亲人
- 错误的解决方案：主角最初用错误的方式应对（逃避、愤怒、否认）
- 接受：最终接受无法改变的现实，找到继续的力量

**节拍特点**：
- 催化剂 = 触发转折的事件（离婚、死亡、失业）
- 娱乐与游戏 = 主角用错误方式应对，情况越来越糟
- 一败涂地 = 错误方式彻底失败，主角无路可逃
- 结局 = 接受真相，以新的方式继续生活

**典型例子**：普通人、克莱默夫妇、毕业生、美国丽人

---

## 6. 伙伴情谊（Buddy Love）

**核心公式**：两个不同的人相遇，互相改变，学会需要彼此

**三个必要元素**：
- 不完整的主角：主角在某方面有缺失，需要伙伴来补完
- 对立的伙伴：伙伴与主角在某个关键维度上截然相反
- 分离与重聚：两人必须经历分离，才能真正理解彼此的价值

**节拍特点**：
- B故事 = 伙伴关系的建立（这个类型中B故事就是A故事）
- 中点 = 两人关系达到顶峰（假胜利）
- 坏人逼近 = 两人之间的冲突爆发，关系破裂
- 结局 = 重聚，两人都因对方而改变

**注意**：爱情片是伙伴情谊的变体，"伙伴"就是爱人。

**典型例子**：阿甘正传、雨人、诺丁山、当哈利遇上莎莉、48小时

---

## 7. 为何如此（Whydunit）

**核心公式**：不是"谁干的"而是"为什么"，通过调查揭示人性的黑暗

**三个必要元素**：
- 侦探/调查者：作为观众的代理人，带领我们进入黑暗
- 谜题：不只是犯罪，而是关于人性的谜题
- 黑暗真相：揭示的不只是罪犯，而是关于人类本性的令人不安的真相

**节拍特点**：
- 催化剂 = 调查的开始
- 娱乐与游戏 = 层层剥开谜题（每一层都更黑暗）
- 中点 = 侦探意识到这比他想象的更深、更黑暗
- 结局 = 真相揭示，但真相本身令人不安（好的 Whydunit 没有简单的正义）

**典型例子**：唐人街、JFK、内幕、七宗罪、公民凯恩

---

## 8. 傻瓜得胜（Fool Triumphant）

**核心公式**：被低估的局外人对抗体制，最终证明自己（或证明体制的荒谬）

**三个必要元素**：
- 傻瓜：被体制低估的局外人（可以是真的傻，也可以是装傻）
- 体制：值得被嘲弄的权威结构
- 同伴：傻瓜通常有一个"正常人"同伴，帮助傻瓜，也帮助观众理解傻瓜

**节拍特点**：
- 铺垫 = 建立傻瓜被低估的状态
- 娱乐与游戏 = 傻瓜用出人意料的方式应对体制（喜剧性）
- 中点 = 傻瓜似乎真的赢了（假胜利）
- 结局 = 傻瓜的胜利揭示了体制的真正问题

**典型例子**：阿甘正传、查理和巧克力工厂、莫扎特传、被嫌弃的松子的一生

---

## 9. 体制内外（Institutionalized）

**核心公式**：个人 vs 群体/机构，关于归属与身份的代价

**三个必要元素**：
- 群体：家庭、公司、帮派、军队、任何有规则的集体
- 选择：加入还是离开？服从还是反抗？
- 代价：无论选择哪边，都有代价

**节拍特点**：
- 铺垫 = 建立群体的规则和主角与群体的关系
- 娱乐与游戏 = 主角在群体内部的挣扎
- 中点 = 主角必须做出选择：完全融入还是彻底反抗
- 结局 = 选择的代价被完全呈现（通常是悲剧性的或苦涩的）

**典型例子**：飞越疯人院、教父、动物屋、美国美人、搏击俱乐部

---

## 10. 超级英雄（Superhero）

**核心公式**：非凡的人在平凡世界中的孤独与代价

**三个必要元素**：
- 超凡能力：可以是超自然的，也可以是极端的智慧、美貌、财富
- 平凡世界的代价：超凡能力带来的孤独、误解、嫉妒
- 对手：通常是主角能力的扭曲版本（同样超凡，但走向了黑暗）

**节拍特点**：
- 铺垫 = 建立超凡能力和它带来的孤独
- 娱乐与游戏 = 超凡能力的展示（观众来看这个）
- 中点 = 主角意识到能力本身不能解决真正的问题
- 结局 = 主角用超凡能力解决问题，但代价是更深的孤独

**注意**：续集往往失败，因为观众对超凡能力的同情会随着熟悉而消失。

**典型例子**：超人、蝙蝠侠、角斗士、美丽心灵、雨人（也可归入此类）

## 类型混用的风险

一个故事可以有两种类型的元素，但必须有一个主导类型。两种类型的核心规则互相冲突时，结构会崩溃。

常见的危险组合：
- 怪物入侵 + 寻宝之旅：封闭空间的规则（不能逃脱）vs 旅程的规则（必须移动）
- 愿望成真 + 成长仪式：魔法解决问题 vs 接受无法改变的现实

安全的组合：
- 伙伴情谊 + 寻宝之旅：旅程中建立关系（公路电影的标准结构）
- 普通人遇上大麻烦 + 超级英雄：普通人发现自己有超凡能力
