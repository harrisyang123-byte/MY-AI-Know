# Skill 2: Spatial Reasoning (空间推理)

## 核心原理

### 什么是 Spatial Reasoning？

在脑海中构建、操作和推理三维空间关系的能力。

**本质**：这是一种"空间想象"能力
- 输入：用户的空间描述（可能模糊、不完整、矛盾）
- 输出：精确的、合理的、可视化的空间布局

### 为什么重要？

AI生成图像时，最容易出错的就是空间关系！

**常见问题**：
- 门在错误的墙上
- 家具尺寸不合理
- 人物位置不对
- 视角和实际布局矛盾
- 物体之间的距离不对

**有了Spatial Reasoning**：
- 能发现用户描述中的矛盾
- 能补全缺失的空间信息
- 能验证生成结果是否合理
- 能精确指导AI生成正确的空间关系

### 关键认知：这是设计师和AI的最大差距！

AI的弱点：
- AI不真正理解"空间"
- AI只是匹配图像模式
- AI容易生成"看起来对，但空间错误"的图像

人类设计师的优势：
- 能在脑海中构建3D模型
- 能推理空间关系
- 能发现矛盾和问题

所以：Spatial Reasoning是你超越AI的关键能力！你要成为AI的"空间导航系统"！

## 5步空间推理流程

### 步骤1：建立坐标系 (Establish Coordinate System)

**目的**：给空间一个"参考框架"

**必须明确**：
1. 方向（东西南北 或 前后左右）
2. 原点（从哪里开始测量）
3. 单位（米、英尺等）

**示例 — 清代办公室**：

```
坐标系:
  - 原点: 房间中心
  - X轴: 东西方向 (东为正)
  - Y轴: 南北方向 (北为正)
  - Z轴: 上下方向 (上为正)
  - 单位: 米

房间边界:
  - 东墙: X = +3m
  - 西墙: X = -3m
  - 南墙: Y = -4m
  - 北墙: Y = +4m
  - 地面: Z = 0m
  - 天花板: Z = 3.5m
```

**为什么重要？**

有了坐标系，所有位置都可以精确表达！

- 没有坐标系："门在东边" → 模糊！东边哪里？
- 有了坐标系："门在东墙中央，X=+3m, Y=0m" → 精确！

---

### 步骤2：定义边界 (Define Boundaries)

**目的**：明确空间的"容器"

**必须明确**：
1. 房间形状（矩形、L型、圆形等）
2. 房间尺寸（长宽高）
3. 墙壁位置
4. 开口位置（门、窗）

**示例 — 清代办公室**：

```
房间形状: 矩形

尺寸:
  - 东西方向(深度): 8米
  - 南北方向(宽度): 6米
  - 高度: 3.5米

墙壁:
  - 东墙: 实墙,有门
  - 西墙: 实墙
  - 南墙: 实墙,有窗
  - 北墙: 实墙

开口:
  - 门: 东墙中央,1.2米宽,2.2米高
  - 窗: 南墙,位置和尺寸待定
```

**关键技巧**：
- 画草图！（哪怕是简单的平面图）
- 标注尺寸
- 标注方向

---

### 步骤3：放置元素 (Place Elements)

**目的**：把物体放到空间中

**每个元素必须明确**：
1. 位置（X, Y, Z坐标 或 相对位置）
2. 尺寸（长宽高）
3. 朝向（面向哪个方向）
4. 与其他元素的关系

**示例 — 清代办公室**：

```
元素1: 门
  位置: 东墙中央 (X=+3m, Y=0m, Z=0m)
  尺寸: 1.2m宽 x 2.2m高
  朝向: 向外开 (向东开)

元素2: 办公桌
  位置: 房间中后部 (X=0m, Y=+1m, Z=0m)
         距离东门3米
  尺寸: 1.5m宽 x 0.8m深 x 0.8m高
  朝向: 桌子长边平行于东西方向
         办公面朝东 (面向门)

元素3: 椅子
  位置: 桌子后面 (X=0m, Y=+1.5m, Z=0m)
  尺寸: 0.6m宽 x 0.6m深 x 1.2m高
  朝向: 面向东 (面向门和桌子)

元素4: 人物(张十三)
  位置: 坐在椅子上 (X=0m, Y=+1.5m, Z=0.5m)
  尺寸: 坐姿高度约0.9m
  朝向: 面向东 (面向门)
```

**关键技巧**：
- 从大到小放置（先大家具，后小物件）
- 考虑功能关系（桌椅要配套）
- 考虑动线（人要能走动）
- 用相对位置描述（"桌子后面3米"）

---

### 步骤4：验证合理性 (Validate Rationality)

**目的**：检查空间布局是否合理

**检查清单**：

**尺寸合理性**：
- 家具尺寸符合人体工程学？
- 房间能容纳所有元素？
- 元素之间有足够空间？

**位置合理性**：
- 元素位置符合功能？
- 元素之间的距离合理？
- 没有重叠或冲突？

**朝向合理性**：
- 元素朝向符合使用习惯？
- 人物朝向合理？

**动线合理性**：
- 人能正常走动？
- 门能正常开关？
- 没有阻塞？

**视角合理性**：
- 从指定视角能看到应该看到的元素？
- 视角和布局一致？

---

### 步骤5：视角转换 (Viewpoint Transformation)

**目的**：从指定视角"看"空间

**必须明确**：
1. 相机位置（X, Y, Z）
2. 相机朝向（看向哪里）
3. 相机高度（eye-level, high angle, low angle）
4. 视野范围（能看到什么，看不到什么）

**示例 — 清代办公室西向东视角**：

```
相机位置:
  X = -2m (西侧,房间中后部)
  Y = 0m (南北居中)
  Z = 1.6m (人眼高度)

相机朝向:
  向东看 (朝向+X方向)

视野范围:
  能看到:
    ✓ 东墙 (正前方)
    ✓ 门 (东墙上)
    ✓ 办公桌 (中间偏右)
    ✓ 椅子 (桌子后面)
    ✓ 张十三 (坐在椅子上)
    ✓ 南墙 (右侧)
    ✓ 北墙 (左侧)

  看不到:
    ✗ 西墙 (在相机后面)
    ✗ 桌子背面 (被桌子挡住)

透视关系:
  - 东墙最远,看起来最小
  - 桌子在中景
  - 相机附近的物体(如果有)在前景,最大
```

**关键技巧**：
- 在脑海中"站"在相机位置
- 想象你能看到什么
- 画视角草图
- 检查透视关系

## 3种实战方法论

### 方法1：平面图法 (Top-Down Method)

**适用场景**：室内场景、建筑布局、城市规划

**步骤**：

1. 画平面图
```
┌─────────────────────┐
│         北          │
│                     │
│  窗                 │
│  ↓                  │
│                     │
│                     │
│         桌          │
│         ■          │
│         椅          │
│         ●          │
│                     │
│                     │
│         门          │
│         ↑          │
│         东          │
└─────────────────────┘
      西
```

2. 标注尺寸
3. 标注坐标
4. 确定视角
5. 推导可见元素

**优势**：直观、精确、容易发现问题

---

### 方法2：剖面图法 (Section Method)

**适用场景**：需要考虑高度的场景、多层空间、复杂的垂直关系

**步骤**：

1. 画剖面图（侧视图）
```
3.5m ┌─────────────────┐ 天花板
     │                 │
     │                 │
2.2m │     ┌──┐       │ 门高
     │     │门│       │
1.6m │  ●  └──┘       │ 人眼高度
     │  人             │
0.8m │  ═══           │ 桌面高度
     │  桌             │
  0m └─────────────────┘ 地面
     西              东
```

2. 标注高度
3. 确定视线

**优势**：清楚显示高度关系，容易检查视线遮挡

---

### 方法3：3D心智模型法 (3D Mental Model)

**适用场景**：快速推理、简单场景、有经验后的快速验证

**步骤**：

1. 在脑海中构建3D模型
   - 想象一个透明的盒子（房间）
   - 把元素一个个"放"进去
   - 想象自己"走"进这个空间

2. 从不同角度"看"
   - 从门口看进去
   - 从窗户看进去
   - 从上方俯视
   - 从人物视角看

3. 检查合理性

**训练方法**：
- 观察真实空间，然后闭眼回忆
- 看平面图，想象3D空间
- 练习画透视图

## 常见空间类型的推理策略

### 室内空间 (Interior Space)

**关键要素**：房间形状和尺寸、墙壁地面天花板、门窗位置、家具布局、人物位置和动线、光源位置

**推理流程**：
1. 确定房间"容器"（形状/尺寸/方向）
2. 放置"固定元素"（门/窗/壁炉/柱子）
3. 放置"家具"（从大到小，考虑功能分区和动线）
4. 放置"人物"（位置/姿态/能看到什么）
5. 确定"视角"（相机位置/朝向/可见范围）

**常见错误**：
- 家具尺寸不合理
- 家具摆放不符合使用习惯
- 门窗位置和视角矛盾
- 人物位置不合理
- 动线被阻塞

### 产品摄影空间 (Product Photography Space)

**关键要素**：产品位置和角度、背景、光源位置、相机位置和角度、道具位置

**推理流程**：
1. 确定产品位置（画面中心/偏左/偏右，高度，角度）
2. 确定相机位置（距离/高度/角度）
3. 确定光源位置（主光/补光/轮廓光）
4. 确定背景（纯色/场景，距离）
5. 放置道具（前景/背景/旁边）

---

## 铁律

1. **摄影机后方的物体永远不可见**
2. **明确使用方位词**（东西南北优于左右）
3. **距离用具体数值**（1.5米，而非"较近"）
4. **尺寸用具体数字**（8m x 6m，而非"大房间"）
5. **视角定义必须包含方向**（"西向东拍"而非"从这边看"）
6. **验证每个元素的可见性**（从指定视角能看到什么？）

## 贯穿案例：清代海军衙门办公室空间推理

以下展示如何对"清代海军衙门办公室，西向东拍"进行完整的5步空间推理：

### Step 1：建立空间容器

```
房间形状: 长方形
尺寸: 8m(东西) x 6m(南北)
高度: 3.5m
方向: 坐北朝南

坐标系:
  原点(0,0,0) = 西南角地面
  X轴: 西→东 (0-8m)
  Y轴: 南→北 (0-6m)
  Z轴: 地面→天花板 (0-3.5m)
```

### Step 2：放置固定元素

```
门:
  位置: 东墙中央
  坐标: X=8m, Y=3m(居中)
  尺寸: 1.2m宽 x 2.2m高
  开启: 向外推开

窗:
  位置: 南墙
  坐标: X=4m, Y=0m
  尺寸: 1.5m宽 x 1.2m高
  高度: 窗台0.8m, 窗顶2.0m
```

### Step 3：放置家具

```
办公桌:
  位置: 房间中后部
  坐标: X=5.5m, Y=3m
  尺寸: 1.5m宽 x 0.8m深 x 0.8m高
  朝向: 面向东(门的方向)

官帽椅:
  位置: 桌子正后方
  坐标: X=5.5m, Y=3.5m
  朝向: 面向东
```

### Step 4：放置人物

```
张十三:
  位置: 坐在官帽椅上
  坐标: X=5.5m, Y=3.5m, Z=0.45m(坐高)
  朝向: 面向东(面向门)
  视线: 向东看
```

### Step 5：确定视角

```
视角: 西向东拍
相机位置: X=2m, Y=3m, Z=1.6m(人眼高度)
相机朝向: 向东(+X方向)

可见性验证:
  ✓ 能看到: 东墙、门、办公桌、椅子、张十三、南墙(右侧)、北墙(左侧)
  ✗ 看不到: 西墙(在相机后方)

平面图验证:
┌─────────────────────────────┐
│           北(Y=6m)           │
│                              │
│  窗                          │
│  (南墙)                      │
│                              │
│         桌■  椅●(张十三)      │
│         X=5.5               │
│                              │
│    📷                        │
│    (相机X=2)                 │
│         →→→ 视线方向 →→→     │
│                              │
│           门                 │
│         (东墙X=8m)           │
│                              │
└─────────────────────────────┘
  西(X=0)              东(X=8m)

从相机位置(西向东看):
  - 前景: 相机附近(空地)
  - 中景: 办公桌+椅子+张十三
  - 背景: 东墙+门
  - 右侧: 南墙+窗
  - 左侧: 北墙
```

### 透视关系推导

```
从西向东拍的透视效果:
  - 东墙最远,看起来最小
  - 桌椅在中景,中等大小
  - 相机附近如果有物体,最大(前景)
  
景深层次:
  前景(0-3m): 空地,地面纹理
  中景(3-6m): 办公桌,椅子,人物
  背景(6-8m): 东墙,门,门框细节
```

## 空间推理检查清单

每次完成空间推理后，检查以下项目：

- □ 房间尺寸是否有具体数字？
- □ 是否建立了坐标系？
- □ 所有固定元素(门/窗)是否放置？
- □ 所有家具是否放置且有具体坐标？
- □ 人物位置是否合理？
- □ 视角是否使用绝对方向？
- □ 是否验证了每个元素的可见性？
- □ 摄影机后方的物体是否标记为不可见？
- □ 透视关系是否正确？
- □ 景深层次是否合理？
