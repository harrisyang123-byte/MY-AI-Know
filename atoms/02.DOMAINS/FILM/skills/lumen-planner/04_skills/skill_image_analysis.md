# Skill 3: Image Analysis (图像分析)

## 核心原理

### 什么是 Image Analysis？

系统化地"读懂"图像，提取关键信息的能力。

**本质**：这是一种"视觉解码"能力
- 输入：用户提供的参考图像
- 输出：结构化的、可用于生成的图像信息

### 为什么重要？

用户经常说："我想要这种风格"、"参考这张图"，但图像包含海量信息！

一张图可能包含：构图、色彩、光线、材质、风格、氛围、细节、技术参数...

如果不系统分析：
- 抓不住重点
- 遗漏关键信息
- 生成结果偏差大
- 无法精确复现

### 关键认知：看图不是"看一眼"，是"解剖"！

普通人看图："哦，这张图不错" → 结束

设计师看图：
- "这张图的构图是三分法"
- "主光源在左上45度"
- "色调是冷色调，饱和度低"
- "材质是哑光木质"
- "风格是写实主义"
- "氛围是宁静、专注"

Image Analysis是你的"设计师之眼"！你要像外科医生解剖人体一样解剖图像！

## 8维度分析框架

### 维度1：场景类型 (Scene Type)

**目的**：快速定位图像的基本类型

**分类**：
1. 室内场景 (Interior) — 办公室、卧室、客厅、餐厅等
2. 室外场景 (Exterior) — 街道、公园、建筑外观等
3. 产品摄影 (Product) — 纯背景产品、场景产品等
4. 人物摄影 (Portrait) — 肖像、全身、半身等
5. 自然风景 (Landscape) — 山川、海洋、森林等
6. 抽象/概念 (Abstract/Concept) — 艺术作品、概念图等

**分析要点**：
- 这是什么类型的场景？
- 主体是什么？（人/物/环境）
- 场景的功能是什么？

**为什么重要**：不同类型的场景，分析重点完全不同！
- 室内场景 → 重点：空间布局、家具、光线
- 产品摄影 → 重点：产品细节、材质、光影
- 人物摄影 → 重点：表情、姿态、服装

---

### 维度2：构图分析 (Composition)

**目的**：理解画面的结构和视觉引导

**关键要素**：

1. **构图法则**
   - 三分法 (Rule of Thirds)
   - 黄金分割 (Golden Ratio)
   - 中心构图 (Center Composition)
   - 对角线构图 (Diagonal)
   - 框架构图 (Frame)
   - 引导线构图 (Leading Lines)

2. **视角 (Viewpoint)**
   - 平视 (Eye-level)
   - 俯视 (High angle / Bird's eye)
   - 仰视 (Low angle / Worm's eye)
   - 角度（正面/侧面/45度等）

3. **景深层次 (Depth)**
   - 前景 (Foreground)
   - 中景 (Midground)
   - 背景 (Background)

4. **画面比例 (Aspect Ratio)**
   - 16:9 (横向宽屏)
   - 9:16 (竖向)
   - 1:1 (方形)
   - 4:3, 3:2 等

5. **留白 (Negative Space)**
   - 留白位置
   - 留白比例
   - 留白作用

**分析模板**：
```
构图法则: [识别使用了哪种构图法]
视角: [相机位置和角度]
景深层次:
  - 前景: [有什么]
  - 中景: [主体内容]
  - 背景: [背景内容]
画面比例: [宽高比]
留白: [哪里有留白，作用是什么]
```

**为什么重要**：构图决定了画面的"骨架"！如果构图错了，其他都白搭！

---

### 维度3：光线分析 (Lighting)

**目的**：理解光线的来源、性质和效果

**关键要素**：

1. **光源类型**
   - 自然光：日光、月光、天光
   - 人造光：烛光、灯光、霓虹灯

2. **光源数量**
   - 单光源
   - 多光源
   - 主光 + 补光 + 轮廓光 (3-point lighting)

3. **光源方向**
   - 顶光、侧光、逆光、前光、45度光

4. **光线性质**
   - 硬光 (Hard Light) — 明显阴影
   - 软光 (Soft Light) — 柔和阴影
   - 散射光 (Diffused Light) — 几乎无阴影

5. **色温 (Color Temperature)**
   - 暖光 (Warm) — 2700-3500K
   - 中性 (Neutral) — 4000-5000K
   - 冷光 (Cool) — 5500-7000K

6. **光线强度**
   - 强光、中等、弱光

7. **阴影**
   - 阴影方向、硬度、长度

**分析模板**：
```
主光源:
  - 类型: [自然光/人造光]
  - 位置: [方向和高度]
  - 性质: [硬光/软光]
  - 色温: [暖/中/冷]
  - 强度: [强/中/弱]

补光源 (如果有):
  - [同上]

阴影:
  - 方向: [阴影指向哪里]
  - 硬度: [清晰/模糊]
  - 长度: [长/短]

整体氛围:
  - [明亮/昏暗/神秘/温暖等]
```

**为什么重要**：光线决定了画面的"灵魂"！同样的场景，不同的光线，完全不同的感觉！

---

### 维度4：色彩分析 (Color)

**目的**：理解色彩的使用和情感表达

**关键要素**：

1. **色调 (Hue)**
   - 主色调：画面的主导颜色
   - 辅助色：次要颜色
   - 点缀色：小面积强调色

2. **饱和度 (Saturation)**
   - 高饱和：鲜艳、活力
   - 中饱和：自然、平衡
   - 低饱和：柔和、复古

3. **明度 (Value/Brightness)**
   - 高调 (High Key)：明亮、轻快
   - 中调 (Mid Key)：平衡
   - 低调 (Low Key)：昏暗、神秘

4. **色彩关系**
   - 单色 (Monochromatic)
   - 类似色 (Analogous)
   - 互补色 (Complementary)
   - 三角色 (Triadic)

5. **色温对比**
   - 暖色调：红、橙、黄
   - 冷色调：蓝、绿、紫
   - 冷暖对比

6. **色彩情感**
   - 红色：热情、危险、力量
   - 蓝色：冷静、专业、忧郁
   - 黄色：快乐、温暖、警示
   - 绿色：自然、和平、生机
   - 紫色：神秘、高贵、创意

**分析模板**：
```
主色调: [主要颜色及占比]
辅助色: [次要颜色及占比]
点缀色: [强调色及占比]

饱和度: [高/中/低]
明度: [高调/中调/低调]

色彩关系: [单色/类似/互补等]
色温: [暖/冷/冷暖对比]

色彩情感: [传达什么情绪]

具体色值 (如果需要精确):
  - 主色: #RRGGBB
  - 辅助色: #RRGGBB
  - 点缀色: #RRGGBB
```

**为什么重要**：色彩决定了画面的"情绪"！色彩是最直接的情感传达工具！

---

### 维度5：材质分析 (Material/Texture)

**目的**：理解物体的表面质感和物理属性

**关键要素**：

1. **材质类型**
   - 木质 (Wood)：纹理、颜色、光泽
   - 金属 (Metal)：反射、高光、颜色
   - 布料 (Fabric)：纹理、褶皱、柔软度
   - 石材 (Stone)：纹理、粗糙度
   - 玻璃 (Glass)：透明度、反射、折射
   - 皮革 (Leather)：纹理、光泽
   - 塑料 (Plastic)：光泽、颜色
   - 纸张 (Paper)：纹理、颜色

2. **表面属性**
   - 光泽度：高光/半光/哑光
   - 粗糙度：光滑/中等/粗糙
   - 透明度：不透明/半透明/透明

3. **纹理细节**
   - 木纹、布纹、石纹等
   - 纹理方向、密度、清晰度

4. **磨损/年代感**
   - 崭新、轻微磨损、明显磨损、古旧

**分析模板**：
```
主要材质:
  材质1: [类型]
    - 光泽度: [高光/半光/哑光]
    - 粗糙度: [光滑/中等/粗糙]
    - 纹理: [描述]
    - 年代感: [新/旧]

材质对比:
  - [硬vs软, 光滑vs粗糙, 新vs旧等]

整体质感:
  - [奢华/朴素/现代/古典等]
```

**为什么重要**：材质决定了画面的"触感"！材质让画面从"平面"变"立体"！

---

### 维度6：风格识别 (Style)

**目的**：识别图像的艺术风格和视觉语言

**关键要素**：

1. **艺术风格**
   - 写实主义 (Realism)
   - 印象派 (Impressionism)
   - 极简主义 (Minimalism)
   - 赛博朋克 (Cyberpunk)
   - 蒸汽朋克 (Steampunk)
   - 动漫/漫画 (Anime/Manga)
   - 水彩 (Watercolor)
   - 油画 (Oil Painting)

2. **视觉语言**
   - 线条风格
   - 色彩处理
   - 光影手法
   - 透视方式

3. **参考作品/艺术家**
   - 类似风格的已知作品
   - 类似风格的艺术家
   - 可以用作参考的影视/游戏

**分析模板**：
```
艺术风格: [主要风格]
视觉语言: [线条/色彩/光影/透视特点]
参考作品: [类似风格的已知作品]
参考艺术家: [类似风格的艺术家]
```

---

### 维度7：氛围与情感 (Mood & Emotion)

**目的**：理解图像传达的情绪和氛围

**关键要素**：

1. **整体氛围**
   - 宁静/热闹/神秘/温暖/冷峻/紧张/轻松

2. **时间感**
   - 清晨/正午/黄昏/深夜
   - 春/夏/秋/冬

3. **情感关键词**
   - 3-5个描述情感的关键词

**分析模板**：
```
整体氛围: [描述]
时间感: [时间/季节]
情感关键词: [关键词1, 关键词2, 关键词3]
```

---

### 维度8：技术参数 (Technical Parameters)

**目的**：提取可复用的技术信息

**关键要素**：

1. **分辨率和比例**
2. **焦距估计**（广角/标准/长焦）
3. **景深**（浅/中/深）
4. **动态范围**（HDR/标准/低对比）
5. **噪点/颗粒感**

**分析模板**：
```
分辨率: [估计]
画面比例: [宽高比]
焦距: [广角/标准/长焦]
景深: [浅/中/深]
动态范围: [HDR/标准/低对比]
噪点: [有/无/胶片感]
```

## 分析流程

### Step 1：快速分类

一眼判断场景类型（室内/室外/产品/人物/风景/抽象）

### Step 2：8维度逐一分析

按维度1→8的顺序，逐个维度分析

### Step 3：提取关键信息

从8个维度中提取对当前任务最有用的信息

### Step 4：转化为提示词

将分析结果转化为可用的提示词元素

### Step 5：验证完整性

检查是否有遗漏的关键信息

## 贯穿案例：清代海军衙门办公室8维度完整分析

以下展示如何对一张已生成的图像进行完整的8维度分析，并提取可复用的提示词元素：

**场景**：一张清代海军衙门办公室的图像，西向东视角，深夜烛光场景

### 维度1：场景类型分析

```
场景类型: 室内场景 — 历史场景
子类型: 官方办公空间
时代: 清代 (1644-1912)
地域: 中国北方沿海
功能: 军事行政办公
```

### 维度2：构图分析

```
构图法则: 中心对称构图
视角: 平视 (eye-level), 西向东
景深层次:
  前景: 办公桌及桌面物品 (0-2m)
  中景: 人物张十三及椅子 (2-4m)
  背景: 东墙、门、窗 (4-8m)
比例: 人物占画面约40%, 桌面占30%, 墙面占30%
视觉引导: 烛光 → 人物面部 → 桌面文件 → 门
```

### 维度3：光线分析

```
主光源: 烛光
  类型: 点光源
  色温: 2800K (暖黄)
  位置: 桌面中央偏前
  强度: 95%总光量
  效果: 柔和琥珀色光晕, 照亮人物面部和上半身

补光源: 月光+雪地反射
  类型: 方向光
  色温: 6000K (冷蓝)
  位置: 南墙窗户进入
  强度: 5%总光量
  效果: 边缘冷色高光, 增加深度和氛围对比

阴影:
  类型: 柔和阴影
  方向: 向西投射(光源在东)
  特征: 边缘柔和, 渐变过渡

氛围: 低调 (low-key), 明暗对比强烈 (chiaroscuro)
```

### 维度4：色彩分析

```
主色调: 深棕色 (60%) — 木质家具和建筑
辅助色: 暗红色 (20%) — 门漆和官服
点缀色: 琥珀色 (10%) — 烛光高光, 蓝灰色 (5%) — 月光边缘, 墨黑色 (5%) — 阴影

饱和度: 低饱和度
明度: 低调 (low-key)

色彩关系: 类似色 (暖色系为主)
色温: 整体暖色调, 局部冷色点缀 (冷暖对比)

色彩情感: 庄重、肃穆、历史厚重感

具体色值:
  - 木质深棕: #3E2723
  - 门漆暗红: #5D1014
  - 烛光琥珀: #FFB74D
  - 月光蓝灰: #546E7A
  - 深处墨黑: #1A1A1A
```

### 维度5：材质分析

```
主要材质:
  木质 (桌面/椅子/书架):
    - 光泽度: 半光 (年代感)
    - 粗糙度: 中等 (手工打磨)
    - 纹理: 细密木纹, 深色胡桃木
    - 年代感: 明显磨损, 包浆

  漆器 (门/窗框):
    - 光泽度: 高光 (局部)
    - 粗糙度: 光滑
    - 纹理: 均匀漆面
    - 年代感: 轻微剥落

  丝绸 (官服):
    - 光泽度: 柔和光泽
    - 粗糙度: 光滑
    - 纹理: 细密织物纹理
    - 年代感: 崭新

  纸张 (文件/窗纸):
    - 光泽度: 哑光
    - 粗糙度: 中等
    - 纹理: 手工纸纤维
    - 年代感: 轻微泛黄

材质对比:
  - 硬(木质) vs 软(丝绸)
  - 光滑(漆器) vs 粗糙(纸张)
  - 新(官服) vs 旧(家具)

整体质感: 古朴、厚重、官式典雅
```

### 维度6：风格识别

```
艺术风格: 超写实主义 (Hyperrealism)
视觉语言:
  线条: 精确细腻, 无风格化变形
  色彩: 自然主义色彩, 忠实还原
  光影: 电影级布光, 明暗对比强烈
  透视: 严格一点透视

参考作品: 《长安三万里》背景美术
参考艺术家: Lights Animation Studio 美术风格
参考影视: 历史题材电影场景设计
```

### 维度7：氛围与情感

```
整体氛围: 肃穆、安静、深沉
时间感: 深夜, 冬季
情感关键词: 庄严, 孤寂, 历史厚重, 使命感

叙事暗示: 深夜仍在办公 → 勤勉/紧急事务
烛光与月光对比 → 人事与天命
独处空间 → 孤独但坚守
```

### 维度8：技术参数

```
分辨率: 高清 (4K+)
画面比例: 16:9
焦距: 35mm (标准广角, 室内空间感)
景深: 中等景深 (前景桌面到背景墙面都清晰)
动态范围: HDR (同时保留烛光高光和阴影细节)
噪点: 无 (干净画面)
```

### 可复用提示词元素提取

```
【场景元素】
Qing Dynasty naval office interior, coastal fortress,
traditional Chinese architecture

【构图元素】
center-symmetry composition, eye-level view,
west-to-east viewpoint, chiaroscuro lighting

【光线元素】
candlelight primary 2800K (95%), moonlight secondary 6000K (5%),
warm amber glow, cool blue rim light, low-key atmosphere

【色彩元素】
deep brown dominant, dark red accent, amber highlights,
low saturation, warm tone with cool accents

【材质元素】
dark walnut wood, lacquered surfaces, silk fabric,
handmade paper, aged patina

【风格元素】
hyperrealistic, cinematic lighting, 'Chang'an 30000 Miles' style,
Lights Animation Studio aesthetic

【技术元素】
35mm lens, medium depth of field, HDR,
dramatic low-light photography, atmospheric perspective
```

## 分析质量检查清单

每次完成图像分析后，检查以下项目：

- □ 8个维度是否全部覆盖？
- □ 每个维度的分析是否足够具体（有具体数值/名称）？
- □ 色温是否量化（K值）？
- □ 光线强度是否量化（百分比）？
- □ 材质是否描述了表面属性？
- □ 是否提取了可复用的提示词元素？
- □ 分析结果是否可以指导下一次生成？
- □ 是否识别了当前图像的问题（与需求不符之处）？
