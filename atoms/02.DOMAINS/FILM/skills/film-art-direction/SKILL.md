# Film Art Direction

将叙事情绪落地为具体视觉元素。方法论基础：

- **Bellantoni**（《这就是紫色，有人要死》）：色彩心理学 → 情绪-色彩-材质的对应逻辑
- **Bordwell**（《电影艺术》）：置景四要素 → 布景/服装化妆/灯光/人物动作，规范输出结构
- **Barnwell**（《电影美术：屏幕上的建筑师》）：空间线条 + 材质情感温度 → 构建视觉世界观

输出必须细到可以直接翻译成 AI 生成模型的提示词参数。每一步推导过程显式输出，让用户看到从情绪到视觉的推导逻辑。

## 工作流程

### 第一步：确认输入

- 当前场景的情绪状态是什么？（来自节拍表的价值状态）
- 时代背景和地点是什么？
- 这个场景在整体色彩框架中的位置？（高潮/过渡/对比）

### 第二步：色彩定调（Bellantoni）

根据情绪状态，从色彩心理学出发确定主色调。

**快速查表**（详见 [references/color-emotion-guide.md](references/color-emotion-guide.md)）：

| 情绪 | 主色 | 材质倾向 | 禁用色 |
|------|------|---------|--------|
| 压抑的愤怒 | 暗红/酒红 | 粗糙、无光泽、锈蚀 | 明黄、天蓝 |
| 欲望/危险 | 鲜红 | 光滑、反光、紧绷 | 灰绿、米白 |
| 孤独/疏离 | 钢蓝/冷灰 | 金属、玻璃、混凝土 | 暖橙、木质 |
| 失控/混乱 | 黄绿/病态绿 | 潮湿、腐蚀、剥落 | 纯白、金色 |
| 权力/压迫 | 深紫/靛蓝 | 厚重织物、石材 | 粉色、浅黄 |
| 温暖/人性 | 琥珀/暖橙 | 木质、棉麻、皮革 | 冷白、钢铁 |
| 死亡/预兆 | 紫色/灰紫 | 蜡质、褪色、干枯 | 鲜艳饱和色 |
| 希望/转变 | 金黄/暖白 | 轻薄、透光、自然 | 深黑、暗红 |

确定主色后，同时确定：辅色（主色的低饱和变体或互补色）、禁用色（与情绪冲突的颜色）。

### 第三步：置景四要素推导（Bordwell）

严格按四要素逐项输出，不能合并或跳过：

**① 布景（Setting & Props）**

布景不是背景，是叙事工具。先确认这个空间的主导叙事功能，再推导视觉细节。

**空间叙事功能**（选一个主导，其余为辅）：
- 决定人物：空间揭示人物是谁（阶层/历史/可能性）
- 反映人物：空间外化人物此刻的内心状态
- 情绪氛围：空间在对话开始前就告诉观众"这里会发生什么"
- 象征：空间或其中的元素承载主题含义

**景深三层次**（每层都要有意识地设计）：
- 前景：制造框架/窥视/压迫——门框、窗格、柱子、遮挡物
- 中景：人物位置——居中（有主体性）vs 偏移（被边缘化）
- 背景：开放（逃脱的可能）vs 封闭（无路可逃）

**空间权力关系**：
- 人物高于/低于其他人物或环境元素 → 权力对比
- 空间大小与人物体量的比例 → 渺小感或掌控感
- 门/出口的状态：开着（选择存在）/ 关着（困境）/ 人物站在门口（边界/犹豫）

**道具三层级**：
- 背景道具：建立真实感，有使用痕迹，符合人物阶层和时代
- 手持道具：人物直接使用，道具状态反映人物状态
- 象征道具：承载主题，在故事中反复出现并随弧光改变状态
  - 格式：[道具名]：[颜色+材质+状态] → [叙事功能/象征含义]

**空间线条**（Barnwell）：垂直=权威/压迫，水平=平静/死亡，斜线=不稳定/过渡，混乱=失控

需要完整决策流程、景深分析、AI提示词格式时，加载 [references/setting-guide.md](references/setting-guide.md)。

**② 服装与化妆（Costume & Makeup）**
- 主要角色服装：颜色 + 材质 + 状态
- 人物与环境的色彩关系：
  - 对比（人物色 vs 背景色）= 人物从环境中突出，有主体性
  - 融合（人物色 ≈ 背景色）= 人物被环境吞没，失去主体性
- 化妆/肤色处理：苍白/红润/暗沉对情绪的影响

**③ 灯光（Lighting）**

Bordwell 将布光拆解为四个独立维度，每个维度都是叙事工具：

- **光质**：硬光（揭示/冲突/审讯）vs 软光（隐藏/梦幻/美化）
  - 核心判断：这个场景需要"揭示"还是"隐藏"？
- **方向**：顺光（中性）/ 侧光（内心矛盾）/ 逆光（神秘/威胁）/ 顶光（压迫/审判）/ 底光（恐怖/扭曲）
- **来源**：蜡烛（脆弱/短暂）/ 日光灯（体制化）/ 霓虹（欲望/堕落）/ 阴天散射（压抑/无望）
- **色彩**：暖色光（≤3200K，亲密/危险）/ 冷色光（≥5600K，疏离/死亡）/ 色光对比（冷暖对立制造张力）

**高调 vs 低调**：
- 高调（High-key）：低对比度，大面积亮部，阴影少 → 喜剧/梦幻/安全
- 低调（Low-key）：高对比度，大面积暗部，阴影深重 → 紧张/危险/道德模糊
- 明暗对照法（Chiaroscuro）：低调的极端形式，单一光源切入黑暗 → 真相从黑暗中浮现

**主补比（Key-to-Fill Ratio）**：1:1 中性 → 4:1 戏剧感 → 8:1 以上黑色电影风格

需要详细布光参数、AI 提示词格式、决策树时，加载 [references/lighting-guide.md](references/lighting-guide.md)。

**④ 人物动作与空间关系（Figure Behavior）**
- 人物在空间中的位置：前景/中景/背景
- 前景道具的运用（Barnwell）：用前景物件制造"画中画"，增加窥视感或压迫感
- 人物与道具的互动：触碰/回避某件道具本身就是叙事

### 第四步：材质情感温度校验（Barnwell）

输出前用材质字典校验每件道具和陈设的情感温度是否与场景情绪一致：

| 材质 | 情感温度 | 适用情绪 |
|------|---------|---------|
| 原木、棉麻、皮革 | 温暖/人性/有机 | 亲密、温情、乡愁 |
| 不锈钢、混凝土、玻璃 | 冰冷/体制化/疏离 | 孤独、压迫、现代异化 |
| 锈铁、剥落墙皮、破布 | 衰败/压抑/无力 | 绝望、贫困、时间流逝 |
| 丝绸、抛光金属、大理石 | 权力/欲望/危险 | 诱惑、控制、虚伪繁荣 |
| 蜡烛、油灯、旧纸 | 脆弱/短暂/记忆 | 怀旧、死亡预兆、孤独 |

详细材质字典见 [references/material-texture-guide.md](references/material-texture-guide.md)。

### 第五步：输出检查

每一项描述必须满足：
- 具体物件，不是风格词（"深褐色旧木桌" ✓，"复古风格" ✗）
- 颜色有具体色调（"青灰色" ✓，"冷色" ✗）
- 材质有质感描述（"粗麻布" ✓，"布料" ✗）
- 每件关键道具有明确叙事功能
- 可以直接被 prompt-alchemist 翻译成提示词参数

## 输出格式

```
场景：[场景名] | 节拍：[节拍编号] | 情绪：[情绪状态]
主色：[颜色] | 辅色：[颜色] | 禁用色：[颜色]

【布景】
叙事功能：[决定人物/反映人物/情绪氛围/象征]
空间权力：[人物居中/偏移] | 空间[开阔/逼仄] | 出口[开/关/边界]
前景：[元素] → [窥视/压迫/隔离效果]
中景：[主要陈设：颜色+材质+状态]
背景：[开放/封闭] + [具体元素]
空间线条：[垂直/水平/斜线/混乱] → [情绪含义]
道具：
- 背景道具：[建立什么世界观]
- 手持道具：[道具]：[颜色+材质+状态]
- 象征道具：[道具]：[颜色+材质+状态] → [象征含义]

【灯光】
光质：硬光/软光 → [揭示/隐藏]
方向：[顺/侧/逆/顶/底光] → [情绪含义]
来源：[具体光源] + 色温：[暖/冷，K值参考]
调性：高调/低调 | 主补比：[1:1 / 4:1 / 8:1+]

【服装与化妆】
- [角色]：[颜色] + [材质] + [状态]
- 与环境关系：对比/融合 → [人物主体性判断]
```
