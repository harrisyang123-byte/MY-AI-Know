# 布景深度指南

基于 Bordwell《电影艺术》的 setting 理论 + Barnwell《电影美术》的空间实践 + Uli Hanisch 等制作设计师的实践经验。

## 核心原则

**空间不是中性的。** 房间携带意义、携带压力，可以强化或重新定向场景中正在发生的事情。布景不是"给人物找个地方站"，而是用空间来讲故事。

---

## 一、Setting 的六种叙事功能（Bordwell）

在设计布景前，先确认这个空间在故事里承担哪种功能——同一个场景可以同时承担多种功能，但必须有主导功能。

### 1. 决定人物（Setting as Determiner of Character）
空间塑造了人物是谁。人物的成长环境、阶层、历史，都写在他所处的空间里。
- 应用：人物第一次出现的空间，必须能告诉观众"这个人是谁"
- 问题：这个空间如何限制或塑造了人物的可能性？

### 2. 反映人物（Setting as Reflection of Character）
空间是人物内心状态的外化。人物的情绪、心理、道德状态，通过他所处的空间被可视化。
- 应用：人物情绪低谷时，空间应该是破败的/逼仄的/冷色的
- 问题：此刻人物的内心状态，如何在空间里找到对应物？

### 3. 建立真实感（Setting for Verisimilitude）
空间的细节密度决定观众是否相信这个世界存在。
- 应用：背景道具的历史感、使用痕迹、不完美细节
- 问题：这个空间里有没有"被使用过"的痕迹？

### 4. 纯视觉冲击（Setting for Visual Impact）
有时空间本身就是奇观，是观众想看的东西。
- 应用：宏大场景、极端环境、视觉上令人震撼的地点
- 注意：视觉冲击不能凌驾于叙事功能之上，否则变成旅游宣传片

### 5. 制造情绪氛围（Setting to Create Emotional Atmosphere）
空间的整体感受决定场景的情绪基调，在对话开始前就已经告诉观众"这里会发生什么"。
- 应用：恐怖片的孤立建筑、黑色电影的潮湿街道
- 问题：观众进入这个空间的第一秒，应该感受到什么？

### 6. 作为象征（Setting as Symbol）
空间或其中的元素代表超越字面意义的东西。
- 应用：《办公室》的格子间代表无意义的存在；监狱代表体制的囚禁
- 问题：这个空间里有没有可以承载主题的象征物？

## 二、景深三层次（Depth Planes）

每个场景的空间都有三个层次，每个层次承担不同的叙事任务。这是 Barnwell 空间理论的核心。

```
前景（Foreground）
├── 距离摄影机最近
├── 叙事功能：制造框架感、窥视感、压迫感
├── 常见元素：门框、窗格、栏杆、柱子、前景道具
└── 情绪效果：
    ├── 遮挡物（门框/窗格）→ 窥视感，人物处于被观察状态
    ├── 大体量物件（柱子/家具）→ 压迫感，人物被环境挤压
    └── 透明/半透明物件（玻璃/纱帘）→ 隔离感，两个世界的边界

中景（Middle Ground）
├── 人物通常所在的层次
├── 叙事功能：展示人物与环境的关系
├── 关键问题：人物在中景中是居中（有主体性）还是偏移（被边缘化）？
└── 人物与空间的权力关系：
    ├── 人物占据空间中心 → 掌控感、主体性
    ├── 人物被挤到画面一角 → 被压制、失去主体性
    └── 人物与背景融合（颜色/材质相近）→ 被环境吞没

背景（Background）
├── 距离摄影机最远
├── 叙事功能：建立世界观、提供上下文、暗示画外空间
├── 常见元素：窗外景色、走廊深处、远处的人群
└── 情绪效果：
    ├── 开放背景（窗外天空/远景）→ 逃脱的可能性、希望
    ├── 封闭背景（实墙/黑暗）→ 无路可逃、绝望
    └── 模糊背景（浅景深）→ 人物与世界的隔绝
```

**三层次的协同**：前景压迫 + 中景人物偏移 + 背景封闭 = 极度压迫感。前景开放 + 中景人物居中 + 背景有光 = 希望与主体性。

## 三、道具的三种类型

道具不是摆设，每件道具都有其叙事层级。

### 类型一：背景道具（Background Props）
- 功能：建立真实感，提供世界观信息
- 不需要被特别注意，但缺少它们世界会显得空洞
- 设计原则：有使用痕迹、有历史感、符合人物的阶层和时代
- 示例：书架上的书（反映人物的知识背景）、墙上的照片（暗示人物关系）

### 类型二：手持道具（Hand Props）
- 功能：人物直接使用，承载行为和情绪
- 必须有具体的视觉描述（颜色+材质+状态）
- 设计原则：道具的状态反映人物的状态（破旧的钱包=经济困境，颤抖的手持杯子=恐惧）
- 示例：武器、食物、信件、手机、钥匙

### 类型三：象征道具（Symbolic Props）
- 功能：承载主题，在多个场景中反复出现，积累意义
- 这是最重要的道具类型，需要在整个故事中追踪
- 设计原则：第一次出现时建立含义，之后每次出现都应该有变化（状态改变=意义改变）
- 示例：《公民凯恩》的雪球（童年/失去的纯真）、《教父》的橙子（死亡预兆）

**道具设计的核心问题**：
1. 这件道具在这个场景里是哪种类型？
2. 如果是象征道具，它在整个故事中的状态变化轨迹是什么？
3. 人物与这件道具的关系（触碰/回避/破坏）本身就是叙事

## 四、空间与权力关系

空间的几何结构直接编码了权力关系。这是 Barnwell 最核心的洞察之一。

### 垂直维度（高低）
- 高处的人物 → 权力、优势、俯视
- 低处的人物 → 弱势、被压制、仰视
- 同一水平线 → 平等关系
- 应用：审讯场景中，审讯者站立/被审讯者坐下；权贵坐在高台上

### 水平维度（中心/边缘）
- 画面中心的人物 → 主体性、掌控感
- 画面边缘的人物 → 被边缘化、失去主体性
- 两人对称居中 → 势均力敌的对抗
- 应用：人物弧光的变化可以通过在画面中的位置变化来体现

### 空间大小（开阔/逼仄）
- 开阔空间 → 自由、孤独（取决于人物是否孤立）、权力（如果人物掌控空间）
- 逼仄空间 → 压迫、亲密（取决于关系）、无法逃脱
- 空间与人物体量的比例：人物在巨大空间中显得渺小 → 孤独/无力

### 门与出口
- 开着的门 → 选择、逃脱的可能性
- 关着的门 → 困境、无路可逃
- 人物站在门口 → 处于两个世界的边界，尚未做出选择
- 人物背对门 → 放弃逃脱，接受当前处境

## 五、空间随弧光变化

同一个人物在不同叙事阶段，所处的空间应该随弧光发生变化。这是布景设计最容易被忽视的维度。

| 弧光阶段 | 空间特征 | 具体表现 |
|---------|---------|---------|
| 故事开始（旧世界） | 熟悉、舒适、但有隐藏的问题 | 整洁但压抑，暖色但封闭 |
| 催化事件后 | 熟悉的空间开始变得陌生 | 同样的房间，但光线变了，道具位置变了 |
| 第二幕（新世界） | 陌生、挑战性、与人物格格不入 | 冷色、开阔或逼仄（取决于类型） |
| 最低点 | 最压迫、最封闭、最黑暗 | 极度逼仄，或极度空旷（孤立感） |
| 第三幕（转变后） | 空间开始与人物和解 | 同样的空间，但人物开始掌控它 |
| 结局 | 与开场呼应，但已经改变 | 开场的空间再次出现，但细节不同 |

## 六、布景设计决策流程

```
1. 这个场景的叙事功能是什么？（六种功能中的哪一种？）
   ↓
2. 人物此刻的弧光状态？（掌控/失控/转变中）
   ↓
3. 空间与人物的权力关系？（人物掌控空间 or 被空间压制）
   ↓
4. 三层次如何分配？
   ├── 前景：制造什么框架效果？
   ├── 中景：人物在哪里，居中还是偏移？
   └── 背景：开放还是封闭？
   ↓
5. 道具清单：
   ├── 背景道具：建立什么世界观？
   ├── 手持道具：人物用什么，状态如何？
   └── 象征道具：这个场景里的象征物是什么，状态有变化吗？
   ↓
6. 空间线条主导方向？（垂直/水平/斜线/混乱）
   ↓
7. 材质情感温度是否与场景情绪一致？（查 material-texture-guide.md）
```

## 七、AI 提示词中的布景描述格式

布景描述需要覆盖三个层次，格式：

```
[前景元素，制造框架/压迫/窥视], [中景：人物位置+主要陈设，颜色+材质+状态], [背景：开放/封闭+具体元素]
空间线条：[主导方向]
关键道具：[道具名] [颜色+材质+状态]（[叙事功能/象征含义]）
```

示例：
```
前景：weathered wooden door frame, peeling red paint, partially blocking view
中景：figure standing off-center left, surrounded by grey concrete walls, cracked plaster
背景：single small window, dark exterior, iron bars
空间线条：vertical (压迫/囚禁)
关键道具：worn leather briefcase on floor, brass clasp broken (曾经的体面，现已失去)
```
