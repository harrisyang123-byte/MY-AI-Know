# Film Performance

表演节奏先于分镜——先想清楚人物在这场戏里**想要什么、在对对方做什么**，分镜才有切换的依据。

方法论基础：
- **Stanislavski**（《演员的自我修养》）：目标/意图/给定情境/贯穿动作，表演的内在逻辑
- **Weston**（《导演演员》）：行动动词 vs 结果方向，脊柱，情绪事件

---

## 核心原则：行动动词，不是结果描述

Weston 最重要的区分：

**结果方向（Result Direction）**——告诉演员应该感受什么或表现什么：
- ✗ `表现得更悲伤一点`
- ✗ `这里要愤怒`
- ✗ `表演幅度 20%`

**行动方向（Action Direction）**——告诉演员在对对方做什么：
- ✓ `他在说服她留下`
- ✓ `她在测试他是否还爱她`
- ✓ `他在用沉默惩罚她`

行动动词让演员专注于对手，而不是自己的情绪。情绪是行动的结果，不是起点。

## 模式一：全片定调（项目开始时用一次）

输入：节拍表 + 人物三维骨架。

### 第一步：建立人物脊柱（Weston）

每个核心人物有一个贯穿全片的核心需求——**脊柱（Spine）**。脊柱不是外部目标，而是驱动所有行为的内在需求。

**脊柱的格式**：用一个动词短语表达，必须是主动的、可以在每个场景中具体执行的。

```
人物：[姓名]
脊柱：[动词短语，如"证明自己值得被爱" / "控制一切以避免再次受伤" / "找到一个真正理解他的人"]
```

脊柱来自人物三维骨架的心理维度（核心欲望 + 最大恐惧）。每个场景的行动都是脊柱在当下情境中的具体表现。

### 第二步：定义给定情境（Stanislavski）

给定情境（Given Circumstances）是人物在故事开始时所处的客观条件——这些条件决定了人物的行动逻辑。

```
时代背景：[时间、地点、社会环境]
人物处境：[人物在故事开始时的具体状态]
人物关系：[与其他核心人物的关系历史]
```

### 第三步：建立表演风格基准

定义这部影片的整体表演策略——不是幅度百分比，而是**行动策略**：

**克制内敛**：
- 行动藏在细节里——一个放下杯子的动作比一句台词更重要
- 沉默是常态，开口意味着重要
- 情绪通过身体的细微变化传递，而非面部表情的夸张
- 参考：侯孝贤、是枝裕和、王家卫

**外化爆发**：
- 行动直接、幅度大，冲突激烈
- 情绪通过声音和身体的全力投入传递
- 参考：张艺谋、科波拉、阿莫多瓦

**混合策略**：
- 日常戏克制，情绪高潮外化，形成对比张力
- 需要明确定义哪些节拍是转折点

同时定义：
- **沉默策略**：沉默是常态还是例外？沉默出现时意味着什么？
- **倾听密度**：人物是真正在听对方说话，还是在等待说话的机会？（Weston：倾听是表演的核心）

### 第四步：全片张力曲线

对照节拍表，用**行动强度**（而非幅度百分比）标注关键节点：

```
节拍 1-8（积累段）：行动策略——[人物在用什么方式压抑/隐藏/等待]
节拍 9（中点）：行动转折——[什么事件改变了人物的行动策略]
节拍 11（一无所有）：行动爆发——[人物放弃了什么防御，做出了什么]
节拍 12-13（余韵段）：行动收缩——[人物如何处理爆发后的状态]
节拍 14（结局）：行动完成——[人物的脊柱在这里得到了什么结果]
```

**输出格式**：

```
人物脊柱：
- [人物A]：[脊柱动词短语]
- [人物B]：[脊柱动词短语]

给定情境：[简述]

表演风格：[克制内敛 / 外化爆发 / 混合策略]
沉默策略：[具体描述]
倾听密度：[具体描述]

张力曲线：
节拍 [X]（[阶段]）：[行动策略描述]
节拍 [X]（[转折]）：[行动转折描述]
...
```

## 模式二：场景执行（每次分镜时调用）

输入：当前场景的叙事目的 + 全片定调基准 + 当前场景的剧本台词。

### 第一步：确认场景的情绪事件（Weston）

情绪事件（Emotional Event）是场景中最重要的情感转折——这是整场戏存在的理由。

- 这场戏的情绪事件是什么？（不是"他们吵架了"，而是"她第一次意识到他不再爱她了"）
- 情绪事件发生在场景的哪个时刻？（这个时刻是镜头设计的核心）

### 第二步：确认人物意图（Stanislavski + Weston）

每个人物在这场戏里的**场景目标（Scene Objective）**：

```
人物A 的场景目标：[他在这场戏里想要什么——用动词短语]
人物A 的行动策略：[他用什么方式追求这个目标——行动动词]

人物B 的场景目标：[她在这场戏里想要什么]
人物B 的行动策略：[她用什么方式追求这个目标]
```

**行动动词的选择原则**（Weston）：
- 必须是及物动词，指向对方：`说服`、`测试`、`惩罚`、`引诱`、`保护`、`挑战`、`安慰`、`回避`
- 不能是情绪描述：`悲伤`、`愤怒`、`紧张`（这些是结果，不是行动）
- 不能是状态描述：`等待`、`思考`（太被动，没有指向性）

### 第三步：设计行动节拍

行动节拍是镜头切换的依据——每个节拍对应一个意图的转变或一个行动的完成。

```
节拍 1：[人物] [行动动词] [对象] → [对方的反应] → 切换时机：[前/中/后]
节拍 2：[人物] [行动动词] [对象] → [对方的反应] → 切换时机：[前/中/后]
...
情绪事件：节拍 [X]，[描述发生了什么]
```

**切换时机的叙事含义**：
- 动作完成**前**切：制造悬念，观众没看到结果
- 动作完成**后**切：确认结果，然后推进
- 动作**中**切：强调过程，节奏感强

### 第四步：对照基准检查

- 这场戏的行动策略是否符合全片风格基准？
- 人物的场景目标是否服务于他的脊柱？
- 情绪事件是否清晰？（如果说不出情绪事件，这场戏可能没有存在的理由）
- 如果偏离基准，叙事理由是什么？

### 输出格式

```
场景：[场景名] | 节拍位置：[节拍编号]

情绪事件：[这场戏最重要的情感转折，发生在节拍X]

人物意图：
- [人物A]：[场景目标] / 行动策略：[行动动词]
- [人物B]：[场景目标] / 行动策略：[行动动词]

行动节拍：
1. [人物]：[行动动词] [对象] — 切换时机：[前/中/后]
2. [人物]：[行动动词] [对象] — 切换时机：[前/中/后]
...
情绪事件：节拍 [X]

基准符合：[是/否] | 偏离原因：[如有]
```

## 常见错误

- **给结果方向**：`表现得更悲伤` → 改为行动动词：`他在用沉默惩罚她`
- **行动动词太被动**：`等待`、`思考` → 改为有指向性的动词：`测试`、`说服`
- **情绪事件不清晰**：说不出这场戏最重要的情感转折 → 这场戏可能是废戏
- **场景目标和脊柱脱节**：人物在这场戏里的行动和他的核心需求没有关系 → 重新检查脊柱
- **忽略倾听**：只设计说话的人的行动，忽略听的人的反应 → 倾听本身就是行动
