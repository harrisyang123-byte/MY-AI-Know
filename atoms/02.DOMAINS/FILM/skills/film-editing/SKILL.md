# Film Editing

将分镜稿转化为可执行的剪辑方案。理论基础：Walter Murch《眨眼之间》六准则、Karen Pearlman《剪辑的节奏》节奏三层次、Ken Dancyger 类型剪辑策略。

**使用前提**：分镜稿（04-directing.md）已完成，每个镜头有景别、时长预估、叙事目的。

---

## 核心决策框架：Murch 六准则

每一个切点都用这六条标准评估，优先级从高到低：

| 准则 | 权重 | 问题 |
|------|------|------|
| 情绪（Emotion） | 51% | 这一刀是否保住了观众此刻应有的情绪状态？ |
| 故事（Story） | 23% | 这一刀是否推动了故事向前？ |
| 节奏（Rhythm） | 10% | 这一刀落在节奏上对吗？ |
| 视线轨迹（Eye Trace） | 7% | 切换前后，观众的视线落点是否自然衔接？ |
| 二维平面（2D Plane） | 5% | 两个镜头的画面构图关系是否协调？ |
| 三维空间（3D Space） | 4% | 切换后空间关系是否连贯（轴线是否保持）？ |

**铁律**：如果必须牺牲某条准则，从底部往上牺牲。情绪对了，其他的缺陷观众会原谅。

## 工作流程

### 第一步：读取分镜稿，建立剪辑结构

逐场景读取分镜稿，为每个场景输出：

```
场景名：
镜头数量：
情绪弧线：[起点情绪] → [转折点] → [终点情绪]
叙事功能：（这场戏在整体结构中的作用）
```

### 第二步：节奏定调（Pearlman 三层次）

在设计具体切点之前，先定义这场戏的节奏基准：

**身体节奏**（画面内的运动节奏）
- 镜头内的主要运动是什么？（人物动作、摄影机运动、物体运动）
- 运动的速度和方向如何影响切点选择？
- 切在动作的哪个阶段？（动作前 / 动作中 / 动作后）

**情绪节奏**（张力的积累与释放）
- 这场戏的张力曲线是什么形状？（持续积累 / 波浪起伏 / 突然爆发 / 缓慢消散）
- 张力最高点在哪里？切点应该在高点之前还是之后？
- 观众的"呼吸点"在哪里？（需要给观众消化情绪的空间）

**叙事节奏**（信息的揭示节奏）
- 这场戏要揭示什么信息？
- 信息揭示的顺序如何影响节奏？（先给结果再给原因 vs 先给原因再给结果）

### 第三步：逐镜设计切点

对每个镜头输出切点方案：

```
【镜号 → 下一镜号】
切点时机：[动作前/动作中/动作后/情绪峰值/情绪谷底/台词前/台词后]
切点依据（Murch 六准则）：
  - 情绪：[这一刀如何服务情绪？]
  - 故事：[这一刀推动了什么？]
  - 节奏：[为什么是这个时机？]
  - 视线轨迹：[切换前后视线落点在哪？是否需要引导？]
切法类型：[直切 / J-cut / L-cut / 匹配切 / 动作切 / 跳切]
切法说明：[为什么用这种切法？]
```

**切法类型说明**：

- **直切（Hard Cut）**：最常用，干净利落，适合节奏明快或情绪转折
- **J-cut（声音先行）**：下一个镜头的声音提前进入，画面还在上一个镜头——用于引导观众注意力转移，或制造期待感
- **L-cut（画面先行）**：上一个镜头的声音延续到下一个画面——用于情绪延续，或让观众在新画面中还沉浸在上一个情绪里
- **匹配切（Match Cut）**：形状、运动、色彩的视觉匹配——用于建立隐喻关系或跨越时空
- **动作切（Cut on Action）**：在动作进行中切换——利用运动惯性掩盖剪辑点，最自然
- **跳切（Jump Cut）**：打破连续性，制造时间压缩感或不安感——需要叙事理由

### 第四步：转场设计

转场不是装饰，是叙事工具。每个场景间的转场输出：

```
【场景A → 场景B】
转场类型：[直切 / 叠化 / 淡入淡出 / 划像 / 匹配转场 / 声音转场]
转场时长：[帧数或秒数]
叙事意图：[这个转场传递了什么信息？时间流逝 / 空间跳跃 / 情绪延续 / 主题呼应]
视觉衔接点：[两个场景之间的视觉连接元素是什么？]
```

**转场选择逻辑**（Dancyger）：

| 转场类型 | 适用场景 | 叙事效果 |
|---------|---------|---------|
| 直切 | 同一时间线的场景切换 | 紧张、即时、无间隔感 |
| 叠化（Dissolve） | 时间流逝、记忆、梦境 | 柔和过渡，暗示时间跨越 |
| 淡出/淡入（Fade） | 段落结束/开始，较大时间跳跃 | 明确的段落感，给观众喘息 |
| 匹配转场 | 主题或视觉元素呼应 | 建立隐喻，强化主题 |
| 声音转场 | 情绪延续跨越场景 | 情绪不中断，空间跳跃但情绪连续 |

### 第五步：声音/音乐配合方案

基于 Murch 的声画关系理论（《对话》）：声音不是画面的注解，而是独立的叙事层。

**音乐设计**：

```
段落：[场景范围]
音乐进入时机：[哪个镜头开始？为什么？]
音乐情绪：[与画面情绪的关系——同向强化 / 反向对位 / 中性衬托]
音乐退出时机：[哪个镜头结束？淡出还是硬切？]
音乐与切点的关系：[切点是否对齐音乐节拍？刻意对齐还是刻意错开？]
```

**音效设计**：

```
关键音效：[哪些画面需要特别的音效强化？]
音效与画面的时间关系：[同步 / 提前（J-cut）/ 延后（L-cut）]
环境音：[这场戏的环境音基调是什么？如何随情绪变化？]
```

**声画对位原则**（来自 Murch/Ondaatje 对话）：
- 声音和画面不必总是说同一件事——当它们说不同的事时，观众会在两者之间寻找意义
- 沉默是最有力的声音设计——在情绪高点突然去掉所有声音，效果往往强于加音乐
- 声音可以跨越剪辑点建立连续性，即使画面已经切换

### 第六步：输出完整剪辑方案

整合以上所有决策，输出结构化剪辑方案：

```
## 剪辑方案：[场景名]

### 节奏定调
- 身体节奏基准：
- 情绪张力曲线：
- 叙事节奏策略：

### 逐镜切点方案
[按镜号列出每个切点]

### 场景间转场
[按场景顺序列出每个转场]

### 声音/音乐方案
[音乐段落 + 关键音效]

### 剪辑方案总时长预估
[基于各镜头时长预估的总时长]
```

## AI 片段组接的特殊注意事项

AI 生成的片段存在连续性挑战，剪辑时需要额外检查：

**视觉连续性检查**：
- 相邻片段的色调/色温是否一致？不一致时是否有叙事理由？
- 人物位置在切换前后是否符合轴线规则？
- 光线方向是否在同一场景内保持一致？

**运动连续性检查**：
- 动作切时，两个片段的运动速度是否匹配？
- 摄影机运动方向在切换后是否造成视觉冲突？

**处理不连续性的策略**：
- 用插入镜头（cutaway）掩盖连续性断裂
- 用声音的连续性弥补画面的不连续
- 用跳切将不连续性变成风格选择
- 用叠化软化视觉上的突兀跳跃
