# Level 4: 建立流程

流程化 = 标准化 + 知识沉淀

## 两个核心技能

| 技能 | 核心问题 | 产出 |
|------|----------|------|
| 4.1 标准化工作流 | 怎么让每次执行都高质量？ | 流程模板 + 检查清单 + 模板库 |
| 4.2 知识积累与复用 | 怎么让经验不流失？ | 经验库 + 个人风格 + 领域知识 |

## 核心原则

1. **流程不是限制，而是保障** — 标准化保证质量下限，不限制创造力上限
2. **模板是起点，不是终点** — 基于模板快速启动，根据具体情况调整
3. **每次执行都是学习机会** — 积累的知识是最宝贵的资产
4. **记录比记忆可靠** — 检查清单防止遗漏，经验库防止重复犯错
