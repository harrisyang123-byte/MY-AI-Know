# Skill 3.1: 提示词结构设计

## 标准8段式结构

```
┌─────────────────────────────────────┐
│ 1. 核心概念 (Core Concept)          │  ← 最重要，放最前面
│    一句话说明要生成什么              │
├─────────────────────────────────────┤
│ 2. 视角与构图 (Camera & Composition)│  ← 决定空间关系
│    机位、角度、景别                  │
├─────────────────────────────────────┤
│ 3. 空间结构 (Spatial Structure)     │  ← 建立坐标系
│    场景的整体布局和方位              │
├─────────────────────────────────────┤
│ 4. 元素描述 (Elements)              │  ← 从背景到前景
│    - 背景层                          │
│    - 中景层                          │
│    - 前景层                          │
├─────────────────────────────────────┤
│ 5. 光线系统 (Lighting)              │  ← 氛围的关键
│    光源类型、位置、强度、色温        │
├─────────────────────────────────────┤
│ 6. 材质与细节 (Materials & Details) │  ← 质感和真实度
│    表面特征、纹理、新旧程度          │
├─────────────────────────────────────┤
│ 7. 风格与氛围 (Style & Mood)        │  ← 整体感觉
│    艺术风格、情绪、时代感            │
├─────────────────────────────────────┤
│ 8. 技术参数 (Technical Specs)       │  ← 渲染质量
│    渲染方式、质量要求                │
└─────────────────────────────────────┘
```

**为什么这个顺序？**
- AI 模型对提示词的前半部分权重更高
- 从宏观到微观，符合人类理解习惯
- 先建立框架，再填充细节

## 8大精确描述技巧

### 技巧1：使用具体数值

| ✗ 差 | ✓ 好 |
|-------|-------|
| "一个很大的房间" | "一个 8m×12m 的房间，层高 4.5m" |
| "微弱的光线" | "单支蜡烛(2800K 琥珀色)，占 95% 光照" |
| "很多窗户" | "东墙有 4 扇深色木质格栅窗，每扇 80×150cm" |

常用数值类型：尺寸、数量、角度、距离、色温(K)、百分比

### 技巧2：使用空间定位语言

| ✗ 差 | ✓ 好 |
|-------|-------|
| "桌子旁边有椅子" | "书桌西侧(前方)0.5m 处，深色高靠背椅，椅背朝向镜头" |

空间定位模板：
```
[物体] 位于 [绝对位置](东/西/南/北墙, 或房间中央)
在 [相对位置](前景/中景/背景, 左/中/右)
距离 [参照物] 约 [数值]
[方位关系](在...前方/后方/左侧/右侧)
```

### 技巧3：使用层次化描述

```
背景层:
  - 东墙: 4扇深色木质格栅窗...
  - 东墙装饰: 泛黄海图...

中景层:
  - 书桌: 深色硬木,长方形...
  - 人物: 张十三,坐在桌北侧...

前景层:
  - 椅子: 深色高靠背,椅背朝向镜头...
  - 地毯: 深蓝色花卉图案...
```

### 技巧4：使用行为动词

| ✗ 差 | ✓ 好 |
|-------|-------|
| "蜡烛的光" | "蜡烛剧烈摇曳，火焰高度变化，制造晃动的阴影" |
| "门的状态" | "门被缓慢推开，仅露出 3-5cm 窄缝，冷白雪光从缝隙射入" |

### 技巧5：使用排除法（负向提示词）

正向提示词：告诉 AI 要什么
负向提示词：告诉 AI 不要什么

```
正向: "摄影机在西墙，面向东墙拍摄"
负向: "不要显示西墙，不要显示西墙上的书柜，
       不要显示摄影机背后的任何物体"
```

常见负向提示词类别：
- 空间错误："不要显示[不可见的墙面]"
- 风格错误："不要卡通化，不要过度饱和"
- 元素错误："不要现代物品，不要电器"
- 质量问题："不要模糊，不要变形，不要重复"

### 技巧6：使用参照物和对比

| ✗ 差 | ✓ 好 |
|-------|-------|
| "一个大桌子" | "一张可容纳 4-6 人的长方形书桌，长约 2.5m(相当于 3 个成年人肩宽)，宽约 1.2m" |

参照物类型：人体尺度、日常物品、建筑标准、相对比例

### 技巧7：使用感官描述

```
"寒冬深夜，室内温度接近冰点，
 可以想象呼出的白气，
 木质家具因寒冷而显得更加坚硬，
 蜡烛火焰在冷空气中剧烈摇曳"
```

感官暗示：温度(寒冷/温暖)、声音(安静/嘈杂)、触觉(粗糙/光滑)、气味(古旧/新鲜)

### 技巧8：使用风格参考

| ✗ 差 | ✓ 好 |
|-------|-------|
| "写实风格" | "in the visual style of 'Chang'an 30000 Miles' background art, Lights Animation Studio style, hyperrealistic environment rendering, cinematic Chinese architecture, photorealistic materials and lighting" |

参考类型：电影/动画、艺术家、艺术运动、技术风格

## 提示词优化实战案例

**初稿** → **精确化** → **结构化** → **空间验证** → **最终版**

```
初稿:
"生成一个清代办公室，从西向东拍，
 有桌子、椅子、窗户，昏暗的光线，
 一个老人坐在桌子旁边"

精确化:
"1895年清代海军衙门办公室，
 摄影机在西墙，平视面向东墙，
 房间8×12m，层高4.5m，
 书桌在中央，距东窗1.5m，
 张十三(老者，长须)坐在桌北侧，
 单支蜡烛在桌东端，剧烈摇曳，
 东墙有4扇木质格栅窗"

结构化:
"【核心概念】1895年寒冬深夜，清代海军衙门办公室内景
 【视角】摄影机在西墙，平视，面向东墙，广角全景
 【空间】房间8×12m，层高4.5m，东墙=背景，西墙=身后(不可见)
 【背景层-东墙】4扇深色木质格栅窗，每扇80×150cm...
 【中景层-书桌区域】书桌250×120cm，距东窗1.5m；张十三...
 【前景层】椅子、地毯...
 【光线】主光:蜡烛(2800K),95%光照；辅光:月光(6000K),极微弱
 【风格】hyperrealistic, Chang'an 30000 Miles style"

空间验证 + 负向提示词:
正向: [同上]
负向: "不要显示西墙，不要显示西墙书柜，
       不要现代物品，不要电器，不要过度明亮"
```
