## 提示词构建核心流程

```
Step 1: 初稿 → 按结构框架填充基本信息
Step 2: 精确化 → 模糊词替换为数值
Step 3: 结构化 → 按背景→中景→前景排序
Step 4: 空间验证 → 检查可见性、遮挡关系
Step 5: 一致性检查 → 时间/风格/光源是否自洽
Step 6: 负向提示词 → 排除错误元素和风格
Step 7: 最终审查 → 脑海中能否清晰"看到"场景？
```

## 迭代优化核心原则

1. **每次只改一个维度** — 便于识别哪个改动有效
2. **从大到小，从重到轻** — 核心概念 > 空间关系 > 关键元素 > 光影氛围 > 细节
3. **保留成功的部分** — 明确"保持 X 不变，只改 Y"
4. **设定迭代上限** — 同一问题不超过 3 次迭代，超过则换策略
