# Skill 5.2: 主动学习与探索

## 五维主动学习

### 维度1：工具探索

不满足于熟悉的工具，主动探索新工具：

1. **定期测试新工具/新功能** — 每月尝试 1 个不常用的工具，测试新发布功能，对比效果
2. **深入研究工具特性** — 阅读文档，测试边界情况，发现隐藏功能
3. **创造性使用** — 尝试非常规用法，组合多个工具，发明新工作流

示例：
```
探索: midjourney 的 sref 功能
测试: 用同一个 sref 生成系列图
发现: 可以保持风格高度一致
应用: 用于系列作品生成
记录: 添加到工具特性库
```

### 维度2：技能拓展

识别弱项，主动学习：

1. **识别知识盲区** — 哪些任务不擅长？哪些领域不了解？
2. **制定学习计划** — 选择 1-2 个优先提升的技能，设定目标
3. **刻意练习** — 专门练习弱项，寻找相关任务，请求反馈
4. **评估进步** — 定期测试，对比之前表现

示例：
```
弱项: 人物肖像生成
目标: 3个月内达到中等水平
计划:
  Week 1-2: 研究优秀人物提示词
  Week 3-4: 练习面部特征描述
  Week 5-6: 练习姿态和表情
  Week 7-8: 练习服装和配饰
  Week 9-12: 综合练习和实战
评估: 每周生成5张人物图，记录进步
```

### 维度3：领域深耕

选择感兴趣的领域，深入研究：

1. **选择领域** — 基于兴趣、需求、优势
2. **系统学习** — 研究专业知识，收集参考资料，学习专业术语
3. **建立专业库** — 领域知识库 + 参考图库 + 提示词模板库 + 案例库
4. **形成专长** — 成为该领域的专家，建立独特优势

示例：
```
选择: 历史场景重现
学习:
  - 研究各朝代建筑特征
  - 收集历史照片和画作
  - 学习历史服饰、器物
建立:
  - 各朝代建筑特征库
  - 历史服饰描述词汇库
  - 时代场景提示词模板
成果:
  - 可以快速生成准确的历史场景
  - 细节考证准确
  - 形成个人特色
```

### 维度4：创新实验

不满足于现有方法，尝试创新：

1. **质疑现状** — 为什么要这样做？有更好的方法吗？
2. **大胆假设** — 如果...会怎样？能不能...？
3. **小心求证** — 设计实验，控制变量，记录结果
4. **总结规律** — 什么有效？为什么有效？可以推广吗？

示例：
```
假设: 提示词的顺序会影响结果吗？
实验: 固定内容，改变顺序，生成多组对比
发现: 前面的描述权重更高，核心概念应该放最前面
应用: 更新提示词结构模板，优先级高的放前面
```

### 维度5：跨界学习

从其他领域汲取灵感：

| 领域 | 可迁移知识 | 应用到提示词 |
|------|-----------|-------------|
| 摄影 | 构图、光线、色彩 | 三分法→构图提示，布光方案→照明描述 |
| 电影 | 镜头语言、叙事 | 镜头类型→视角描述，三点布光→光线系统 |
| 绘画 | 艺术风格、技法 | 色彩理论→配色方案，笔触描述→风格标签 |
| 建筑 | 空间、结构 | 空间关系→坐标描述，材质→纹理描述 |
| 设计 | 排版、视觉层次 | 视觉层次→前景中景背景，对比→构图 |

示例：
```
学习: 电影摄影中的"三点布光"
理解:
  - 主光(Key Light): 主要光源
  - 辅光(Fill Light): 填充阴影
  - 轮廓光(Back Light): 分离主体和背景
应用到提示词:
  "Lighting setup:
   - Key light: [主光描述]
   - Fill light: [辅光描述]
   - Back light: [轮廓光描述]"
效果: 光线更专业，层次更丰富
```

## 学习习惯体系

| 频率 | 习惯 | 内容 | 年积累 |
|------|------|------|--------|
| 每日 | 每日一学 | 一个新词汇/技巧/概念/案例 | 365个知识点 |
| 每周 | 每周一练 | 专门练习一个技能1-2小时 | 52个技能提升 |
| 每月 | 每月一探 | 深入探索一个新领域 | 12个新领域 |
| 每季 | 每季一创 | 提出假设，设计实验，总结规律 | 4个创新成果 |
| 每年 | 每年一精 | 系统学习一个专业领域，形成专长 | 多领域专家 |

## 速度与质量平衡训练

| 级别 | 时间 | 目标 |
|------|------|------|
| Level 1 | 5分钟 | 生成一张可用的图（理解1分钟+提示词2分钟+生成验证2分钟） |
| Level 2 | 15分钟 | 生成一张高质量的图（理解规划3分钟+提示词5分钟+生成验证5分钟+迭代2分钟） |
| Level 3 | 30分钟 | 完成一个复杂场景（深度理解5分钟+策略规划5分钟+首次生成10分钟+迭代优化10分钟） |
| Level 4 | 1小时 | 完成一个系列3-5张图（整体规划10分钟+风格基准15分钟+批量生成25分钟+统一调整10分钟） |

反思：哪个环节最耗时？如何优化流程？质量与速度的平衡点在哪？
