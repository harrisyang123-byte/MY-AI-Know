# Skill 5.1: 元认知能力

## 三层元认知

### 层次1：执行中的自我监控

在任务进行时，同时观察自己：
- 我现在在做什么？
- 这个方法有效吗？
- 有更好的方法吗？
- 我是否偏离了目标？

实践方法：
- **内心对话**："我现在在构建提示词，我是否包含了所有关键元素？这个描述够精确吗？"
- **检查点**：在关键步骤设置检查点
  - 理解阶段结束 → 我真的理解了吗？
  - 规划完成 → 这个计划合理吗？
  - 生成完成 → 结果符合预期吗？

### 层次2：执行后的反思

任务完成后，回顾整个过程：

**反思框架**：

1. **结果评估**
   - 任务完成得如何？(1-10分)
   - 用户满意吗？达到预期了吗？

2. **过程分析**
   - 哪个阶段最顺利？为什么？
   - 哪个阶段最困难？为什么？
   - 时间分配合理吗？

3. **决策复盘**
   - 关键决策有哪些？
   - 每个决策的依据是什么？
   - 决策正确吗？如果重来会怎么决策？

4. **知识提取**
   - 学到了什么新知识？
   - 发现了什么新模式？
   - 哪些可以复用？

5. **改进计划**
   - 下次要改进什么？
   - 具体怎么改进？

### 层次3：模式识别与系统优化

跨任务地观察，识别更大的模式：

1. **定期回顾**（每周/每月）
   - 成功率如何？常见问题是什么？有重复的错误吗？

2. **分类统计**
   ```
   任务类型   | 数量 | 成功率 | 平均耗时 | 常见问题
   -----------|------|--------|----------|----------
   简单生成   | 20   | 95%    | 5分钟    | 少
   复杂场景   | 10   | 70%    | 30分钟   | 空间关系
   系列作品   | 3    | 100%   | 2小时    | 一致性
   ```

3. **错误模式分析**
   - 空间关系错误: 5次 → 模式: 对"摄影机位置"理解不够 → 改进: 建立标准化空间分析流程
   - 风格偏离: 3次 → 模式: 风格描述词不够强 → 改进: 增加风格关键词库

4. **成功模式提取**
   - 详细的空间描述 → 100%成功 → 结论: 空间描述是成功关键 → 行动: 将空间描述标准化
   - 使用参考图 → 90%成功 → 结论: 参考图大幅提升成功率 → 行动: 主动建议用户提供参考图

5. **方法论进化**
   ```
   旧方法: 直接开始写提示词
   问题: 容易遗漏，结构混乱
   新方法: 先建立框架，再填充细节
   效果: 成功率提升20%
   ```

## 元认知实践工具

### 工具1：执行日志

```
日期: YYYY-MM-DD
任务: [任务描述]
耗时: [总耗时]
迭代次数: [N次]
结果: [成功/失败/部分成功]

过程记录:
  - 理解阶段: [耗时]，[做了什么]
  - 规划阶段: [耗时]，[做了什么]
  - 首次生成: [结果]
  - 迭代: [每次迭代的关键改动]

反思:
  做得好: [什么做对了]
  可改进: [什么可以更好]
  学到的: [新知识/新模式]
  可复用: [哪些经验可以用到其他项目]
```

### 工具2：能力雷达图

定期评估自己的能力维度：

```
理解能力:     ████████░░ 8/10
规划能力:     ███████░░░ 7/10
提示词工程:   █████████░ 9/10
工具精通:     ████████░░ 8/10
质量控制:     ████████░░ 8/10
效率:         ██████░░░░ 6/10
创新:         █████░░░░░ 5/10
```

分析：
- 优势：提示词工程(9分) → 继续保持，形成特色
- 弱项：创新(5分) → 需要加强，多尝试新方法
- 改进中：效率(6分) → 通过模板和流程提升

### 工具3：决策树优化

记录每次的决策和结果，优化决策树：

```
情况: 用户需求模糊
  旧决策: 直接按自己理解生成 → 50%需要重做
  新决策: 先问清楚再生成 → 95%一次成功
  → 更新决策树: 需求模糊 → 必须先澄清

情况: 复杂场景
  旧决策: 一次性生成所有元素 → 70%有问题
  新决策: 评估复杂度，必要时分步 → 90%成功
  → 更新决策树: 复杂度>阈值 → 分步生成
```

### 工具4：个人知识图谱

将所有知识连接起来：

```
[清代建筑] ─关联─> [室内场景生成]
     │                    │
   包含                 需要
     │                    │
[时代特征] ←─影响─ [提示词选择]
     │                    │
   决定                 使用
     │                    │
[描述词汇库] ─支持─> [详细描述]
```

使用：遇到"清代建筑"任务 → 激活相关知识节点 → 调用"时代特征"、"描述词汇库" → 应用到"提示词选择"和"详细描述"
