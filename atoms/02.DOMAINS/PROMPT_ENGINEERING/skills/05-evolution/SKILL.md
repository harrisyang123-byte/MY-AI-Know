# Level 5: 持续进化

进化 = 元认知 + 主动学习

## 两个核心技能

| 技能 | 核心问题 | 产出 |
|------|----------|------|
| 5.1 元认知能力 | 我是怎么思考的？怎么改进？ | 执行日志 + 能力雷达图 + 决策树优化 |
| 5.2 主动学习与探索 | 怎么持续进步？ | 工具探索 + 技能拓展 + 领域深耕 + 创新实验 |

## 元认知三个层次

```
层次1: 执行中的自我监控
  "我现在在做什么？这个方法有效吗？有更好的方法吗？"

层次2: 执行后的反思
  "什么做得好？什么可以改进？学到了什么？"

层次3: 模式识别与系统优化
  "我总是在哪个环节出问题？我的方法论需要如何进化？"
```

## 主动学习五个维度

```
维度1: 工具探索 — 定期测试新工具/新功能，创造性使用
维度2: 技能拓展 — 识别弱项，刻意练习
维度3: 领域深耕 — 选择领域，系统学习，建立专业库
维度4: 创新实验 — 质疑现状，大胆假设，小心求证
维度5: 跨界学习 — 从摄影/电影/绘画/建筑/设计汲取灵感
```
