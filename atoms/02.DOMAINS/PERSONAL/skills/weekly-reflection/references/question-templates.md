# 反思问题模板库

这个文件提供了各类反思问题的模板，帮助你针对不同情况提出有深度的问题。

## 关于时间分配的问题

### 模板1：优先级倒挂
```
你说[目标A]很重要，但本周你在[活动B]上花了X小时，在[目标A]上只花了Y小时。
这种时间分配是如何发生的？是[目标A]实际上没那么紧急，还是[活动B]提供了某种[目标A]无法提供的满足感？
```

### 模板2：碎片化时间
```
你的记录显示本周有[N]个不同的小任务/想法，但没有一个深入推进。
如果只能选一个在下周投入10小时，你会选哪个？为什么其他的不值得这个投入？
```

### 模板3：时间黑洞
```
[活动X]在你的记录中反复出现，但你从未提到它带来的具体成果。
这个活动是在帮你接近目标，还是在帮你逃避某个更困难的任务？
```

## 关于行为模式的问题

### 模板4：重复的拖延
```
这是第[N]周你提到要做[X]但没做。
如果这件事真的重要，你会如何重新设计你的一天/一周来保护这个时间？
还是说，它在你的优先级里其实没那么高，只是你觉得"应该"做？
```

### 模板5：情绪触发点
```
你在[情境A]中感到[情绪X]，在[情境B]中也是。这两个情境的共同点是什么？
这个情绪是在提醒你什么？是能力边界、价值观冲突，还是未满足的需求？
```

### 模板6：替代行为
```
你用[活动A]替代了[活动B]，这不是第一次了。
[活动A]提供了什么[活动B]没有的？是即时反馈、确定性、还是更低的心理门槛？
```

## 关于目标与价值观的问题

### 模板7：目标的真实性
```
你的季度目标是[X]，但本周的行为更像是在追求[Y]。
如果[Y]才是你真正想要的，为什么不把它设为目标？
如果[X]才是真正重要的，是什么让你在日常选择中忽略了它？
```

### 模板8：灵感与行动的鸿沟
```
你记录了[N]个"可以做"的想法/灵感，但没有一个进入执行阶段。
是因为它们不够好，还是因为你在用"收集想法"来替代"执行想法"？
如果只能选一个在明天开始做，你会选哪个？
```

### 模板9：外部期待 vs 内在动力
```
你提到[X]时用了"应该"、"需要"这样的词，而提到[Y]时明显更有热情。
[X]是你真正想做的，还是你觉得别人期待你做的？
如果没有外部评价，你还会做[X]吗？
```

## 关于学习与成长的问题

### 模板10：知识的应用
```
你接触了[概念A]、[概念B]、[概念C]，都觉得"有用"、"可以用在..."。
但这些知识有哪个真正改变了你本周的行为或决策？
如果没有，是因为它们不够重要，还是因为从"知道"到"做到"之间缺了什么？
```

### 模板11：舒适区边界
```
本周你在[领域X]上很顺利，在[领域Y]上卡住了。
[领域Y]让你不舒服的具体是什么？是技能缺口、不确定性、还是害怕失败？
如果你的目标是成长，为什么不把更多时间投入到让你不舒服的地方？
```

### 模板12：反馈循环
```
你在[活动X]上投入了时间，但没有提到任何反馈或成果。
你如何知道自己在进步？如果没有反馈机制，你是在练习，还是在重复？
```

## 关于人际关系的问题

### 模板13：能量流动
```
你提到和[人A]的互动让你[情绪X]，和[人B]的互动让你[情绪Y]。
这些关系是在给你充电还是耗电？如果是耗电，你为什么继续投入？
如果是充电，你有没有主动创造更多这样的互动？
```

### 模板14：边界问题
```
你因为[他人的需求/请求]而[改变了自己的计划/感到压力]。
这是你主动选择的，还是你觉得"不得不"这样做？
如果是后者，是什么让你难以说不？
```

## 关于身心状态的问题

### 模板15：身体信号
```
你提到[身体症状X]（如疲劳、失眠、食欲变化），同时在推进[高压任务Y]。
你的身体在告诉你什么？如果继续忽略这些信号，三个月后会怎样？
```

### 模板16：恢复机制
```
本周你的记录里几乎都是"做事"，很少有"休息"或"玩"。
你上一次真正放松（不带任何目的性）是什么时候？
如果你相信可持续性，为什么不把恢复也当作一个需要完成的任务？
```

## 使用指南

### 如何选择合适的模板

1. 先识别用户碎碎念中的核心矛盾
2. 从上述模板中选择最接近的1-2个
3. 用用户的具体行为和语言替换模板中的占位符
4. 确保问题包含具体证据（时间、次数、情绪词）

### 如何改编模板

好的改编：
- 加入用户的原话（用引号标注）
- 使用具体的数字和时间
- 连接到用户明确提到的目标

差的改编：
- 使用模糊的"有时候"、"经常"
- 加入用户没提到的假设
- 使用说教性的语气

### 三个问题的组合策略

推荐组合：
1. 一个关于时间/行为的问题（基于事实）
2. 一个关于模式/习惯的问题（基于观察）
3. 一个关于价值观/目标的问题（基于深层动机）

这样的组合能从表层行为逐步深入到底层信念。

## 问题质量自检

一个好的反思问题应该：
- ✅ 让用户停下来思考至少30秒
- ✅ 包含具体的行为证据
- ✅ 暴露一个矛盾或盲点
- ✅ 没有预设的"正确答案"
- ✅ 用"如果...会怎样"引导思考

一个差的反思问题：
- ❌ 可以用"是"或"不是"回答
- ❌ 用户一看就知道"应该"怎么回答
- ❌ 包含评判性的形容词
- ❌ 太抽象，缺乏具体情境
- ❌ 实际上是在给建议而非提问
