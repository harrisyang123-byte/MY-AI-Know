# Skill: 竞品分析

## 概述

不是泛泛而谈，而是像产品经理做竞品调研一样——拆解每一个细节，找到可复用的规律和可切入的空白。

## 分析框架

### 第一步：账号定位解读

- 账号名称、简介传递的核心定位
- 目标受众判断（谁在看？）
- 内容价值主张（解决什么问题？）
- 发布频率和节奏

### 第二步：选题规律分析

对所有文章标题进行分类，提炼：

**话题分布**（按频次排序）：
- 各话题类别占比
- 高频关键词
- 明显回避的话题（内容缺口）

**选题触发逻辑**：
- 情绪触发型（愤怒/焦虑/共鸣/温暖）
- 场景触发型（具体生活场景）
- 身份触发型（"有孩子的必看"）
- 时效触发型（节日/热点）

### 第三步：标题公式拆解

从标题中提炼可复用的公式：

**结构模式**：
- 警示型：`警惕！/警告！` + 危害描述
- 否定型：`别再...` + 错误行为
- 反问型：`你到底是...？`
- 数字型：`X个/X种/X%`
- 对比型：`不是...而是...`

**情绪触发词库**：记录高频出现的情绪词

**标题长度和节奏**：平均字数、标点使用习惯

### 第四步：内容结构解析

选取3-5篇代表性文章，逐篇拆解：

```
开头（前200字）：如何切入？用什么钩子？
中段：论证逻辑是什么？用了哪些素材？
结尾：行动号召是什么？情绪落点在哪里？
```

**常见结构模式**：
- 痛点切入 → 共情父母 → 转折批判 → 解决方案 → 行动号召
- 场景还原 → 情绪共鸣 → 原因分析 → 正确做法

### 第五步：爆款特征识别

基于标题和内容，识别高互动内容的共同特征：

- 情绪强度（是否制造紧迫感/焦虑感）
- 受众代入感（读者能否立刻对号入座）
- 实用性（是否给出具体可操作的建议）
- 话题敏感度（是否触碰代际冲突的核心矛盾）

### 第六步：差异化机会

对比竞品定位和自身定位（产品经理视角的银发解决方案），找到：

- **竞品在做、我们也要做**：已验证的高需求内容
- **竞品在做、我们做得更好**：有差异化空间的内容
- **竞品没做、我们可以做**：内容空白地带
- **竞品在做、我们不做**：主动放弃的方向
